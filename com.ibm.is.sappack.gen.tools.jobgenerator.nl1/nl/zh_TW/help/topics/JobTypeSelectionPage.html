選取工作類型

使用此頁面，可以選取要產生的 IBM® InfoSphere™ DataStage® 工作的類型。您可以使用此頁面來產生 ABAP Extract 工作、IDoc Extract 工作或 IDoc Load 工作。

ABAP Extract 工作
選取此選項，可以產生從 SAP 系統中擷取邏輯表格的工作。明確地將 SAP 邏輯表格 Meta 資料匯入至邏輯資料模型，或是選取 IDoc 的檢查表格選項來進行相同的作業，就可以使用此選項。
IDoc Extract 工作
選取此選項，可以建立工作，以收集先前傳送給 InfoSphere DataStage Server 上所配置「IDoc 接聽器」服務的 IDoc。
IDoc Load 工作
選取此選項，可以產生讀取 IDoc 並將它們傳送給 SAP 的工作。
