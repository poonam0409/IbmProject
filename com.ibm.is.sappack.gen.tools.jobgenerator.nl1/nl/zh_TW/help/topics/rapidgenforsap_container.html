IBM InfoSphere Rapid Generator for SAP Applications

您可以透過使用 IBM InfoSphere Rapid Modeler for SAP Applications 所匯入的 SAP meta 資料，來產生 InfoSphere™ DataStage® 工作。這些 InfoSphere DataStage 工作會擷取或載入 SAP 物件的實際資料，而 SAP 物件是透過匯入至關聯式資料庫或純文字檔或從中匯入的 Meta 資料來說明。

  • 選取工作類型
    使用此頁面，可以選取要產生的 IBM® InfoSphere DataStage 工作的類型。您可以使用此頁面來產生 ABAP Extract 工作、IDoc Extract 工作或 IDoc Load 工作。
  • InfoSphere Information Server 明細
    指定 IBM InfoSphere Information Server 網域明細及 IBM InfoSphere DataStage 認證。
  • ABAP Extract 選項
    使用此頁面，可以在工作產生設計中配置 ABAP Extract 選項。
  • IDoc Extract 選項
    使用此頁面，可以指定 IDoc Extract 工作的來源及目標選項。
  • IDoc Load 選項
    使用此頁面，可以配置 IDoc Load 工作的選項。
  • 定義工作參數
    此頁面會顯示使用者指定的所有工作參數及參數集，以供目標及來源內容使用。您可以修改參數的所有欄位（參數名稱除外）。
  • 定義直欄衍生
    使用此頁面，可以衍生要套用至特定類型的所有直欄的表示式。這些表示式是定義於所產生工作的 Transformer 暫置內。您可以交互參照含有 $1 的轉換器輸入引數。部分範例為：TrimLeadingTrailing($1) 或 NullToEmpty($1)。
  • 解析工作參數
    您可以指定 IBM InfoSphere DataStage 工作參數的值。這裡所指定的值是用於該工作參數在所有工作及暫置內的所有實例
  • ABAP 程式設定
    使用此頁面，可以針對要為 IBM InfoSphere DataStage ABAP Extract 工作產生的程式，配置 ABAP 程式設定。
  • 驗證結果
    顯示所配置 SAP 設定的驗證結果。 綠色勾號指出已順利驗證設定，紅色勾號則指出發生錯誤。
  • 工作產生摘要
    顯示工作產生摘要及新工作的名稱。如果您選取啟動 DataStage 多個工作編譯器勾選框，則會在按一下完成之後啟動 DataStage 的「多個工作編譯器」精靈。 此精靈在開啟時已選取產生的工作，讓您可以同時編譯所有產生的工作。
