IDoc Extract 選項

使用此頁面，可以指定 IDoc Extract 工作的來源及目標選項。

註： 使用 "#" 符號括住文字，就可以針對文字欄位使用 InfoSphere™ DataStage® 工作參數。 此參數集是以語法 "#" #ParameterSetName.ParameterName# 表示。如需參數及參數集的相關資訊，請參閱《IBM® InfoSphere DataStage 及 QualityStage™ Designer 用戶端手冊》。
來源選項
指定 IDoc 擷取來源選項：
DataStage SAP 連線
IBM InfoSphere DataStage 中配置的 SAP 連線名稱。
改寫連線預設值
如果您要改寫 IBM InfoSphere DataStage 中配置的連線設定，請選取此勾選框。
SAP 使用者
SAP 使用者的使用者名稱。
SAP 使用者密碼
所指定 SAP 使用者的密碼。
SAP 使用者用戶端號碼
所指定 SAP 使用者的用戶端號碼。
SAP 使用者語言
所指定 SAP 使用者的語言。
目標選項
指定 IDoc Extract 的下列目標選項：
暫置類型
選取所產生目標暫置的類型：
ODBC Connector
指定 ODBC Connector 的選項：
ODBC 資料來源
從 SAP 擷取的物件會儲存在配置的 ODBC 資料來源表格中。
ODBC 使用者
改寫資料來源配置中所配置的使用者。它是選用欄位。
ODBC 密碼
改寫資料來源配置中所配置的密碼。它是選用欄位。
選取表格動作
您可以選取 APPEND 或 TRUNCATE 表格動作。如需相關資訊，請參閱 ODBC Connector 暫置文件。
循序檔
循序檔具有下列參數：
檔案
指定名稱，以用作用來儲存輸出的所有檔案的字首。
檔案更新模式
可能的選項為 APPEND、CREATE（如果存在則發生錯誤）或 OVERWRITE。如需相關資訊，請參閱 Sequential File 暫置文件。
第一行是直欄名稱
如果您要循序檔中的第一行包含直欄名稱，請選取此選項。
