Opciones de ABAP Extract

Utilice esta página para configurar las opciones de ABAP Extract en el diseño de generación del trabajo.

Nota: Puede utilizar los parámetros de trabajo de InfoSphere DataStage para los campos de texto si escribe el texto entre símbolos "#". Los conjuntos de parámetros vienen indicados con la sintaxis "#" #ParameterSetName.ParameterName#. Para obtener más información sobre los parámetros y los conjuntos de parámetros, consulte IBM® InfoSphere DataStage and QualityStage Designer Client Guide
Opciones de origen
Conexión de SAP de DataStage
Nombre de la conexión de SAP configurada en DataStage.
Usuario de SAP
ID del usuario de SAP que se va a utilizar cuando se conecte a SAP.
Contraseña de usuario de SAP
Contraseña del usuario de SAP.
Número de cliente del usuario de SAP
Número de cliente del usuario de SAP.
Idioma del usuario de SAP
Idioma del usuario de SAP.
Opciones de destino
Tipo de etapa
Tipo de etapas generadas como destino de las tablas lógicas de SAP. Por ejemplo, Conector ODBC o Archivo secuencial.
Conector ODBC
Especifique las siguientes opciones para el conector ODBC:
Origen de datos ODBC
El objeto extraído de SAP se almacena en la tabla de origen de datos ODBC configurada.
Usuario de ODBC
Sobrescribe el usuario en la configuración del origen de datos. Este campo es opcional.
Contraseña de ODBC
Sobrescribe la contraseña en la configuración del origen de datos. Este campo es opcional.
Seleccionar la acción de tabla
Puede seleccionar las acciones de tabla APPEND o TRUNCATE. Para obtener más información, consulte la documentación de la etapa Conector ODBC.
Archivo secuencial
Especifique una de estas opciones para la etapa Archivo secuencial:
Prefijo de nombre de archivo
Prefijo de todos los archivos utilizados para almacenar la salida.
Modalidad de actualización de archivos
Puede seleccionar APPEND, CREATE (Error si existe) o OVERWRITE. Para obtener más información, consulte la documentación de la etapa Archivo secuencial.
La primera línea son nombres de columna
Seleccione si desea que la primera línea del archivo secuencial contenga los nombres de columna
Opciones del método de transferencia RFC
Especifique las opciones del método de transferencia RFC:
Sistema principal de pasarela de SAP
Nombre de sistema principal de la pasarela de SAP que ofrece servicios para comunicarse con el servidor SAP.
Servicio de pasarela de SAP
Nombre del servicio de pasarela.
Destinos de RFC
Un archivo de texto simple donde cada línea está formada por dos entradas separadas por un espacio en blanco. La primera entrada es el nombre del destino de RFC configurado en el sistema SAP. La segunda entrada es el ID de programa correspondiente del destino de RFC.
Núm. máximo de flujos en un trabajo
Los trabajos de ABAP Extract pueden generarse con varias etapas ABAP para reducir así el número de trabajos. No obstante, en un trabajo de ABAP Extract, cada etapa ABAP requiere la configuración de un destino de RFC exclusivo en SAP. Por lo tanto, el número máximo de etapas ABAP está limitado por el número de destinos de RFC proporcionados. También define el número máximo de flujos por trabajo o de instancias de etapa ABAP con un solo trabajo de IBM InfoSphere DataStage, limitado por el número de destinos de RFC proporcionados.
Crear destino de RFC automáticamente
Seleccione esta opción si el trabajo de IBM InfoSphere DataStage va a crear el destino de RFC en el tiempo de ejecución.
Suprimir destino de RFC si es necesario
Seleccione esta opción para permitir la supresión del destino de RFC si es necesario.
Validar valores de SAP
Pulse este botón para validar los valores de la conexión de SAP.
