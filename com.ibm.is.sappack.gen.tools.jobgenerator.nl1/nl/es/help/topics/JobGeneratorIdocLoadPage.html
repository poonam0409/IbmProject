Opciones de IDoc Load

Utilice esta página para configurar las opciones de los trabajos de IDoc Load.

Nota: Puede utilizar los parámetros de trabajo de InfoSphere DataStage para los campos de texto si escribe el texto entre símbolos "#". Los conjuntos de parámetros vienen indicados con la sintaxis "#" #ParameterSetName.ParameterName#. Para obtener más información sobre los parámetros y los conjuntos de parámetros, consulte IBM® InfoSphere DataStage and QualityStage Designer Client Guide
Opciones de origen
Especifique las siguientes opciones de origen para IDoc Load:
Tipo de etapa
Tipo de etapas generadas como origen de los IDocs. Por ejemplo, Conector ODBC o Archivo secuencial.
Conector ODBC
Especifique las siguientes opciones para el conector ODBC:
Origen de datos ODBC
Los objetos cargados en SAP se leen de las tablas que el origen de datos ODBC ha configurado en el sistema operativo.
Usuario de ODBC
Sobrescribe el usuario configurado en la configuración del origen de datos. Este campo es opcional.
Contraseña de ODBC
Sobrescribe la contraseña configurada en la configuración del origen de datos. Este campo es opcional.
Seleccionar la acción de tabla
Puede seleccionar las acciones de tabla APPEND, CREATE, REPLACE o TRUNCATE. Para obtener más información, consulte la documentación de la etapa Conector ODBC.
Condición SQL WHERE de ODBC
Especifique la cláusula SQL where que se va a utilizar cuando se consulten los datos de origen. La cláusula where que se especifique aquí se aplicará a todas las tablas que se consulten. Esta opción se utiliza normalmente junto con los campos técnicos que se añaden utilizando IBM InfoSphere Data Architect Rapid Modeler for SAP application.
Opciones de destino
Especifique las siguientes opciones de destino para la etapa IDoc Load:
Tipo de mensaje SAP
Especifique el tipo de mensaje SAP que se va a utilizar.
Conexión de SAP de DataStage
Nombre de la conexión de SAP configurada en DataStage.
Sobrescribir valores predeterminados de la conexión
Seleccione este recuadro de selección si desea sobrescribir los valores definidos en la conexión en IBM InfoSphere DataStage.
Usuario de SAP
ID de usuario del usuario de SAP.
Contraseña de usuario de SAP
Contraseña del usuario de SAP especificado.
Número de cliente del usuario de SAP
Número de cliente del usuario de SAP especificado.
Idioma del usuario de SAP
Idioma del usuario de SAP especificado.
