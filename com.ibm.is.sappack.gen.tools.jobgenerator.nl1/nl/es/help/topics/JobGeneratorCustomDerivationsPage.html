Definir derivaciones de columnas

Utilice esta página para derivar las expresiones que se van a aplicar a todas las columnas de un determinado tipo. Estas expresiones se definen en las etapas Transformer de los trabajos generados. Puede hacer referencias cruzadas del argumento de entrada del transformador con $1. Por ejemplo: TrimLeadingTrailing($1) o NullToEmpty($1).

Utilizar derivaciones personalizadas
Utilice este recuadro de selección para asignar una expresión para cada tipo de InfoSphere DataStage. Sólo puede asignar una correlación por tipo de datos. Puede utilizar el término especial $1 para hacer referencia a la columna de entrada en las expresiones. Por ejemplo, para ejecutar la función NullToEmpty de IBM® InfoSphere DataStage para todas las columnas de tipo CHAR, especifique CHAR en el campo Tipo de datos y NullToEmpty($1) en el campo Derivación. En el tiempo de ejecución, $1 se sustituye por el valor de columna correspondiente.
