Opciones de IDoc Extract

Utilice esta página para especificar las opciones de origen y destino de un trabajo de IDoc Extract.

Nota: Puede utilizar los parámetros de trabajo de InfoSphere DataStage para los campos de texto si escribe el texto entre símbolos "#". Los conjuntos de parámetros vienen indicados con la sintaxis "#" #ParameterSetName.ParameterName#. Para obtener más información sobre los parámetros y los conjuntos de parámetros, consulte IBM® InfoSphere DataStage and QualityStage Designer Client Guide
Opciones de origen
Especifique las opciones de origen de extracción de IDoc:
Conexión de SAP de DataStage
Nombre de la conexión de SAP configurada en IBM InfoSphere DataStage.
Sobrescribir valores predeterminados de la conexión
Seleccione este recuadro de selección si desea sobrescribir los valores de conexión configurados en IBM InfoSphere DataStage.
Usuario de SAP
Nombre de usuario del usuario de SAP.
Contraseña de usuario de SAP
Contraseña del usuario de SAP especificado.
Número de cliente del usuario de SAP
Número de cliente del usuario de SAP especificado.
Idioma del usuario de SAP
Idioma del usuario de SAP especificado.
Opciones de destino
Especifique las siguientes opciones de destino para IDoc Extract:
Tipo de etapa
Seleccione el tipo de etapas de destino generadas:
Conector ODBC
Especifique las opciones del conector ODBC:
Origen de datos ODBC
El objeto extraído de SAP se almacena en la tabla de origen de datos ODBC configurada.
Usuario de ODBC
Sobrescribe el usuario configurado en la configuración del origen de datos. Este campo es opcional.
Contraseña de ODBC
Sobrescribe la contraseña configurada en la configuración del origen de datos. Este campo es opcional.
Seleccionar la acción de tabla
Puede seleccionar las acciones de tabla APPEND o TRUNCATE. Para obtener más información, consulte la documentación de la etapa Conector ODBC.
Archivo secuencial
El archivo secuencial tiene los siguientes parámetros:
Archivo
Especifique el nombre que se va a utilizar como prefijo para todos los archivos utilizados para almacenar la salida.
Modalidad de actualización de archivos
Las opciones posibles son APPEND, CREATE (Error si existe) o OVERWRITE. Para obtener más información, consulte la documentación de la etapa Archivo secuencial.
La primera línea son nombres de columna
Seleccione si desea que la primera línea del archivo secuencial contenga los nombres de columna.
