Seleccionar un tipo de trabajo

Utilice esta página para seleccionar los tipos de trabajo de IBM® InfoSphere DataStage que se van a generar. Puede generar un trabajo de ABAP Extract, un trabajo de IDoc Extract o el trabajo de IDoc Load utilizando esta página.

Trabajo de ABAP Extract
Seleccione esta opción para generar trabajos que extraen tablas lógicas de un sistema SAP. Esta opción está disponible cuando se importan metadatos de tabla lógica de SAP al modelo de datos lógico de forma explícita o seleccionando las opciones de tabla de comprobación de los IDocs.
Trabajo de IDoc Extract
Seleccione esta opción para crear trabajos que recopilan los IDocs que se han enviado previamente al servicio de escucha de IDoc configurado en el servidor de InfoSphere DataStage.
Trabajo de IDoc Load
Seleccione esta opción para generar un trabajo que lee IDocs y los envía a SAP.
