IBM InfoSphere Rapid Generator for SAP Applications

Puede generar trabajos de InfoSphere DataStage con los metadatos de SAP importados utilizando IBM InfoSphere Rapid Modeler for SAP Applications. Estos trabajos de InfoSphere DataStage extraen o cargan los datos reales de los objetos SAP descritos por los metadatos que se han importado hacia o desde archivos sin formato o una base de datos relacional.

  • Seleccionar un tipo de trabajo
    Utilice esta página para seleccionar los tipos de trabajo de IBM® InfoSphere DataStage que se van a generar. Puede generar un trabajo de ABAP Extract, un trabajo de IDoc Extract o el trabajo de IDoc Load utilizando esta página.
  • Detalles de InfoSphere Information Server
    Especifique los detalles del dominio de IBM InfoSphere Information Server y las credenciales de IBM InfoSphere DataStage.
  • Opciones de ABAP Extract
    Utilice esta página para configurar las opciones de ABAP Extract en el diseño de generación del trabajo.
  • Opciones de IDoc Extract
    Utilice esta página para especificar las opciones de origen y destino de un trabajo de IDoc Extract.
  • Opciones de IDoc Load
    Utilice esta página para configurar las opciones de los trabajos de IDoc Load.
  • Definir parámetros de trabajo
    Esta página muestra todos los parámetros de trabajo y los conjuntos de parámetros especificados para su uso en las propiedades de origen y destino. Puede modificar todos los campos de un parámetro, excepto Nombre de parámetro.
  • Definir derivaciones de columnas
    Utilice esta página para derivar las expresiones que se van a aplicar a todas las columnas de un determinado tipo. Estas expresiones se definen en las etapas Transformer de los trabajos generados. Puede hacer referencias cruzadas del argumento de entrada del transformador con $1. Por ejemplo: TrimLeadingTrailing($1) o NullToEmpty($1).
  • Resolver parámetros de trabajo
    Puede especificar valores para los parámetros de trabajo de IBM InfoSphere DataStage. El valor que se especifica aquí se utiliza para todas las instancias de ese parámetro de trabajo en todos los trabajos y todas las etapas
  • Valores del programa ABAP
    Utilice esta página para configurar los valores del programa ABAP para los programas que se van a generar para los trabajos de ABAP Extract de IBM InfoSphere DataStage.
  • Resultados de la validación
    Muestra los resultados de la validación para el valor de SAP configurado. Una marca de selección verde indica que los valores se han validado satisfactoriamente y una marca de selección roja indica un error.
  • Resumen de generación de trabajos
    Muestra el resumen de generación de trabajos y los nombres de los nuevos trabajos. Si ha seleccionado el recuadro de selección Iniciar compilador de varios trabajos de DataStage, se inicia el asistente del compilador de varios trabajos de DataStage después de pulsar Finalizar. Este asistente se abre con los trabajos generados seleccionados para que pueda compilar todos los trabajos generados al mismo tiempo.
