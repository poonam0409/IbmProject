Detalles de InfoSphere Information Server

Especifique los detalles del dominio de IBM® InfoSphere Information Server y las credenciales de IBM InfoSphere DataStage.

Detalles del dominio de Information Server
Dominio
Nombre de sistema principal y número de puerto del servidor IBM InfoSphere DataStage con el formato <nombrehost>:<número de puerto>.
Usuario
Especifique el nombre de usuario de IBM InfoSphere DataStage.
Contraseña
Contraseña del usuario de IBM InfoSphere DataStage.
Detalles del trabajo de DataStage
Proyecto
Nombre del proyecto de IBM InfoSphere DataStage en el que se generan los trabajos.
Carpeta
Especifique el nombre de una carpeta directamente en la carpeta Trabajos en IBM InfoSphere DataStage.
Prefijo de nombre del trabajo
Especifique un prefijo para todos los trabajos creados durante esta ejecución del asistente. El nombre del trabajo se calcula utilizando este prefijo.
Sobrescribir el trabajo si ya existe
Seleccione este recuadro de selección para sobrescribir los trabajos si ya existen en el servidor.
