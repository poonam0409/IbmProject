IBM InfoSphere Rapid Generator for SAP Applications

IBM InfoSphere Rapid Modeler for SAP Applications を使用してインポートした SAP メタデータを使用して InfoSphere™ DataStage® ジョブを 生成できます。これらの InfoSphere DataStage ジョブは、 インポートされたメタデータによって記述された SAP オブジェクトの実際のデータを、リレーショナル・データベース またはフラット・ファイルから抽出したり、それらにロードしたりします。

  • ジョブ・タイプの選択
    このページを使用して、生成する IBM® InfoSphere DataStage ジョブの タイプを選択します。このページでは、ABAP Extract ジョブ、IDoc Extract ジョブ、または IDoc Load ジョブを生成できます。
  • InfoSphere Information Server の詳細
    IBM InfoSphere Information Server ドメインの詳細と IBM InfoSphere DataStage 証明情報を指定します。
  • ABAP Extract オプション
    このページを使用して、ジョブ生成デザインの ABAP Extract オプションを構成します。
  • IDoc Extract オプション
    このページを使用して、IDoc Extract ジョブのソース・オプションとターゲット・オプションを指定します。
  • IDoc Load オプション
    このページを使用して、IDoc Load ジョブのオプションを構成します。
  • ジョブ・パラメーターの定義
    このページには、ターゲット・プロパティーおよびソース・プロパティー用に、ユーザーによって 指定されたすべてのジョブ・パラメーターとパラメーター・セットが表示されます。パラメーターの各フィールドは「パラメーター名」を除いて すべて変更できます。
  • 列の出力仕様を定義
    このページを使用して、特定タイプのすべての列に適用する式を導出します。これらの式は、生成されるジョブの Transformer ステージ内に定義されます。トランスフォーマーの入力引数は $1 を使用して相互参照できます。例えば、TrimLeadingTrailing($1) や NullToEmpty($1) です。
  • ジョブ・パラメーターの解決
    IBM InfoSphere DataStage ジョブ・パラメーターの 値を指定できます。ここで指定する値は、すべてのジョブおよびステージ内で、その ジョブ・パラメーターのすべてのインスタンスに使用されます。
  • ABAP プログラムの設定
    このページを使用して、IBM InfoSphere DataStage ABAP Extract ジョブ用に生成されるプログラムの ABAP プログラム設定を構成します。
  • 検証結果
    構成済み SAP 設定の検証結果を表示します。 緑のチェック・マークは、設定が正常に検証されたことを表し、赤のチェック・マークは、エラーを表します。
  • ジョブ生成のサマリー
    ジョブ生成のサマリーと新規ジョブの名前が表示されます。「DataStage マルチ・ジョブ・コンパイラーの起動」チェック・ボックスを 選択した場合、「完了」を クリックすると DataStage の「マルチ・ジョブ・コンパイル」ウィザードが開始します。 このウィザードは、生成されたジョブが あらかじめ選択された状態で開始するため、すべての生成されたジョブを一度にコンパイルできます。
