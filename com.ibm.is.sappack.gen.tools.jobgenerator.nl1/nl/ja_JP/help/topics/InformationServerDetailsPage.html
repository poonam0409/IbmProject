InfoSphere Information Server の詳細

IBM® InfoSphere™ Information Server ドメインの詳細と IBM InfoSphere DataStage® 証明情報を指定します。

Information Server ドメインの詳細
ドメイン
IBM InfoSphere DataStage サーバーの ホスト名とポート番号 (<ホスト名>:<ポート番号> の形式)。
ユーザー
IBM InfoSphere DataStage ユーザー名を指定します。
パスワード
IBM InfoSphere DataStage ユーザーのパスワード。
DataStage ジョブの詳細
プロジェクト
ジョブを生成する先の IBM InfoSphere DataStage プロジェクトの名前。
フォルダー
IBM InfoSphere DataStage の「ジョブ」フォルダーの下に 直接含まれるフォルダーの名前を指定します。
ジョブ名接頭部
このウィザードの実行中に作成されるすべてのジョブに使用される接頭部を指定します。 ジョブの名前は、この接頭部を使用して計算されます。
ジョブが既に存在する場合は上書き
このチェック・ボックスを選択して、ジョブが既にサーバー内に存在する場合、ジョブを上書きします。
