IDoc Extract オプション

このページを使用して、IDoc Extract ジョブのソース・オプションとターゲット・オプションを指定します。

注: テキストを # 記号で囲むと、InfoSphere™ DataStage® ジョブ・パラメーターを テキスト・フィールドに使用できます。 パラメーター・セットは構文 #ParameterSetName.ParameterName# を使用して表します。パラメーターおよびパラメーター・セットの詳細については、「IBM® InfoSphere DataStage and QualityStage™ デザイナー・クライアント・ガイド」を参照してください。
ソース・オプション
IDoc Extract ソース・オプションを指定します。
DataStage SAP 接続
IBM InfoSphere DataStage 内に構成されている SAP 接続の名前。
接続のデフォルト値を上書き
IBM InfoSphere DataStage 内に構成されている接続設定を 上書きする場合、このチェック・ボックスを選択します。
SAP ユーザー
SAP ユーザーのユーザー名。
SAP ユーザーのパスワード
指定した SAP ユーザーのパスワード。
SAP ユーザーのクライアント番号
指定した SAP ユーザーのクライアント番号。
SAP ユーザーの言語
指定した SAP ユーザーの言語。
ターゲット・オプション
IDoc Extract の次のターゲット・オプションを指定します。
ステージ・タイプ
生成されるターゲット・ステージのタイプを選択します。
ODBC コネクター
ODBC コネクターに関するオプションを指定します。
ODBC データ・ソース
SAP から抽出したオブジェクトを構成済み ODBC データ・ソース表に保管します。
ODBC ユーザー
データ・ソース構成内に構成されているユーザーを上書きします。 このフィールドはオプションです。
ODBC パスワード
データ・ソース構成内に構成されているパスワードを上書きします。 このフィールドはオプションです。
表アクションの選択
「追加」または「切り捨て」の表のアクションを 選択できます。詳しくは、ODBC コネクター・ステージの資料を参照してください。
順次ファイル
順次ファイルには以下のパラメーターがあります。
ファイル
出力の保管に使用されるすべてのファイルの接頭部として使用する名前を指定します。
ファイル更新モード
可能なオプションは「追加」、「作成 (既存の場合はエラー)」、 または「上書き」です。詳しくは、Sequential File ステージの資料を参照してください。
最初の行は列名
順次ファイルの最初の行に列名を含めるかどうか選択します。
