IDoc Load オプション

このページを使用して、IDoc Load ジョブのオプションを構成します。

注: テキストを # 記号で囲むと、InfoSphere™ DataStage® ジョブ・パラメーターを テキスト・フィールドに使用できます。 パラメーター・セットは構文 #ParameterSetName.ParameterName# を使用して表します。パラメーターおよびパラメーター・セットの詳細については、「IBM® InfoSphere DataStage and QualityStage™ デザイナー・クライアント・ガイド」を参照してください。
ソース・オプション
IDoc Load の以下のソース・オプションを指定します。
ステージ・タイプ
IDoc のソースとして生成されるステージのタイプ。例えば、ODBC コネクターまたは順次ファイル。
ODBC コネクター
ODBC コネクターに関する次のオプションを指定します。
ODBC データ・ソース
SAP にロードするオブジェクトを、オペレーティング・システムに構成されている ODBC データ・ソースの 表から読み取ります。
ODBC ユーザー
データ・ソース構成内に構成されているユーザーを上書きします。 このフィールドはオプションです。
ODBC パスワード
データ・ソース構成内に構成されているパスワードを上書きします。 このフィールドはオプションです。
表アクションの選択
「追加」、「作成」、「置換」、 または「切り捨て」の表のアクションを選択できます。詳しくは、ODBC コネクター・ステージの資料を参照してください。
ODBC SQL WHERE 条件
ソース・データを照会するときに使用する SQL where 節を 指定します。ここで指定する where 節は、照会対象の すべての表に適用されます。通常このオプションは、IBM InfoSphere Data Architect Rapid Modeler for SAP Applications を使用して追加されるテクニカル・フィールドと組み合わせて使用します。
ターゲット・オプション
IDoc Load ステージに関する次のターゲット・オプションを指定します。
SAP メッセージ・タイプ
使用する SAP メッセージ・タイプを指定します。
DataStage SAP 接続
DataStage 内に構成されている SAP 接続の名前。
接続のデフォルト値を上書き
IBM InfoSphere DataStage 内の接続に 定義されている設定を上書きする場合、このチェック・ボックスを選択します。
SAP ユーザー
SAP ユーザーのユーザー ID。
SAP ユーザーのパスワード
指定した SAP ユーザーのパスワード。
SAP ユーザーのクライアント番号
指定した SAP ユーザーのクライアント番号。
SAP ユーザーの言語
指定した SAP ユーザーの言語。
