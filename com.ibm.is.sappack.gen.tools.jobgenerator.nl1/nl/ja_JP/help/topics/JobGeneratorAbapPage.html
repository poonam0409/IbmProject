ABAP Extract オプション

このページを使用して、ジョブ生成デザインの ABAP Extract オプションを構成します。

注: テキストを # 記号で囲むと、InfoSphere™ DataStage® ジョブ・パラメーターを テキスト・フィールドに使用できます。 パラメーター・セットは構文 #ParameterSetName.ParameterName# を使用して表します。パラメーターおよびパラメーター・セットの詳細については、「IBM® InfoSphere DataStage and QualityStage™ デザイナー・クライアント・ガイド」を参照してください。
ソース・オプション
DataStage SAP 接続
DataStage 内に構成されている SAP 接続の名前。
SAP ユーザー
SAP に接続するときに使用する SAP ユーザーの ID。
SAP ユーザーのパスワード
SAP ユーザーのパスワード。
SAP ユーザーのクライアント番号
SAP ユーザーのクライアント番号。
SAP ユーザーの言語
SAP ユーザーの言語。
ターゲット・オプション
ステージ・タイプ
SAP 論理表のターゲットとして生成されるステージのタイプ。 例えば、ODBC コネクターまたは順次ファイル。
ODBC コネクター
ODBC コネクターに関する以下のオプションを指定します。
ODBC データ・ソース
SAP から抽出したオブジェクトを構成済み ODBC データ・ソース表に保管します。
ODBC ユーザー
データ・ソース構成内のユーザーを上書きします。このフィールドはオプションです。
ODBC パスワード
データ・ソース構成内のパスワードを上書きします。このフィールドはオプションです。
表アクションの選択
「追加」または「切り捨て」の表のアクションを 選択できます。詳しくは、ODBC コネクター・ステージの資料を参照してください。
順次ファイル
Sequential File ステージに関する以下のオプションを指定します。
ファイル名接頭部
出力の保管に使用されるすべてのファイルの接頭部。
ファイル更新モード
「追加」、「作成 (既存の場合はエラー)」、または「上書き」を選択できます。詳しくは、Sequential File ステージの資料を参照してください。
最初の行は列名
順次ファイルの最初の行に列名を含めるかどうか選択します。
RFC 転送方式のオプション
RFC 転送方式のオプションを指定します。
SAP Gateway ホスト
SAP サーバーと通信する SAP Gateway オファリング・サービスのホスト名。
SAP Gateway サービス
ゲートウェイ・サービスの名前。
RFC 宛先
各行が、ブランク (スペース) で区切られた 2 つのエントリーで構成される単純なテキスト・ファイル。最初のエントリーは、SAP システムに構成されている RFC 宛先の名前です。2 番目のエントリーは、RFC 宛先の対応するプログラム ID です。
ジョブ内のフローの最大数
ABAP Extract ジョブは、ジョブの数を少なくするために、複数の ABAP ステージを使用して生成できます。しかし、単一 ABAP Extract ジョブ内の各 ABAP ステージは、SAP 内に構成されている固有の RFC 宛先を必要とします。したがって、ABAP ステージの最大数は、提供される RFC 宛先の数によって制限されます。また、これにより 1 ジョブあたりのフローの最大数、または単一 IBM InfoSphere DataStage ジョブ内の ABAP ステージ・インスタンスの最大数が定義され、これらの最大数も、提供される RFC 宛先の数によって制限されます。
RFC 宛先を自動的に作成
実行時に RFC 宛先を IBM InfoSphere DataStage ジョブで作成する場合、このオプションを選択します。
必要に応じて既存の RFC 宛先を削除
必要な場合に RFC 宛先を削除できるようにするには、このオプションを選択します。
SAP 設定の検証
このボタンをクリックして、SAP 接続設定を検証します。
