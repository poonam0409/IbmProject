IBM InfoSphere Rapid Generator for SAP Applications

É possível gerar tarefas do InfoSphere DataStage com os metadados SAP importados usando o IBM InfoSphere Rapid Modeler for SAP Applications. Essas tarefas InfoSphere DataStage extraem ou carregam os dados reais dos objetos SAP descritos pelos metadados que foram importados para/de um banco de dados relacional ou arquivos simples.

  • Selecione um Tipo de Tarefa
    Use esta página para selecionar os tipos de tarefas IBM® InfoSphere DataStage a serem gerados. É possível gerar uma tarefa de extração de ABAP, uma tarefa de extração de IDoc ou a tarefa de carregamento de IDoc usando esta página.
  • Detalhes do InfoSphere Information Server
    Especifique os detalhes de domínio do IBM InfoSphere Information Server e as credenciais do IBM InfoSphere DataStage.
  • Opções de Extração do ABAP
    Use esta página para configurar as opções de extração do ABAP no design de geração de tarefa.
  • Opções de Extração do IDoc
    Use essa página para especificar a opção de origem e de destino para uma tarefa de extração de IDoc.
  • Opções de Carregamento de IDoc
    Use essa página para configurar as opções de tarefas de Carregamento de IDoc.
  • Definir Parâmetros da Tarefa
    Essa página exibe todos os parâmetros e conjuntos de parâmetros de tarefa especificados pelo usuário para uso pelas propriedades de destino e de origem. É possível modificar todos os campos para um parâmetro, exceto Nome do parâmetro.
  • Definir derivações de coluna
    Use esta página para derivar expressões a serem aplicadas a todas as colunas de um tipo particular. Essas expressões são definidas nos estágios Transformer das tarefas geradas. É possível estabelecer uma referência cruzada no argumento de entrada do transformador com $1. Alguns exemplos são:TrimLeadingTrailing($1) ou NullToEmpty($1).
  • Resolver Parâmetros de Tarefa
    É possível especificar valores para os parâmetros de tarefa do IBM InfoSphere DataStage. O valor especificado aqui é usado para todas as instâncias desse parâmetro de tarefa dentro de todas as tarefas e estágios
  • Configurações do Programa ABAP
    Use esta pagina para definir as configurações do programa ABAP para os programas a serem gerados para as tarefas de extração do ABAPIBM InfoSphere DataStage.
  • Resultados da Validação
    Exibe os resultados da validação para a configuração SAP definida. Uma marca de seleção verde indica que as configurações são validadas com êxito, e uma marca de seleção vermelha indica um erro.
  • Resumo da Geração de Tarefa
    Exibe o resumo da geração de tarefa e os nomes das novas tarefas. Se você tiver marcado a caixa de seleção Iniciar Compilador de Diversas Tarefas do DataStage, o assistente Compilador de Diversas Tarefas do DataStage será iniciado depois de clicar em Concluído. Esse assistente abre com as tarefas geradas já selecionadas para que seja possível compilar todas as tarefas geradas ao mesmo tempo.
