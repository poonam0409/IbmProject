Selecione um Tipo de Tarefa

Use esta página para selecionar os tipos de tarefas IBM® InfoSphere DataStage a serem gerados. É possível gerar uma tarefa de extração de ABAP, uma tarefa de extração de IDoc ou a tarefa de carregamento de IDoc usando esta página.

Tarefa de Extração de ABAP
Selecione esta opção para gerar tarefas que extraiam tabelas lógicas de um sistema SAP. Esta opção está disponível quando os metadados da tabela lógica SAP são importados para o modelo de dados lógicos explicitamente ou selecionando-se as opções de tabela de verificação para IDocs.
Tarefa de Extração de IDoc
Selecione esta opção para criar tarefas que coletem IDocs que foram previamente enviados ao serviço de listener do IDoc configurado no servidorInfoSphere DataStage.
Tarefa de Carregamento de IDoc
Selecione esta opção para gerar uma tarefa que leia IDocs e os envie ao SAP.
