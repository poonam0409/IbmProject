Opções de Carregamento de IDoc

Use essa página para configurar as opções de tarefas de Carregamento de IDoc.

Nota: É possível usar os parâmetros da tarefa InfoSphere DataStage para os campos de texto que incluem o texto com o símbolo "#". Os conjuntos de parâmetros são indicados com a sintaxe "#" #ParameterSetName.ParameterName#. Para obter informações adicionais sobre parâmetros e conjuntos de parâmetros, consulte o IBM® InfoSphere DataStage e QualityStage: Guia do Cliente do Designer
Opções de origem
Especifique as seguintes opções de origem para carregamento de IDoc:
Tipo de Estágio
Tipos de estágios gerados como origem dos IDocs. Por exemplo, o Conector ou o Arquivo Sequencial ODBC.
Conector ODBC
Especifique as seguintes opções para o conector ODBC:
Origem de dados do ODBC
Os objetos carregados no SAP são lidos nas tabelas da origem de dados do ODBC configuradas nos sistemas operacionais.
Usuário ODBC
Sobrescreve o usuário configurado na configuração de origem de dados. É um campo opcional.
Senha ODBC
Sobrescreve a senha configurada na configuração de origem de dados. É um campo opcional.
Selecione a ação da tabela
É possível selecionar as ações de tabela APPEND, CREATE, REPLACE ou TRUNCATE. Para obter informações adicionais, consulte a documentação do estágio do conector do ODBC.
Condição SQL WHERE de ODBC
Especifique uma cláusula where SQL a ser usada durante a consulta de origem de dados. A cláusula where especificada aqui é aplicada a todas as tabelas que estão sendo consultadas. Essa opção geralmente é usada em conjunto com os campos técnicos incluídos por meio do IBM InfoSphere Data Architect Rapid Modeler para aplicativos SAP.
Opções de destino
Especifique as seguintes opções de destino para estágio de carregamento de IDoc:
Tipo de Mensagem do SAP
Especifique o tipo de mensagem do SAP a ser usado.
Conexão do DataStage SAP
Nome da conexão SAP configurada no DataStage.
Sobrescrever padrões da conexão
Marque essa caixa de seleção se você quiser sobrescrever as configurações definidas na conexão no IBM InfoSphere DataStage.
Usuário SAP
Id do usuário SAP.
Senha do Usuário SAP
Senha para o usuário SAP especificado.
Número de Cliente do Usuário de SAP
Número de cliente do usuário SAP especificado.
Idioma do Usuário SAP
Idioma do usuário SAP especificado.
