Opções de Extração do IDoc

Use essa página para especificar a opção de origem e de destino para uma tarefa de extração de IDoc.

Nota: É possível usar os parâmetros da tarefa InfoSphere DataStage para os campos de texto que incluem o texto com o símbolo "#". Os conjuntos de parâmetros são indicados com a sintaxe "#" #ParameterSetName.ParameterName#. Para obter informações adicionais sobre parâmetros e conjuntos de parâmetros, consulte o IBM® InfoSphere DataStage e QualityStage: Guia do Cliente do Designer
Opções de Origem
Especifique as opções de origem de extração do IDoc:
Conexão do DataStage SAP
Nome da conexão SAP configurada no IBM InfoSphere DataStage.
Sobrescrever padrões da conexão
Marque essa caixa de seleção se você quiser sobrescrever as configurações de conexão definidas no IBM InfoSphere DataStage.
Usuário SAP
Nome do usuário SAP.
Senha do Usuário SAP
Senha para o usuário SAP especificado.
Número de Cliente do Usuário de SAP
Número de cliente do usuário SAP especificado.
Idioma do Usuário SAP
Idioma do usuário SAP especificado.
Opções de destino
Especifique as seguintes opções de destino para a extração de IDoc:
Tipo de Estágio
Selecione o tipo de estágios de destino gerados:
Conector ODBC
Especifique as opções para o conector ODBC:
Origem de dados do ODBC
O objeto extraído do SAP é armazenado na tabela de origem de dados do ODBC configurado.
Usuário ODBC
Sobrescreve o usuário configurado na configuração de origem de dados. É um campo opcional.
Senha ODBC
Sobrescreve a senha configurada na configuração de origem de dados. É um campo opcional.
Selecione a ação da tabela
É possível selecionar as ações da tabela APPEND ou TRUNCATE. Para obter informações adicionais, consulte a documentação do estágio do conector do ODBC.
Arquivo Sequencial
O arquivo sequencial possui estes parâmetros:
Arquivo
Especifique um nome a ser usado como prefixo para todos os arquivos usados para armazenar a saída.
Modo de Atualização de Arquivo
As possíveis opções são APPEND, CREATE (Erro se existir) ou OVERWRITE. Para obter informações adicionais, consulte a documentação do estágio de arquivo sequencial.
A Primeira Linha são os nomes das colunas
Selecione se você quiser que a primeira linha no arquivo sequencial contenha os nomes de colunas.
