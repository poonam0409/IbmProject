Detalhes do InfoSphere Information Server

Especifique os detalhes de domínio do IBM® InfoSphere Information Server e as credenciais do IBM InfoSphere DataStage.

Detalhes do domínio do Information Server
Domínio
Nome do host e número da porta do servidor do IBM InfoSphere DataStage no formato <nome do host>:<número da porta>.
Usuário
Especifique o nome de usuário do IBM InfoSphere DataStage.
Senha
Senha do usuário do IBM InfoSphere DataStage.
Detalhes de tarefa do DataStage
Projeto
Nome do projeto do IBM InfoSphere DataStage no qual as tarefas são geradas.
Pasta
Especifique o nome de uma pasta diretamente na pasta Tarefas no IBM InfoSphere DataStage.
Prefixo de nome da tarefa
Especifique um prefixo para todas as tarefas criadas durante essa execução do assistente. O nome da tarefa é calculado usando esse prefixo.
Sobrescrever a tarefa se ele já existir
Marque essa caixa de seleção para sobrescrever as tarefas se elas já existirem no servidor.
