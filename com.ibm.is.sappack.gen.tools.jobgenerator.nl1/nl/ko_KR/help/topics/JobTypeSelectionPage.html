작업 유형 선택

이 페이지에서 생성할 IBM® InfoSphere™ DataStage® 작업의 유형을 선택할 수 있습니다. 이 페이지를 이용해 ABAP 추출 작업, IDoc 추출 작업 또는 IDoc 로드 작업을 생성할 수 있습니다.

ABAP 추출 작업
SAP 시스템에서 논리 테이블을 추출하는 작업을 생성하려면 이 옵션을 선택하십시오. 명시적으로 또는 IDoc에 대한 검사 테이블 옵션을 선택하여 논리 데이터 모델로 SAP 논리 테이블 메타데이터를 가져올 때 이 옵션을 사용할 수 있습니다.
IDoc 추출 작업
InfoSphere DataStage 서버에서 구성된 IDoc 리스너 서비스로 이전에 보낸 IDoc를 수집하는 작업을 작성하려면 이 옵션을 선택하십시오.
IDoc 로드 작업
IDoc를 읽고 이를 SAP으로 보내는 작업을 생성하려면 이 옵션을 선택하십시오.
상위 주제: IBM InfoSphere Rapid Generator for SAP Applications
관련 태스크
Rapid Generator for SAP을 사용하여 InfoSphere DataStage 작업 생성
ABAP 추출 작업용 작업 생성 구성
IDoc 추출 작업을 위한 작업 생성 구성
IDoc 로드 작업용 작업 생성 구성
