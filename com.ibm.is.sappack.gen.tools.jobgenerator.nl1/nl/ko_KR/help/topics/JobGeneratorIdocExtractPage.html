IDoc 추출 옵션

이 페이지에서 IDoc 추출 작업에 대한 소스 및 대상 옵션을 지정할 수 있습니다.

주: 텍스트를 "#" 기호로 둘러싸 텍스트 필드에 대해 InfoSphere™ DataStage® 작업 매개변수를 사용할 수 있습니다. 매개변수 세트는 "#" #ParameterSetName.ParameterName# 구문으로 표시됩니다. 매개변수 및 매개변수 세트에 대한 자세한 정보는 IBM® InfoSphere DataStage and QualityStage™ 디자이너 클라이언트 안내서의 내용을 참조하십시오.
소스 옵션
IDoc 추출 소스 옵션을 지정하십시오.
DataStage SAP 연결
IBM InfoSphere DataStage에서 구성된 SAP 구성의 이름입니다.
연결 기본값 겹쳐쓰기
IBM InfoSphere DataStage에서 구성된 연결 설정을 겹쳐쓰려면 이 선택란을 선택하십시오.
SAP 사용자
SAP 사용자의 사용자 이름입니다.
SAP 사용자 비밀번호
지정된 SAP 사용자에 대한 비밀번호입니다.
SAP 사용자 클라이언트 번호
지정된 SAP 사용자의 클라이언트 번호입니다.
SAP 사용자 언어
지정된 SAP 사용자의 언어입니다.
대상 옵션
IDoc 추출을 위한 다음 대상 옵션을 지정하십시오.
스테이지 유형
생성되는 대상 스테이지의 유형을 선택하십시오.
ODBC Connector
ODBC Connector를 위한 옵션을 지정하십시오.
ODBC 데이터 소스
SAP에서 추출된 오브젝트는 구성된 ODBC 데이터 소스 테이블에 저장됩니다.
ODBC 사용자
데이터 소스 구성에서 구성된 사용자를 겹쳐씁니다. 이는 선택적 필드입니다.
ODBC 비밀번호
데이터 소스 구성에서 구성된 비밀번호를 겹쳐씁니다. 이는 선택적 필드입니다.
테이블 조치 선택
APPEND 또는 TRUNCATE 테이블 조치를 선택할 수 있습니다. 자세한 정보는 ODBC Connector 스테이지 문서를 참조하십시오.
순차 파일
순차 파일에는 다음과 같은 매개변수가 있습니다.
파일
출력을 저장하는 데 사용되는 모든 파일의 접두부로 사용할 이름을 지정합니다.
파일 업데이트 모드
가능한 옵션은 APPEND, CREATE(존재하는 경우 오류) 또는 OVERWRITE입니다. 자세한 정보는 순차 파일 스테이지 문서를 참조하십시오.
첫 번째 행은 열 이름
순차 파일의 첫 행에 열 이름을 포함하려는 경우 선택하십시오.
