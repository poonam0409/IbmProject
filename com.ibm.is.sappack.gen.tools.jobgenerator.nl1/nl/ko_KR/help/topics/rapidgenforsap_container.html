IBM InfoSphere Rapid Generator for SAP Applications

IBM InfoSphere Rapid Generator for SAP Applications을 사용하여 가져온 SAP 메타데이터로 InfoSphere™ DataStage® 작업을 생성할 수 있습니다. 이런 InfoSphere DataStage 작업은 관계형 데이터베이스 또는 플랫 파일 간에 가져온 메타데이터에 의해 설명되는 SAP 오브젝트의 실제 데이터를 추출하거나 로드합니다.

  • 작업 유형 선택
    이 페이지에서 생성할 IBM® InfoSphere DataStage 작업의 유형을 선택할 수 있습니다. 이 페이지를 이용해 ABAP 추출 작업, IDoc 추출 작업 또는 IDoc 로드 작업을 생성할 수 있습니다.
  • InfoSphere Information Server 세부사항
    IBM InfoSphere Information Server 도메인 세부사항과 IBM InfoSphere DataStage 신임 정보를 지정하십시오.
  • ABAP 추출 옵션
    이 페이지에서는 작업 생성 디자인에서 ABAP 추출 옵션을 구성할 수 있습니다.
  • IDoc 추출 옵션
    이 페이지에서 IDoc 추출 작업에 대한 소스 및 대상 옵션을 지정할 수 있습니다.
  • IDoc 로드 옵션
    이 페이지에서는 IDoc 로드 작업을 위한 옵션을 구성할 수 있습니다.
  • 작업 매개변수 정의
    이 페이지에는 대상 및 소스 특성에서 사용하도록 사용자가 지정하는 모든 작업 매개변수 및 매개변수 세트가 표시됩니다. 매개변수 이름을 제외하고, 매개변수에 대한 모든 필드를 수정할 수 있습니다.
  • 열 파생 정의
    이 페이지에서 특정 유형의 모든 열에 적용할 표현식을 파생할 수 있습니다. 이런 표현식은 생성된 작업의 Transformer 스테이지 내에 정의됩니다. Transformer의 입력 인수를 $1와 상호 참조할 수 있습니다. TrimLeadingTrailing($1) 또는 NullToEmpty($1)을 예로 들 수 있습니다.
  • 작업 매개변수 해결
    IBM InfoSphere DataStage 작업 매개변수에 대한 값을 지정할 수 있습니다. 여기서 지정된 값은 모든 작업과 스테이지 내에서 그 작업 매개변수의 모든 인스턴스에 사용됨
  • ABAP 프로그램 설정
    이 페이지에서는 프로그램이 IBM InfoSphere DataStage ABAP 추출 작업용으로 생성되도록 ABAP 프로그램 설정을 구성할 수 있습니다.
  • 유효성 검증 결과
    구성된 SAP 설정에 대한 유효성 검증 결과를 표시합니다. 녹색 확인 표시는 설정이 올바로 유효성 검증됨을 표시하고 빨간색 확인 표시는 오류를 표시합니다.
  • 작업 생성 요약
    작업 생성 요약과 새 작업의 이름을 표시합니다. DataStage 다중 작업 컴파일러 시작 선택란을 선택한 경우 완료를 클릭한 후 DataStage의 다중 작업 컴파일러 마법사가 시작됩니다. 이 마법사는 생성된 모든 작업을 동시에 컴파일할 수 있도록 생성된 작업을 이미 선택한 상태로 열립니다.
