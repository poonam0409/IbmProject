ABAP-Extraktionsoptionen

Auf dieser Seite können Sie die ABAP-Extraktionsoptionen im Jobgenerierungsentwurf konfigurieren.

Anmerkung: Sie können InfoSphere DataStage-Jobparameter für Textfelder verwenden, indem Sie den Text in das Symbol "#" einschließen. Für die Angabe der Parametersätze gilt die Syntax #Parametersatzname.Parametername#. Weitere Informationen zu Parametern und Parametersätzen finden Sie in IBM® InfoSphere DataStage and QualityStage Designer Client Guide.
Quellenoptionen
DataStage-SAP-Verbindung
Name der in DataStage konfigurierten SAP-Verbindung.
SAP-Benutzer
ID des SAP-Benutzers, über den eine Verbindung zu SAP hergestellt werden soll.
Kennwort des SAP-Benutzers
Kennwort des SAP-Benutzers.
Nummer des SAP-Benutzerclients
Clientnummer des SAP-Benutzers.
Sprache des SAP-Benutzers
Sprache des SAP-Benutzers.
Zieloptionen
Stagetyp
Typ der als Ziel der logischen SAP-Tabellen generierten Stages. Beispielsweise ODBC-Connector oder Sequenzielle Datei.
ODBC-Connector
Geben Sie diese Optionen für den ODBC-Connector an:
ODBC-Datenquelle
Das aus SAP extrahierte Objekt wird in der ODBC-Datenquellentabelle konfiguriert und gespeichert.
ODBC-Benutzer
Überschreibt den in der Datenquellenkonfiguration angegebenen Benutzer. Eine Eingabe in diesem Feld ist nicht unbedingt erforderlich.
ODBC-Kennwort
Überschreibt das in der Datenquellenkonfiguration angegebenen Kennwort. Eine Eingabe in diesem Feld ist nicht unbedingt erforderlich.
Tabellenaktion auswählen
Sie können die Tabellenaktion APPEND (Anhängen) oder TRUNCATE (Abschneiden) auswählen. Weitere Informationen finden Sie in der Dokumentation zur Stage ODBC Connector.
Sequenzielle Datei
Geben Sie diese Optionen für die Stage Sequential file (Sequenzielle Datei) an:
Dateinamenpräfix
Präfix für alle Dateien, die zum Speichern von Ausgaben verwendet werden.
Dateiaktualisierungsmodus
Sie können APPEND (Anhängen), CREATE (Error if exists) (Erstellen (Fehler wenn vorhanden)) oder OVERWRITE (Überschreiben) auswählen. Weitere Informationen finden Sie in der Dokumentation zur Stage Sequential File (Sequenzielle Datei).
Spaltennamen in erster Zeile
Wählen Sie diese Option aus, wenn die erste Zeile in der sequenziellen Datei die Spaltennamen enthalten soll.
RFC-Übertragungsmethode - Optionen
Geben Sie die Optionen für die RFC-Übertragungsmethode an:
SAP-Gateway-Host
Hostname der SAP-Gateway-Services für die Kommunikation mit dem SAP-Server.
SAP-Gateway-Service
Name des Gateway-Service.
RFC-Ziele
Eine einfache Textdatei, in der jede Zeile aus zwei durch Leerzeichen voneinander getrennten Einträgen besteht. Der erste Eintrag ist der Name des auf dem SAP-System konfigurierten RFC-Ziels. Der zweite Eintrag ist die entsprechende Programm-ID des RFC-Ziels.
Max. Anzahl Datenflüsse in einem Job
ABAP-Extraktionsjobs können mit mehreren ABAP-Stages generiert werden, um die Anzahl der Jobs zu reduzieren. In einem ABAP-Extraktionsjob muss jedoch für jede ABAP-Stage ein eindeutiges RFC-Ziel in SAP konfiguriert werden. Daher ist die maximale Anzahl der ABAP-Stages begrenzt durch die Anzahl der bereitgestellten RFC-Ziele. Außerdem werden die maximale Anzahl der Datenflüsse pro Job definiert oder die ABAP-Stage-Instanzen innerhalb eines einzelnen IBM InfoSphere DataStage-Jobs, die begrenzt sind auf die Anzahl bereitgestellter RFC-Ziele.
RFC-Ziel automatisch erstellen
Wählen Sie diese Option aus, wenn das RFC-Ziel während der Ausführung von dem IBM InfoSphere DataStage-Job erstellt werden soll.
Vorhandenes RFC-Ziel löschen, falls erforderlich
Wählen Sie diese Option aus, um das Löschen von RFC-Zielen bei Bedarf zuzulassen.
SAP-Einstellungen prüfen
Klicken Sie auf diese Schaltfläche, um die SAP-Verbindungseinstellungen zu prüfen.
