IDoc-Ladeoptionen

Auf dieser Seite können Sie die Optionen für IDoc-Ladejobs konfigurieren.

Anmerkung: Sie können InfoSphere DataStage-Jobparameter für Textfelder verwenden, indem Sie den Text in das Symbol "#" einschließen. Für die Angabe der Parametersätze gilt die Syntax #Parametersatzname.Parametername#. Weitere Informationen zu Parametern und Parametersätzen finden Sie in IBM® InfoSphere DataStage and QualityStage Designer Client Guide.
Quellenoptionen
Geben Sie die folgenden Quellenoptionen für IDoc-Ladejobs an:
Stagetyp
Typ der als Quelle der IDocs generierten Stages. Beispielsweise ODBC-Connector oder Sequenzielle Datei.
ODBC-Connector
Geben Sie die folgenden Optionen für den ODBC-Connector an:
ODBC-Datenquelle
Die in SAP geladenen Objekte werden aus den Tabellen gelesen, die von der ODBC-Datenquelle im Betriebssystem konfiguriert wurden.
ODBC-Benutzer
Überschreibt den in der Datenquellenkonfiguration angegebenen Benutzer. Dies ist ein optionales Feld.
ODBC-Kennwort
Überschreibt das in der Datenquellenkonfiguration angegebenen Kennwort. Dies ist ein optionales Feld.
Tabellenaktion auswählen
Sie können die Tabellenaktionen APPEND (Anhängen), CREATE (Erstellen), REPLACE (Ersetzen) oder TRUNCATE (Abschneiden) auswählen. Weitere Informationen finden Sie in der Dokumentation zur Stage ODBC Connector.
ODBC-SQL-WHERE-Bedingung
Geben Sie eine SQL-Klausel where an, die bei der Abfrage von Quellendaten verwendet werden soll. Die hier angegebene Klausel where wird auf alle abgefragten Tabellen angewendet. Diese Option wird normalerweise in Verbindung mit technischen Feldern verwendet, die über die Anwendung IBM InfoSphere Data Architect Rapid Modeler for SAP hinzugefügt werden.
Zieloptionen
Geben Sie die folgenden Zieloptionen für IDoc-Ladestage an:
SAP-Nachrichtentyp
Geben Sie den zu verwendenden SAP-Nachrichtentyp an.
DataStage-SAP-Verbindung
Name der in DataStage konfigurierten SAP-Verbindung.
Verbindungsstandardwerte überschreiben
Wählen Sie dieses Kontrollkästchen aus, wenn Sie die in der Verbindung in IBM InfoSphere DataStage definierten Einstellungen überschreiben wollen.
SAP-Benutzer
Benutzer-ID des SAP-Benutzers.
Kennwort des SAP-Benutzers
Kennwort für den angegebenen SAP-Benutzer.
Nummer des SAP-Benutzerclients
Clientnummer des angegebenen SAP-Benutzers.
Sprache des SAP-Benutzers
Sprache des angegebenen SAP-Benutzers.
