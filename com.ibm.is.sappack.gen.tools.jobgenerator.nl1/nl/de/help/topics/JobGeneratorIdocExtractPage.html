IDoc-Extraktionsoptionen

Auf dieser Seite können Sie die Quellen- und Zieloptionen für einen IDoc-Extraktionsjob angeben.

Anmerkung: Sie können InfoSphere DataStage-Jobparameter für Textfelder verwenden, indem Sie den Text in das Symbol "#" einschließen. Für die Angabe der Parametersätze gilt die Syntax #Parametersatzname.Parametername#. Weitere Informationen zu Parametern und Parametersätzen finden Sie in IBM® InfoSphere DataStage and QualityStage Designer Client Guide.
Quellenoptionen
Geben Sie die Quellenoptionen für IDoc-Extraktion an:
DataStage-SAP-Verbindung
Name der in IBM InfoSphere DataStage konfigurierten SAP-Verbindung.
Verbindungsstandardwerte überschreiben
Wählen Sie dieses Kontrollkästchen aus, wenn Sie die in IBM InfoSphere DataStage konfigurierten Verbindungseinstellungen überschreiben wollen.
SAP-Benutzer
Benutzername des SAP-Benutzers.
Kennwort des SAP-Benutzers
Kennwort für den angegebenen SAP-Benutzer.
Nummer des SAP-Benutzerclients
Clientnummer des angegebenen SAP-Benutzers.
Sprache des SAP-Benutzers
Sprache des angegebenen SAP-Benutzers.
Zieloptionen
Geben Sie die folgenden Zieloptionen für IDoc-Extraktion an:
Stagetyp
Wählen Sie den Typ der generierten Zielstages aus:
ODBC-Connector
Geben Sie die Optionen für den ODBC-Connector an:
ODBC-Datenquelle
Das aus SAP extrahierte Objekt wird in der konfigurierten ODBC-Datenquellentabelle gespeichert.
ODBC-Benutzer
Überschreibt den in der Datenquellenkonfiguration angegebenen Benutzer. Dies ist ein optionales Feld.
ODBC-Kennwort
Überschreibt das in der Datenquellenkonfiguration angegebenen Kennwort. Dies ist ein optionales Feld.
Tabellenaktion auswählen
Sie können die Tabellenaktion APPEND (Anhängen) oder TRUNCATE (Abschneiden) auswählen. Weitere Informationen finden Sie in der Dokumentation zur Stage ODBC Connector.
Sequenzielle Datei
Die sequenzielle Datei hat die folgenden Parameter:
Datei
Geben Sie einen Namen an, der als Präfix für alle Dateien verwendet werden soll, die zum Speichern von Ausgaben verwendet werden.
Dateiaktualisierungsmodus
Mögliche Optionen sind APPEND (Anhängen), CREATE (Error if exists) (Erstellen (Fehler wenn vorhanden)) oder OVERWRITE (Überschreiben). Weitere Informationen finden Sie in der Dokumentation zur Stage Sequential File (Sequenzielle Datei).
Spaltennamen in erster Zeile
Wählen Sie diese Option aus, wenn die erste Zeile in der sequenziellen Datei die Spaltennamen enthalten soll.
