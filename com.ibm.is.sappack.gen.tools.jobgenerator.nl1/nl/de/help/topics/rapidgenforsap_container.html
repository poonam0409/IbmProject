IBM InfoSphere Rapid Generator for SAP Applications

Sie können InfoSphere DataStage-Jobs mit den SAP-Metadaten generieren, die mit IBM InfoSphere Rapid Modeler for SAP Applications importiert wurden. Diese InfoSphere DataStage-Jobs extrahieren oder laden die eigentlichen Daten der SAP-Objekte, die von den Metadaten beschrieben werden, die in eine relationale Datenbank oder unstrukturierte Dateien bzw. aus einer relationalen Datenbank oder unstrukturierten Dateien importiert wurden.

  • Jobtyp auswählen
    Auf dieser Seite können Sie die zu generierenden Typen von IBM® InfoSphere DataStage-Jobs auswählen. Sie können einen ABAP-Extraktionsjob, einen IDoc-Extraktionsjob oder den IDoc-Ladejob mittels dieser Seite generieren.
  • InfoSphere Information Server-Details
    Geben Sie die IBM InfoSphere Information Server-Domänendetails und IBM InfoSphere DataStage-Berechtigungsnachweise an.
  • ABAP-Extraktionsoptionen
    Auf dieser Seite können Sie die ABAP-Extraktionsoptionen im Jobgenerierungsentwurf konfigurieren.
  • IDoc-Extraktionsoptionen
    Auf dieser Seite können Sie die Quellen- und Zieloptionen für einen IDoc-Extraktionsjob angeben.
  • IDoc-Ladeoptionen
    Auf dieser Seite können Sie die Optionen für IDoc-Ladejobs konfigurieren.
  • Jobparameter definieren
    Auf dieser Seite werden alle Jobparameter und Parametersätze angezeigt, die vom Benutzer zur Verwendung durch Ziel- und Quelleneigenschaften angegeben wurden. Sie können alle Felder eines Parameters ändern mit Ausnahme von Parametername.
  • Spaltenableitungen definieren
    Auf dieser Seite können Sie Ausdrücke ableiten, die auf alle Spalten eines bestimmten Typs angewendet werden sollen. Diese Ausdrücke werden in den Stages Transformer der generierten Jobs definiert. Mit $1 können Sie Querverweise auf das Eingabeargument der Umsetzung angeben. Beispiele: TrimLeadingTrailing($1) oder NullToEmpty($1).
  • Jobparameter auflösen
    Sie können für Jobparameter von IBM InfoSphere DataStage Werte angeben. Der hier angegebene Wert wird für alle Instanzen des betreffenden Jobparameters in allen Jobs und Stages verwendet.
  • ABAP-Programmeinstellungen
    Auf dieser Seite können Sie die ABAP-Programmeinstellungen für Programme konfigurieren, die für die IBM InfoSphere DataStage-ABAP-Extraktionsjobs generiert werden sollen.
  • Prüfungsergebnisse
    Zeigt Prüfungsergebnisse für die konfigurierte SAP-Einstellung an. Ein grüner Haken weist auf die erfolgreiche Prüfung der Einstellungen hin und ein roter Haken weist auf einen Fehler hin.
  • Zusammenfassung der Jobgenerierung
    Zeigt die Zusammenfassung der Jobgenerierung und die Namen der neuen Jobs an. Wenn Sie das Kontrollkästchen Compiler für mehrere DataStage-Jobs starten ausgewählt haben, wird der DataStage-Assistent zum Kompilieren mehrerer Jobs gestartet, nachdem Sie auf Fertig stellen geklickt haben. Wenn dieser Assistent geöffnet wird, sind die generierten Jobs bereits ausgewählt, sodass Sie alle generierten Jobs gleichzeitig kompilieren können.
