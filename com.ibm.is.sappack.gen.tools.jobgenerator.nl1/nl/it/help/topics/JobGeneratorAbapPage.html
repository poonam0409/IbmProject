Opzioni di estrazione ABAP

Utilizzare questa pagina per configurare le opzioni di estrazione ABAP nella progettazione della generazione del job.

Nota: È possibile utilizzare i parametri del job InfoSphere DataStage per il campi di testo, racchiudendo il testo tra il simbolo "#". Le serie di parametri sono indicate dalla sintassi "#" #ParameterSetName.ParameterName#. Per ulteriori informazioni sui parametri e sulle serie di parametri, vedere IBM® InfoSphere DataStage and QualityStage Designer Client Guide
Opzioni origine
Connessione DataStage SAP
Nome della connessione SAP configurata in DataStage.
Utente SAP
ID dell'utente SAP da utilizzare per connettersi a SAP.
Password utente SAP
Password dell'utente SAP.
Numero client utente SAP
Il numero cliente dell'utente SAP.
Lingua utente SAP
La lingua dell'utente SAP.
Opzioni di destinazione
Tipo di stage
Tipi di stage generati come destinazione delle tabelle logiche SAP. Ad esempio, ODBC Connector o Sequential File.
ODBC Connector
Specificare le seguenti opzioni per ODBC connector:
Origine dati ODBC
L'oggetto estratto da SAP viene archiviato in una tabella di origine dati ODBC configurata.
Utente ODBC
Sovrascrive l'utente nella configurazione dell'origine dati. Questo campo è facoltativo.
Password ODBC
Sovrascrive la password nella configurazione dell'origine dati. Questo campo è facoltativo.
Selezionare l'azione tabella
È possibile selezionare le azioni tabella APPEND o TRUNCATE. Per ulteriori informazioni, vedere la documentazione dello stage ODBC Connector.
File sequenziale
Specificare queste opzioni per lo stage Sequential File:
Prefisso nome file
Prefisso per tutti i file utilizzati per archiviare l'output.
Modalità di aggiornamento file
È possibile selezionare APPEND, CREATE (Errore se esiste) o OVERWRITE. Per ulteriori informazioni, fare riferimento alla documentazione dello stage Sequential File.
Nomi colonna nella prima riga
Selezionare se si desidera che la prima riga nel file sequenziale contenga i nomi colonna
Opzioni del metodo di trasferimento RFC
Specificare le opzioni per il metodo di trasferimento RFC:
Host gateway SAP
Nome host del gateway SAP che offre i servizi di comunicazione con il server SAP.
Servizio Gateway SAP
Nome del servizio gateway.
Destinazioni RFC
Un file di testo semplice in cui ogni linea è composta da due voci separate da uno spazio. La prima voce è il nome della destinazione RFC configurata sul sistema SAP. La seconda voce è l'ID programma corrispondente della destinazione RFC.
Numero massimo di flussi in un job
I job di estrazione ABAP possono essere generati con più stage ABAP per ridurre il numero di job. Tuttavia, in un job di estrazione ABAP, ogni stage ABAP richiede che venga configurata una destinazione RFC univoca in SAP. Di conseguenza, il numero massimo di stage ABAP è limitato dal numero di destinazioni RFC fornite. Inoltre, definisce il numero massimo di flussi per job o istanze dello stage ABAP in un singolo job IBM InfoSphere DataStage limitato al numero di destinazioni RFC fornite.
Crea destinazione RFC automaticamente
Selezionare questa opzione se la destinazione RFC deve essere creata al runtime dal job IBM InfoSphere DataStage.
Elimina destinazione RFC se necessario
Selezionare questa opzione per consentire l'eliminazione della destinazione RFC se necessario.
Convalida impostazioni SAP
Fare clic su questo pulsante per convalidare le impostazioni di connessione SAP.
