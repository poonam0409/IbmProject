Opzioni di estrazione IDoc

Utilizzare questa pagina per specificare l'opzione di origine e di destinazione per un job di estrazione IDoc.

Nota: È possibile utilizzare i parametri del job InfoSphere DataStage per il campi di testo, racchiudendo il testo tra il simbolo "#". Le serie di parametri sono indicate dalla sintassi "#" #ParameterSetName.ParameterName#. Per ulteriori informazioni sui parametri e sulle serie di parametri, vedere IBM® InfoSphere DataStage and QualityStage Designer Client Guide
Opzioni origine
Specificare le opzioni di origine per l'estrazione di IDoc:
Connessione DataStage SAP
Nome della connessione a SAP configurata in IBM InfoSphere DataStage.
Sovrascrivi impostazioni predefinite connessione
Selezionare questa casella di spunta se si desidera sovrascrivere le impostazioni di connessione configurate in IBM InfoSphere DataStage.
Utente SAP
Nome utente dell'utente SAP.
Password utente SAP
Password per l'utente SAP specificato SAP.
Numero client utente SAP
Numero client dell'utente SAP specificato.
Lingua utente SAP
La lingua dell'utente SAP specificato.
Opzioni di destinazione
Specificare le seguenti opzioni di destinazione per l'estrazione di IDoc:
Tipo di stage
Selezionare il tipo di stage di destinazione generati:
ODBC Connector
Specificare le seguenti opzioni per ODBC connector:
Origine dati ODBC
L'oggetto estratto da SAP viene archiviato in una tabella di origine dati ODBC configurata.
Utente ODBC
Sovrascrive l'utente configurato nella configurazione dell'origine dati. Questo campo è facoltativo.
Password ODBC
Sovrascrive la password configurata nella configurazione dell'origine dati. Questo campo è facoltativo.
Selezionare l'azione tabella
È possibile selezionare le azioni tabella APPEND o TRUNCATE. Per ulteriori informazioni, vedere la documentazione dello stage ODBC Connector.
File sequenziale
Il file sequenziale ha questi parametri:
File
Specificare un nome da utilizzare come prefisso per tutti i campi utilizzati per archiviare l'output.
Modalità di aggiornamento file
Le opzioni possibili sono APPEND, CREATE (Errore se esiste) o OVERWRITE. Per ulteriori informazioni, fare riferimento alla documentazione dello stage Sequential File.
Nomi colonna nella prima riga
Selezionare se si desidera che la prima riga nel file sequenziale contenga i nomi colonna.
