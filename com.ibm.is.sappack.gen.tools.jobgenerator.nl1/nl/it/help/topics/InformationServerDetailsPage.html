Dettagli su InfoSphere Information Server

Specificare i dettagli sul dominio di IBM® InfoSphere Information Server e le credenziali di IBM InfoSphere DataStage.

Dettagli sul dominio di Information Server
Dominio
Nome host e numero di porta di IBM InfoSphere DataStage Server nel formato <nomehost>:<numero porta>.
Utente
Specificare il nome utente di IBM InfoSphere DataStage.
Password
Password dell'utente di IBM InfoSphere DataStage.
Dettagli sul job DataStage
Progetto
Nome del progetto IBM InfoSphere DataStage in cui verranno generati i job.
Cartella
Specificare il nome di una cartella direttamente nella cartella Jobs di IBM InfoSphere DataStage.
Prefisso nome job
Specificare un prefisso per tutti i job creati durante l'esecuzione di questa procedura guidata. Il nome del job viene elaborato utilizzando questo prefisso.
Sovrascrivi il job se esiste già
Selezionare questa casella di spunta per sovrascrivere i job se esistono già nel server.
