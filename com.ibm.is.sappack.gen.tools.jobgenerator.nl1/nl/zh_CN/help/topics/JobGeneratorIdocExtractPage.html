反馈

IDoc 抽取选项

使用该页面可指定 IDoc 抽取作业的源选项和目标选项。

注： 通过使用“#”符号括起文本，可以对文本字段使用 InfoSphere™ DataStage® 作业参数。使用语法“#”指示参数集：#ParameterSetName.ParameterName#。有关参数和参数集的更多信息，请参阅 IBM® InfoSphere DataStage and QualityStage™ Designer 客户机指南
源选项
指定 IDoc 抽取源选项：
DataStage SAP 连接
IBM InfoSphere DataStage 中配置的 SAP 连接的名称。
覆盖连接缺省值
如果要覆盖 IBM InfoSphere DataStage 中配置的连接设置，请选中该复选框。
SAP 用户
SAP 用户的用户名。
SAP 用户密码
指定的 SAP 用户的密码。
SAP 用户客户机编号
指定的 SAP 用户的客户机编号。
SAP 用户语言
指定的 SAP 用户的语言。
目标选项
指定 IDoc 抽取的以下目标选项：
阶段类型
选择生成的目标阶段的类型：
ODBC 连接器
指定 ODBC 连接器的以下选项：
ODBC 数据源
从 SAP 抽取的对象存储在配置的 ODBC 数据源表中。
ODBC 用户
覆盖数据源配置中所配置的用户。它是可选字段。
ODBC 密码
覆盖数据源配置中所配置的密码。它是可选字段。
选择表操作
您可以选择 APPEND 或 TRUNCATE 表操作。有关更多信息，请参阅 ODBC Connector 阶段文档。
顺序文件
顺序文件包含以下参数：
文件
指定一个名称，此名称将用作用于存储输出的所有文件的前缀。
文件更新方式
可用选项包括 APPEND、CREATE（存在时出错）或 OVERWRITE。有关更多信息，请参阅 Sequential File 阶段文档。
第一行为列名称
如果希望顺序文件中的第一行包含列名称，请选择该选项。
