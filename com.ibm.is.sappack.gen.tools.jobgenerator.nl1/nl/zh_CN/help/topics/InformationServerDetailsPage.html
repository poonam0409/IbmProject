IBM InfoSphere Information Server Pack for SAP Applications V7.0
反馈

InfoSphere Information Server 详细信息

指定 IBM® InfoSphere™ Information Server 域详细信息和 IBM InfoSphere DataStage® 凭证。

Information Server 域详细信息
域
IBM InfoSphere DataStage 服务器的主机名和端口号，格式为：<主机名>:<端口号>。
用户
指定 IBM InfoSphere DataStage 用户名。
密码
IBM InfoSphere DataStage 用户的密码。
DataStage 作业详细信息
项目
生成作业的 IBM InfoSphere DataStage 项目的名称。
文件夹
在 IBM InfoSphere DataStage 的作业文件夹中直接指定文件夹的名称。
作业名前缀
在该向导运行期间，指定创建的所有作业的前缀。使用该前缀计算作业的名称。
已存在时覆盖该作业
选中该复选框可在服务器中已存在作业时覆盖这些作业。
