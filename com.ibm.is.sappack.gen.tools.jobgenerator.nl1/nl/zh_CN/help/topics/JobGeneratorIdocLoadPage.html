IDoc 装入选项

使用该页面可配置 IDoc 装入作业的选项。

注： 通过使用“#”符号括起文本，可以对文本字段使用 InfoSphere™ DataStage® 作业参数。使用语法“#”指示参数集：#ParameterSetName.ParameterName#。有关参数和参数集的更多信息，请参阅 IBM® InfoSphere DataStage and QualityStage™ Designer 客户机指南
源选项
指定 IDoc 装入的以下源选项：
阶段类型
作为 IDoc 源生成的阶段的类型。例如，ODBC Connector 或 Sequential File。
ODBC 连接器
指定 ODBC 连接器的以下选项：
ODBC 数据源
装入 SAP 的对象是从操作系统配置的 ODBC 数据源的表中读取。
ODBC 用户
覆盖数据源配置中所配置的用户。它是可选字段。
ODBC 密码
覆盖数据源配置中所配置的密码。它是可选字段。
选择表操作
您可以选择 APPEND、CREATE、REPLACE 或 TRUNCATE 表操作。 有关更多信息，请参阅 ODBC Connector 阶段文档。
ODBC SQL WHERE 条件
指定在查询源数据时要使用的 SQL where 子句。这里指定的 where 子句将应用于所查询的所有表。该选项通常与通过 IBM InfoSphere Data Architect Rapid Modeler for SAP 应用程序添加的技术字段一起使用。
目标选项
指定 IDoc 装入阶段的以下目标选项：
SAP 消息类型
指定要使用的 SAP 消息类型。
DataStage SAP 连接
DataStage 中配置的 SAP 连接的名称。
覆盖连接缺省值
如果要覆盖 IBM InfoSphere DataStage 的连接中定义的设置，请选中该复选框。
SAP 用户
SAP 用户的用户标识。
SAP 用户密码
指定的 SAP 用户的密码。
SAP 用户客户机编号
指定的 SAP 用户的客户机编号。
SAP 用户语言
指定的 SAP 用户的语言。
