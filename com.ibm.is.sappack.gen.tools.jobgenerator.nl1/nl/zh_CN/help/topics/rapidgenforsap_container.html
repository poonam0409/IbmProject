IBM InfoSphere Rapid Generator for SAP Applications

您可以使用由 IBM InfoSphere Rapid Modeler for SAP Applications 导入的 SAP 元数据来生成 InfoSphere™ DataStage® 作业。这些 InfoSphere DataStage 作业将抽取或装入元数据（导入到关系数据库或平面文件，或者从关系数据库或平面文件导入）描述的 SAP 对象的实际数据。

  • 选择作业类型
    使用该页面可选择要生成的 IBM® InfoSphere DataStage 作业的类型。您可以使用该页面来生成 ABAP 抽取作业、IDoc 抽取作业或 IDoc 装入作业。
  • InfoSphere Information Server 详细信息
    指定 IBM InfoSphere Information Server 域详细信息和 IBM InfoSphere DataStage 凭证。
  • ABAP 抽取选项
    使用该页面可配置作业生成设计中的 ABAP 抽取选项
  • IDoc 抽取选项
    使用该页面可指定 IDoc 抽取作业的源选项和目标选项。
  • IDoc 装入选项
    使用该页面可配置 IDoc 装入作业的选项。
  • 定义作业参数
    该页面显示用户指定的所有作业参数和参数集，由目标属性和源属性使用。您可以修改参数的所有字段，但参数名除外。
  • 定义列派生
    使用该页面可派生要应用于特定类型的所有列的表达式。这些表达式在已生成作业的 Transformer 阶段内定义。您可以使用 $1 交叉引用变换器的输入参数。一些示例如下：TrimLeadingTrailing($1) 或 NullToEmpty($1)。
  • 解析作业参数
    您可以指定 IBM InfoSphere DataStage 作业参数的值。这里指定的值将用于所有作业和阶段中该作业参数的所有实例。
  • ABAP 程序设置
    使用该页面可配置为 IBM InfoSphere DataStage ABAP 抽取作业生成的程序的 ABAP 程序设置。
  • 验证结果
    显示已配置的 SAP 设置的验证结果。绿色的复选标记表示成功验证设置，而红色的复选标记表示错误。
  • 作业生成摘要
    显示作业生成摘要和新作业的名称。如果选中启动 DataStage 多作业编译器复选框，那么在单击完成后，将启动 DataStage 的“多作业编译器”向导。将打开该向导，其中已选中生成的作业，以便您可以同时编译所有生成的作业。
