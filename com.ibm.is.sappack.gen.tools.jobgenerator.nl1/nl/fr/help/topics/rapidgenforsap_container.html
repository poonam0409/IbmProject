IBM InfoSphere Rapid Generator for SAP Applications

Vous pouvez générer des travaux InfoSphere DataStage avec les métadonnées SAP importées à l'aide de IBM InfoSphere Rapid Modeler for SAP applications. Ces travaux InfoSphere DataStage extraient ou charge les données effectives des objets SAP décrits par les métadonnées importées depuis ou vers un base de données relationnelle ou des fichiers à plat.

  • Sélectionner un type de travail
    Cette page permet de sélectionner les types de travail IBM® InfoSphere DataStage à générer. Vous pouvez générer un travail d'extraction ABAP, un travail d'extraction d'IDoc ou un travail de chargement d'IDoc à partir de cette page.
  • Détails d'InfoSphere Information Server
    Indiquez les détails du domaine IBM InfoSphere Information Server et les données d'identification IBM InfoSphere DataStage.
  • Options d'extraction ABAP
    Cette page permet de configurer les options d'extraction ABAP au niveau de la conception de la génération des travaux.
  • Options d'extraction d'IDoc
    Cette page permet de spécifier l'option source et cible d'un travail d'extraction d'IDoc.
  • Options de chargement d'IDoc
    Cette page permet de configurer les options des travaux de chargement d'IDoc.
  • Définir des paramètres de travail
    Cette page affiche tous les paramètres et jeux de paramètres de travail spécifiés par l'utilisateur pour les propriétés cible et source. Toutes les zones d'un paramètre sont modifiables sauf Nom du paramètre.
  • Définir des dérivations de colonne
    Cette page permet de dériver des expressions à appliquer à toutes les colonnes d'un type particulier. Ces expressions sont définies dans les étapes Transformer des travaux générés. Vous pouvez établir une référence croisée pour l'argument d'entrée du transformateur avec $1. Exemples : TrimLeadingTrailing($1) ou NullToEmpty($1).
  • Résolution des paramètres de travail
    Vous pouvez indiquer des valeurs pour les paramètres de travailIBM InfoSphere DataStage. La valeur indiqué ici est utilisée pour toutes les instances du paramètre de travail dans tous les travaux et étapes.
  • Paramètres du programme ABAP
    Cette page permet de configurer les paramètre du programme ABAP pour les programmes à générer pour les travaux d'extraction ABAP deIBM InfoSphere DataStage.
  • Résultats de validation
    Affiche les résultats de la validation des paramètres SAP configurés. Une coche verte signale les paramètres validés et une coche rouge ceux qui présentent une erreur.
  • Récapitulatif de la génération de travaux
    Affiche un récapitulatif du processus de génération de travaux et les noms des nouveaux travaux. Si vous avez coché la case Lancer le compilateur de travaux multiples DataStage, l'assistant Compilateur de travaux multiples de DataStage démarre lorsque vous cliquez sur Terminer. Lorsque cet assistant s'ouvre, les travaux générés sont déjà sélectionnés de sorte que vous pouvez tous les compiler en une seule opération.
