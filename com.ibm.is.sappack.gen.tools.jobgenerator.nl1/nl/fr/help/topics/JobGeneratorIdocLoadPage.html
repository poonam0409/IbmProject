Options de chargement d'IDoc

Cette page permet de configurer les options des travaux de chargement d'IDoc.

Remarque : Vous pouvez utiliser les paramètres du travail InfoSphere DataStage pour les zones de texte en encadrant le texte du symbole "#". Les ensembles de paramètres sont signalés avec la syntaxe "#" #NomEnsembleParamètres.NomParamètre#. Pour plus d'informations sur le paramètres et les ensembles de paramètres, voir IBM® InfoSphere DataStage and QualityStage Designer Client Guide
Options source
Indiquez les options source suivantes pour l'étape de chargement d'IDoc :
Type d'étape
Type des étapes générées comme source des IDoc. Par exemple, ODBC Connector ou Sequential File.
ODBC Connector
Spécifiez les options suivantes pour l'étape ODBC Connector :
Source de données ODBC
Les objets chargés dans le système SAP sont lus à partir des tables de la source de données ODBC configurée dans le système d'exploitation.
Utilisateur ODBC
Remplace l'utilisateur désigné dans la configuration de la source de données. Cette zone est facultative.
Mot de passe ODBC
Remplace le mot de passe indiqué dans la configuration de la source de données. Cette zone est facultative.
Sélectionner l'action de table
Vous pouvez sélectionner l'action de table Ajouter, Créer, Remplacer ou Tronquer. Pour plus d'informations, voir la documentation relative à l'étape ODBC Connector.
Condition WHERE pour l'ODBC SQL
Spécifiez une clause SQL WHERE à utiliser lors d'une requête de source de données. La clause WHERE spécifiée ici s'applique à toutes les tables interrogées. Cette option est généralement utilisée conjointement à des zones techniques ajoutées via l'application Rapid Modeler for SAP de IBM InfoSphere Data Architect.
Options cible
Indiquez les options cibles suivantes pour l'étape de chargement d'IDoc :
Type de message SAP
Indiquez le type de message SAP à utiliser.
Connexion SAP DataStage
Nom de la connexion SAP configurée dans DataStage.
Remplacer les paramètres par défaut de la connexion
Cochez cette case si vous voulez remplacer les paramètres définis pour la connexion dans IBM InfoSphere DataStage.
Utilisateur SAP
Identificateur de l'utilisateur SAP.
Mot de passe utilisateur SAP
Mot de passe de l'utilisateur SAP désigné.
Numéro de client utilisateur SAP
Numéro de client de l'utilisateur SAP désigné.
Langue de l'utilisateur SAP
Langue de l'utilisateur SAP désigné.
