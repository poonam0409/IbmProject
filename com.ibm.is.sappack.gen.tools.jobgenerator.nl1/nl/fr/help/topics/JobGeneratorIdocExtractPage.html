Options d'extraction d'IDoc

Cette page permet de spécifier l'option source et cible d'un travail d'extraction d'IDoc.

Remarque : Vous pouvez utiliser les paramètres du travail InfoSphere DataStage pour les zones de texte en encadrant le texte du symbole "#". Les ensembles de paramètres sont signalés avec la syntaxe "#" #NomEnsembleParamètres.NomParamètre#. Pour plus d'informations sur le paramètres et les ensembles de paramètres, voir IBM® InfoSphere DataStage and QualityStage Designer Client Guide
Options source
Spécifiez les options source d'extraction d'IDoc suivantes :
Connexion SAP DataStage
Nom de la connexion SAP configurée dans IBM InfoSphere DataStage.
Remplacer les paramètres par défaut de la connexion
Cochez cette case si vous voulez remplacer les paramètres de connexion configurés dans IBM InfoSphere DataStage.
Utilisateur SAP
Nom de l'utilisateur SAP.
Mot de passe utilisateur SAP
Mot de passe de l'utilisateur SAP désigné.
Numéro de client utilisateur SAP
Numéro de client de l'utilisateur SAP désigné.
Langue de l'utilisateur SAP
Langue de l'utilisateur SAP désigné.
Options cible
Spécifiez les options cible d'extraction d'IDoc suivantes :
Type d'étape
Sélectionnez le type d'étape cible généré :
ODBC Connector
Spécifiez les options pour l'étape ODBC Connector :
Source de données ODBC
L'objet extrait de SAP est stocké dans la table de source de données ODBC configurée.
Utilisateur ODBC
Remplace l'utilisateur désigné dans la configuration de la source de données. Cette zone est facultative.
Mot de passe ODBC
Remplace le mot de passe indiqué dans la configuration de la source de données. Cette zone est facultative.
Sélectionner l'action de table
Vous pouvez sélectionner l'action de table Ajouter et Tronquer. Pour plus d'informations, voir la documentation relative à l'étape ODBC Connector.
Sequential File
L'étape Sequential File a les paramètres suivants :
Fichier
Indiquez un nom à utiliser comme préfixe pour tous les fichiers utilisés pour stocker la sortie.
Mode de mise à jour du fichier
Les options possibles sont Ajouter, Créer (Erreur s'il existe) ou Remplacer. Pour plus d'informations, voir la documentation relative à l'étape Sequential File.
Première ligne réservée aux noms de colonne
Sélectionnez cette option si vous voulez que la première ligne du fichier séquentiel contienne les noms de colonne.
