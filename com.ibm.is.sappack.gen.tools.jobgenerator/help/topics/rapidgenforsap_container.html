InfoSphere Rapid Generator for SAP Applications

You can generate InfoSphere™ DataStage® jobs with the SAP metadata imported by using InfoSphere Rapid Modeler for SAP applications. These InfoSphere DataStage jobs extract or load the actual data of the SAP objects described by the metadata that was imported to or from a relational database or flat files.

  • Select a job type
    Use this page to select the types of IBM® InfoSphere DataStage job to be generated. You can generate an ABAP extract job, an IDoc extract job, or the IDoc load job by using this page.
  • InfoSphere Information Server Details
    Specify theIBM InfoSphere Information Server domain details and IBM InfoSphere DataStage credentials.
  • ABAP extract options
    Use this page to configure the ABAP extract options in the job generation design.
  • IDoc extract options
    Use this page to specify the source and target option for an IDoc extract job.
  • IDoc load options
    Use this page to configure the options for IDoc Load jobs.
  • Define job parameters
    This page displays all job parameters and parameter sets specified by the user for use by target and source properties. You can modify all fields for a parameter except Parameter name.
  • Define column derivations
    Use this page to derive expressions to be applied to all columns of a particular type. These expressions are defined within the Transformer stages of the generated jobs. You can cross-reference the input argument of the transformer with $1. Some examples are: TrimLeadingTrailing($1) or NullToEmpty($1).
  • Resolve job parameters
    You can specify values for IBM InfoSphere DataStage job parameters. The value specified here is used for all instances of that job parameter within all jobs and stages
  • ABAP program settings
    Use this page to configure the ABAP program settings for the programs to be generated for the IBM InfoSphere DataStage ABAP extract jobs.
  • Validation results
    Displays validation results for the SAP setting configured. A green check mark indicates that the settings are validated successfully and red check mark indicates an error.
  • Job generation summary
    Displays the job generation summary and the names of the new jobs. If you selected the Launch DataStage Multiple Job Compiler check box, the Multiple Job Compiler wizard of DataStage is started after clicking Finish. This wizard opens with the generated jobs already selected so that you can compile all generated jobs at the same time.
