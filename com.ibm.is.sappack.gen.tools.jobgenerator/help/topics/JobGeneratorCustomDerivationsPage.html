Define column derivations

Use this page to derive expressions to be applied to all columns of a particular type. These expressions are defined within the Transformer stages of the generated jobs. You can cross-reference the input argument of the transformer with $1. Some examples are: TrimLeadingTrailing($1) or NullToEmpty($1).

Use Custom Derivations
Use this check box to assign expression for each InfoSphere™ DataStage® type. You can assign only one mapping per data type. You can use the special term $1 to refer to the input column within the expressions. For example, to run the NullToEmpty function of IBM® InfoSphere DataStage for all CHAR type columns, specify CHAR in the DataType field and NullToEmpty($1) in the Derivation field. At run time, $1 is replaced with the respective column value.
