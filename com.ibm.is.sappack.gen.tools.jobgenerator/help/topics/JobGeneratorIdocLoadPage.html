IDoc load options

Use this page to configure the options for IDoc Load jobs.

Note: You can use InfoSphere™ DataStage® job parameters for text fields by enclosing the text with the "#" symbol. The parameter sets are indicated with the syntax "#" #ParameterSetName.ParameterName#. For more information about parameters and parameter sets, see the IBM® InfoSphere DataStage and QualityStage™ Designer Client Guide
Source options
Specify the following source options for IDoc load:
Stage Type
Type of stages generated as the source of the IDocs. For example, ODBC Connector or Sequential File.
ODBC Connector
Specify the following options for the ODBC connector:
ODBC data source
The objects loaded to the SAP are read from the tables the ODBC data source configured in the operating system.
ODBC User
Overwrites the user configured in the data source configuration. It is an optional field.
ODBC Password
Overwrites the password configured in the data source configuration. It is an optional field.
Select the table action
You can select APPEND, CREATE, REPLACE, or TRUNCATE table actions. For more information, see the ODBC connector stage documentation.
ODBC SQL WHERE condition
Specify an SQL where clause to be used when querying for the source data. The where clause specified here is applied to all tables being queried. This option is usually used in conjunction with technical fields added through IBM InfoSphere Data Architect Rapid Modeler for SAP application.
Target options
Specify the following target options for IDoc load stage:
SAP Message Type
Specify the SAP message type to be used.
DataStage SAP Connection
Name of the SAP connection configured in DataStage.
Overwrite connection defaults
Select this check box if you want to overwrite the settings defined in the connection in IBM InfoSphere DataStage.
SAP User
User ID of the SAP user.
SAP User Password
Password for the specified SAP user.
SAP User Client Number
Client number of the specified SAP user.
SAP User Language
Language of the specified SAP user.
