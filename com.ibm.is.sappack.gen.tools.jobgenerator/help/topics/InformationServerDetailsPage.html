InfoSphere Information Server Details

Specify theIBM® InfoSphere™ Information Server domain details and IBM InfoSphere DataStage® credentials.

Information Server Domain Details
Domain
Hostname and port number of the IBM InfoSphere DataStage server in the <hostname>:<port number> format.
User
Specify the IBM InfoSphere DataStage user name.
Password
Password of the IBM InfoSphere DataStage user.
DataStage job details
Project
Name of the IBM InfoSphere DataStage project into which the jobs are generated.
Folder
Specify the name of a folder directly within the Jobs folder in IBM InfoSphere DataStage.
Job name prefix
Specify a prefix for all the jobs created during this wizard run. The name of the job is computed using this prefix.
Overwrite the job if it already exists
Select this check box to overwrite the jobs if they already exist in the server.
