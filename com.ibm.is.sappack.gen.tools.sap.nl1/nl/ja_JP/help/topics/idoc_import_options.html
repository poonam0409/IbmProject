IDoc インポート・オプション

このページを使用して、論理データ・モデルに IDoc タイプをインポートする 方法を構成します。各セグメントはリレーショナル表にマップされ、追加フィールドを使用して IDoc セグメント階層の親子関係が設定されます。

IDoc セグメント・フィールド
タイプが異なる IDoc セグメント・フィールドを論理データ・モデル内の属性に マップする方法を指定します。
すべてのセグメント・フィールドを NULL 可能にする
すべてのフィールドに NULL 可能フラグを設定します。
すべての列に対しすべてのタイプを許可
属性のタイプは、セグメント・フィールドのデータ・タイプ (VARCHAR、DATE など) から 派生します。
VARCHAR のみを使用
すべての属性のタイプに VARCHAR を使用します。このオプションを選択すると、長さの 係数を定義でき、元の長さにこの係数を掛けることができます。結果表を使用して、SAP で定義されている 最大長を超える長さのデータを一時的に保持する必要がある場合、このオプションが便利です。
チェック表オプション
特定のフィールドについての有効な値を含んでいる関連するチェック表を抽出するためのオプションを指定します。
関連するチェック表のメタデータをインポート
インポート対象の IDoc セグメント・フィールドのすべてのチェック表をインポートします。
関連するテキスト表のメタデータをインポート
すべてのチェック表の関連するテキスト表をインポートします。これらのテキスト表は、チェック表の 値の説明を含んでいます。
チェック表のブラックリスト
チェック表のインポート時に避ける必要があるチェック表のリストが入ったファイル。1 行につき 1 つの表名を指定します。
