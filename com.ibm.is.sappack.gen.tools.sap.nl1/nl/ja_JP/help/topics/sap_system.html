SAP システム接続の管理

このページを使用して、SAP システムへの接続を管理します。

既存の接続
すべての既存の SAP 接続が表示されます。接続を選択して、接続パラメーターを変更したり、接続を削除したりします。
接続の詳細
選択された接続の接続パラメーターが表示されます。
SAP システム名
特定 SAP システムへの接続の名前。
ロード・バランシングの使用
メッセージ・サーバーを使用して SAP システム間のロード・バランシングを有効にし、関連オプションを指定する場合、このオプションを選択します。
SAP ホスト
SAP システムのアプリケーション・ホスト名。
SAP システム番号
SAP システムの 2 桁の番号。
SAP クライアント番号
SAP システムへの接続時に使用するクライアント ID (数値)。
ルーター・ストリング
SAP システムのルーター・ストリング。
SAP ユーザー
選択された接続の SAP ユーザーの名前。
SAP ユーザーの言語
SAP システムへのログイン時に使用する言語を指定します。
SAP ユーザーのパスワード
SAP ユーザーのパスワード。パスワードはクライアント・マシン上に保管されません。SAP ユーザーは、InfoSphere Data Architect のセッションごとにパスワードの入力を求められます。
