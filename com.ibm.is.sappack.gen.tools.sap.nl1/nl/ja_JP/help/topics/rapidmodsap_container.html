IBM InfoSphere Data Architect Rapid Modeler for SAP Applications

IBM® InfoSphere™ Data Architect Rapid Modeler for SAP Applications ウィザードは、SAP システムから IBM InfoSphere Data Architect の 論理データ・モデルにメタデータを抽出します。

  • SAP システムの選択
    このページを使用して、メタデータの抽出に使用する SAP システムを選択し、構成します。
  • 「事前定義 SAP モデルから選択」ページ
    このページを使用して、ツリーから事前定義 SAP モデルを選択します。
  • SAP 内の表を参照
    このページを使用して、メタデータのインポート元の論理表を参照し、選択します。
  • SAP 内の IDoc を参照
    このページを使用して、論理データ・モデルにインポートする IDoc タイプを検索します。論理データ・モデルごとに 1 つの IDoc タイプを抽出できます。
  • 表を手動で指定
    このページを使用して、インポートする表の名前を手動で指定します。
  • IDoc タイプおよびセグメントを手動で指定
    このページを使用して、IDoc タイプおよびセグメントを手動で指定します。
  • SAP システム接続の管理
    このページを使用して、SAP システムへの接続を管理します。
  • 追加のテクニカル・フィールド
    このページを使用して、追加のテクニカル・フィールドまたは属性を構成します。これらの属性は、論理データ・モデル内に作成される すべてのエンティティーに追加されます。
  • 論理データ・モデル内のインポート用のパッケージの選択
    このページを使用して、論理データ・モデル内で、メタデータのインポート先となるデフォルト以外のパッケージを選択します。
  • メタデータ・インポート・サマリー
    このページには、作成されたエンティティーの数、実行された名前変更の数など、メタデータ・インポート・プロセスの サマリーが表示されます。「完了」をクリックすると、メタデータが論理データ・モデルにインポートされます。
  • SAP からの論理データ・モデルの再抽出
    このオプションを使用して、SAP メタデータを新規論理データ・モデルに 再抽出します。このページには、モデルに対して前に実行されたインポートとインポートされた オブジェクトの説明 (略記) が表示されます。
