論理データ・モデル内のインポート用のパッケージの選択

このページを使用して、論理データ・モデル内で、メタデータのインポート先となるデフォルト以外のパッケージを選択します。

特定のパッケージを使用
このオプションを使用して、既存の論理データ・モデル・パッケージを選択します。

パッケージを選択するには、目的のパッケージを強調表示します。

選択のクリア
既に選択したパッケージをクリアするには、このオプションを選択します。
