IBM InfoSphere Data Architect Rapid Modeler for SAP Applications

La procedura guidata IBM® InfoSphere Data Architect Rapid Modeler for SAP Applications estrae i metadati da un sistema SAP in un modello di dati logico di IBM InfoSphere Data Architect.

  • Selezionare un sistema SAP
    Utilizzare questa pagina per selezionare e configurare il sistema SAP che si desidera utilizzare per estrarre e configurare il sistema SAP che si desidera utilizzare per estrarre i metadati.
  • Pagina Selezionare dai modelli SAP predefiniti
    Utilizzare questa pagina per selezionare modelli SAP predefiniti da una struttura ad albero.
  • Cerca tabelle in SAP
    Utilizzare questa pagina per cercare e selezionare tabelle logiche dalle quali importare i metadati.
  • Cerca IDoc in SAP
    Utilizzare questa pagina per cercare tipi di IDoc che si desidera importare nel modello di dati logico. È possibile estrarre un tipo di IDoc per modello di dati logico.
  • Specificare le tabelle manualmente
    Utilizzare questa pagina per specificare manualmente i nomi delle tabelle da importare.
  • Specificare tipi di IDoc e segmenti manualmente
    Utilizzare questa pagina per specificare tipi di IDoc e segmenti manualmente.
  • Gestione delle connessioni al sistema SAP
    Utilizzare questa pagina per gestire le connessioni ai sistemi SAP.
  • Campi tecnici supplementari
    Utilizzare questa pagina per configurare campi tecnici o attributi supplementari. Questi attributi vengono aggiunti a ogni entità creata nel modello di dati logico.
  • Selezionare un package in un modello di dati logico da importare
    Utilizzare questa pagina per selezionare package diversi dal package predefinito nel modello di dati logico per importare i metadati.
  • Riepilogo importazione metadati
    Questa pagina mostra un riepilogo del processo di importazione metadati, ad esempio il numero di entità create o il numero di modifiche ai nomi eseguite. Fare clic su Fine per importare i metadati nel modello di dati logico.
  • Nuova estrazione del modello di dati logico da SAP
    Utilizzare questa opzione per estrarre nuovamente i metadati SAP in un nuovo modello di dati logico. La pagina mostra le importazioni precedenti eseguite sul modello insieme a una breve descrizione degli oggetti importati.
