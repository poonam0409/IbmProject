Opzioni di importazione IDoc

Utilizzare questa pagina per configurare come verranno importati i tipi di IDoc in un modello di dati logico. Ogni segmento è associato a una tabella relazionale e vengono utilizzati campi supplementari per stabilire la relazione principale/secondario della gerarchia di segmenti IDoc.

Campi Segmento IDoc
Specificare come vengono associati i diversi tipi di campi Segmento IDoc agli attributi nel modello di dati logico:
Rendi tutti i campi segmento null
Imposta l'indicatore consenti valori null su tutti i campi.
Consenti tutti i tipi per tutte le colonne
I tipi di attributi sono derivati dai tipi di dati dei campi segmento, ad esempio VARCHAR o DATE.
Usa solo VARCHAR
Utilizzare VARCHAR come tipo per tutti gli attributi. Se questa opzione è selezionata, è possibile definire un fattore di lunghezza per il quale verrà moltiplicata la lunghezza originale. Questa opzione è utile quando si desidera utilizzare le tabelle risultanti per contenere temporaneamente i dati la cui lunghezza supera la lunghezza massima definita da SAP.
Opzioni di verifica tabella
Specificare le opzioni per estrarre tabelle di verifica correlate che contengono valori validi per determinati campi.
Importare metadati per le tabelle di verifica correlate
Importa tutte le tabelle di verifica per i campi segmento IDoc importati.
Importare metadati per le tabelle di testo correlate
Importa la tabella di testo correlata per ogni tabella di verifica. Queste tabelle di testo contengono le descrizioni dei valori delle tabelle di verifica.
Lista nera di tabelle di verifica
Un file contenente un elenco di tabelle di controllo da evitare durante l'importazione delle tabelle di verifica. Specificare un nome tabella per riga.
