Gestione delle connessioni al sistema SAP

Utilizzare questa pagina per gestire le connessioni ai sistemi SAP.

Connessioni esistenti
Visualizza tutte le connessioni SAP esistenti. Selezionare una connessione per modificare i parametri della connessione o per eliminare una connessione.
Dettagli connessione
Visualizza i parametri di connessione per la connessione selezionata:
Nome sistema SAP
Nome della connessione a un particolare sistema SAP.
Utilizza bilanciamento del carico
Selezionare questa opzione per abilitare il bilanciamento del carico utilizzando un server di messaggistica tra i sistemi SAP e specificare le opzioni correlate.
Host SAP
Nome dell'host di applicazione del sistema SAP.
Numero sistema SAP
Numero di due cifre del sistema SAP.
Numero client SAP
ID client numerico da utilizzare per connettersi al sistema SAP.
Stringa router
La stringa router del sistema SAP.
Utente SAP
Nome dell'utente SAP per la connessione selezionata.
Lingua utente SAP
Specifica la lingua da utilizzare per collegarsi al sistema SAP.
Password utente SAP
Password dell'utente SAP. La password non viene archiviata sulla macchina client. All'utente SAP viene richiesto di immettere la password una sola volta per la sessione di InfoSphere Data Architect.
