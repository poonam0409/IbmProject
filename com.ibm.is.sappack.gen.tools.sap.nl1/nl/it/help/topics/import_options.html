Opzioni di importazione tabella

Utilizzare questa pagina per configurare le tabelle selezionate da importare in un modello di dati logico.

Tabelle di verifica e di testo
Importare metadati per le tabelle di verifica correlate
Selezionare questa opzione se si desidera che la procedura guidata importi le tabelle di verifica correlate alle tabelle selezionate. Ad esempio, se una tabella a un campo di codice paese, la procedura guidata estrae i metadati della tabella di controllo correlata che contiene tutti i valori codice di paese validi. Selezionando questa opzione, è possibile specificare queste ulteriori opzioni:
Raccogliere tabelle di verifica ricorsivamente
Selezionare questa opzione per estrarre metadati dalle tabelle di controllo delle tabelle di controllo. Utilizzare questa opzione con prudenza perché potrebbe comportare l'estrazione di molte tabelle.
Importare metadati per le tabelle di testo correlate
Selezionare questa opzione per importare il le tabelle di testo corrispondenti alle tabelle di controllo. Ad esempio, per la tabella di controllo contenente un codice di paese valido, ad esempio US, la tabella di testo corrispondente contiene i nomi degli stati degli Stati Uniti.
Lista nera di tabelle di verifica
File lista nera
Questi file contiene un elenco di tabelle di controllo da escludere durante l'importazione delle tabelle di controllo. Assicurarsi che questo file contenga solo un nome tabella per riga, separato da un ritorno a capo.
