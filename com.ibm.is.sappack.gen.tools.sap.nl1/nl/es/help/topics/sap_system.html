Gestionar conexiones de sistemas SAP

Utilice esta página para gestionar las conexiones con los sistemas SAP.

Conexiones existentes
Muestra todas las conexiones SAP existentes. Seleccione una conexión para modificar los parámetros de conexión o para suprimir una conexión.
Detalles de la conexión
Muestra los parámetros de conexión de la conexión seleccionada:
Nombre de sistema SAP
Nombre de la conexión con un determinado sistema SAP.
Utilizar equilibrio de carga
Seleccione esta opción para habilitar el equilibrio de carga utilizando un servidor de mensajes entre los sistemas SAP y especificar las opciones correspondientes.
Sistema principal de SAP
Nombre del sistema principal de aplicaciones del sistema SAP.
Número de sistema SAP
Número de dos dígitos del sistema SAP.
Número de cliente de SAP
ID de cliente numérico que se utiliza cuando se conecta al sistema SAP.
Serie del direccionador
Serie del direccionador del sistema SAP.
Usuario de SAP
Nombre del usuario de SAP para la conexión seleccionada.
Idioma del usuario de SAP
Especifique el idioma que se va a utilizar mientras se inicia una sesión en el sistema SAP.
Contraseña de usuario de SAP
Contraseña del usuario de SAP. La contraseña no se almacena en la máquina cliente. El usuario de SAP debe especificar la contraseña una vez para cada sesión de InfoSphere Data Architect.
