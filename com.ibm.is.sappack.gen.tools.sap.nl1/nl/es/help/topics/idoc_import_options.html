Opciones de importación de IDoc

Utilice esta página para configurar cómo se importan los tipos de IDoc a un modelo de datos lógico. Cada segmento se correlaciona con una tabla relacional y se utilizan campos adicionales para establecer la relación padre e hijo de la jerarquía de segmentos de IDoc.

Campos de segmento de IDoc
Especifique cómo se correlacionan los distintos tipos de campos de segmento de IDoc con los atributos del modelo de datos lógico:
Todos los campos de segmento admiten nulos
Establece el distintivo Con posibilidades de nulos en todos los campos.
Permitir todos los tipos para todas las columnas
Los tipos de los atributos se derivan de los tipos de datos de los campos de segmento como, por ejemplo, VARCHAR o DATE.
Utilizar VARCHAR sólo
Utiliza VARCHAR como tipo para todos los atributos. Si esta opción está seleccionada, puede definir un factor de longitud por el que se multiplica la longitud original. Esta opción es muy útil si desea utilizar las tablas resultantes para contener temporalmente datos cuya longitud excede la longitud máxima definida por SAP.
Opciones de las tablas de comprobación
Especifique las opciones para extraer las tablas de comprobación relacionadas que contienen valores válidos para determinados campos.
Importar metadatos para las tablas de comprobación relacionadas
Importa todas las tablas de comprobación para los campos de segmento de IDoc importados.
Importar metadatos para las tablas de texto relacionadas
Importa la tabla de texto relacionada de cada tabla de comprobación. Estas tablas de texto contienen descripciones de los valores de las tablas de comprobación.
Lista negra de tablas de comprobación
Un archivo que contiene una lista de las tablas de comprobación que se deben evitar durante la importación de tablas de comprobación. Especifique un nombre de tabla por línea.
