IBM InfoSphere Data Architect Rapid Modeler for SAP Applications

El asistente de IBM® InfoSphere Data Architect Rapid Modeler for SAP Applications extrae metadatos de un sistema SAP en un modelo de datos lógico de IBM InfoSphere Data Architect.

  • Seleccionar sistema SAP
    Utilice esta página para seleccionar y configurar el sistema SAP que desea utilizar para extraer metadatos.
  • Página Seleccionar entre los modelos SAP predefinidos
    Utilice esta página para seleccionar los modelos SAP predefinidos en un árbol.
  • Examinar SAP para buscar tablas
    Utilice esta página para examinar y seleccionar las tablas lógicas de las que desea importar metadatos
  • Examinar SAP para buscar IDocs
    Utilice esta página para buscar los tipos de IDoc que desea importar al modelo de datos lógico. Puede extraer un tipo de IDoc para cada modelo de datos lógico.
  • Especificar tablas manualmente
    Utilice esta página para especificar manualmente los nombres de las tablas que se van a importar.
  • Especificar segmentos y tipos de IDoc manualmente
    Utilice esta página para especificar segmentos y tipos de IDoc manualmente.
  • Gestionar conexiones de sistemas SAP
    Utilice esta página para gestionar las conexiones con los sistemas SAP.
  • Campos técnicos adicionales
    Utilice esta página para configurar atributos o campos técnicos adicionales. Estos atributo se añaden a cada entidad creada en el modelo de datos lógico.
  • Seleccionar un paquete en el modelo de datos lógico para importarlo
    Utilice esta página para seleccionar paquetes distintos al paquete predeterminado en el modelo de datos lógico para importar metadatos.
  • Resumen de importación de metadatos
    Esta página muestra un resumen del proceso de importación de metadatos como, por ejemplo, el número de entidades creadas o el número de cambios de nombre realizados. Pulse Finalizar para importar los metadatos al modelo de datos lógico.
  • Volver a extraer modelo de datos lógico de SAP
    Utilice esta opción para volver a extraer los metadatos de SAP en un nuevo modelo de datos lógico. La página muestra las importaciones anteriores realizadas en el modelo junto con una breve descripción de los objetos que se han importado.
