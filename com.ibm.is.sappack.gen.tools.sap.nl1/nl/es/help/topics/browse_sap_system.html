Examinar SAP para buscar tablas

Utilice esta página para examinar y seleccionar las tablas lógicas de las que desea importar metadatos

Buscar nombre o patrón
Especifique el nombre de una tabla y pulse Buscar para examinar el sistema SAP. Puede utilizar "*" o "%" como comodín.
Tablas
Se muestra una lista de las tablas que coinciden con la consulta.
Tablas encontradas
En la lista de tablas que coinciden con la consulta, seleccione las tablas cuyos metadatos desea importar y pulse Añadir tablas.
Tablas seleccionadas
En esta sección se muestran las tablas seleccionadas. Para eliminar una tabla de la lista, selecciónela y pulse Eliminar tablas.
