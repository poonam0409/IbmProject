Opciones de importación de tablas

Utilice esta página para configurar las tablas seleccionadas que se van a importar a un modelo de datos lógico.

Tablas de comprobación y de texto
Importar metadatos para las tablas de comprobación relacionadas
Seleccione esta opción si desea que el asistente importe las tablas de comprobación relacionadas con las tablas seleccionadas. Por ejemplo, si una tabla tiene un campo de código de país, el asistente extrae los metadatos de la tabla de comprobación relacionada que contiene todos los valores de código de país válidos. Si selecciona esta opción, puede especificar estas opciones adicionales:
Recopilar tablas de comprobación de forma recursiva
Seleccione esta opción para extraer metadatos de las tablas de comprobación de las tablas de comprobación. Utilice esta opción con precaución ya que puede provocar la extracción de muchas tablas.
Importar metadatos para las tablas de texto relacionadas
Seleccione esta opción para importar de manera adicional las tablas de texto correspondientes de las tablas de comprobación. Por ejemplo, para la tabla de comprobación que contiene un código de país válido como, por ejemplo, US, la tabla de texto correspondiente contiene el nombre de país Estados Unidos.
Lista negra de tablas de comprobación
Archivo de lista negra
Este archivo contiene una lista de las tablas de comprobación que se deben excluir durante la importación de tablas de comprobación. Asegúrese de que este archivo contenga un nombre de tabla por línea separado por un retorno de carro.
