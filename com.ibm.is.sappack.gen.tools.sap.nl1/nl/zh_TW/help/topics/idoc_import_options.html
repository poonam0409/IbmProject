IDoc 匯入選項

使用此頁面，可以配置如何將 IDoc 類型匯入至邏輯資料模型。每個區段都會對映至一個關聯式表格，而其他欄位則是用來建立 IDoc 區段階層的母項及子項關係。

IDoc 區段欄位
指定如何將不同類型的 IDoc 區段欄位對映至邏輯資料模型中的屬性：
將所有區段欄位設為可為空值
設定所有欄位的可為空值旗標。
允許所有直欄的所有類型
屬性的類型是衍生自區段欄位的資料類型（例如 VARCHAR 或 DATE）。
只使用 VARCHAR
使用 VARCHAR 作為所有屬性的類型。如果選取此選項，則可以定義原始長度所乘上的長度係數。如果您要使用產生的結果來暫時保留其長度超過 SAP 所定義長度上限的資料，則此選項十分有用。
檢查表格選項
指定選項，以擷取含有特定欄位的有效值的相關檢查表格。
匯入相關檢查表格的 Meta 資料
匯入已匯入 IDoc 區段欄位的所有檢查表格。
匯入相關文字表格的 Meta 資料
匯入每一個檢查表格的相關文字表格。這些文字表格包含檢查表格值的說明。
檢查表格黑名單
一個檔案，內含要在檢查表格匯入期間避免使用的檢查表格清單。每行指定一個表格名稱。
