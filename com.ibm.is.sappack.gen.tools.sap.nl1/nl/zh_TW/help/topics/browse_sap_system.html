瀏覽 SAP 以找出表格

使用此頁面，可以瀏覽及選取要從中匯入 Meta 資料的邏輯表格

搜尋名稱或型樣
指定表格的名稱，並按一下搜尋以瀏覽 SAP 系統。您可以使用 "*" 或 "%" 作為萬用字元。
表格
畫面上會顯示與您查詢相符的表格清單。
找到的表格
從與查詢相符的表格清單中，選取要匯入其 Meta 資料的表格，然後按一下新增表格。
選取的表格
在此區段中，會顯示選取的表格。若要移除此清單中的表格，請選取表格，然後按一下移除表格。
