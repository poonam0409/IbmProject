IBM InfoSphere Information Server Pack for SAP Applications 7.0 版
讀者意見

選取 SAP 系統

使用此頁面，可以選取及配置要用來擷取 Meta 資料的 SAP 系統。

SAP 系統
選取要連接的 SAP 系統。按一下編輯以建立連線（如果沒有 SAP 連線），或修改現有連線。
擷取選項
選取下列其中一個擷取選項：
從預先定義的 SAP 模型選取
從預先定義的物件模型中，選取 SAP 商業物件及其相關的邏輯表格。您也可以載入自訂模型。
瀏覽 SAP 以找出表格
瀏覽 SAP 系統以選取要使用的邏輯表格。
瀏覽 SAP 以找出 IDoc
瀏覽 SAP 系統，並選取要使用的 IDoc 類型及區段。
手動指定表格
選取此選項，可以手動指定表格的清單。
手動指定 IDoc
選取此選項，可以手動指定 IDoc 區段的清單。
