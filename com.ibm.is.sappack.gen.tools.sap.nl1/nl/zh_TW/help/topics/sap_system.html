管理 SAP 系統連線

使用此頁面，可以管理與 SAP 系統的連線。

現有連線
顯示所有現有 SAP 連線。請選取連線以修改連線參數，或刪除連線。
連線明細
顯示所選取連線的連線參數：
SAP 系統名稱
與特定 SAP 系統的連線的名稱。
使用負載平衡
選取此選項，可以在 SAP 系統之間使用訊息伺服器來啟用負載平衡，以及指定相關選項。
SAP 主機
SAP 系統的應用程式主機名稱。
SAP 系統號碼
以兩位數表示的 SAP 系統號碼。
SAP 用戶端號碼
要在連接至 SAP 系統時使用的數值用戶端 ID。
路由器字串
SAP 系統的路由器字串。
SAP 使用者
所選取連線的 SAP 使用者名稱。
SAP 使用者語言
指定要在登入 SAP 系統時使用的語言。
SAP 使用者密碼
SAP 使用者的密碼。密碼不是儲存在用戶端機器上。系統會提示 SAP 使用者，對於每個 InfoSphere Data Architect 階段作業都輸入密碼一次。
