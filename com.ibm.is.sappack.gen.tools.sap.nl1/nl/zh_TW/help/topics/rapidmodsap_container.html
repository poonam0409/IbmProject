IBM InfoSphere Data Architect Rapid Modeler for SAP Applications

IBM® InfoSphere™ Data Architect Rapid Modeler for SAP Applications 精靈可將 SAP 系統中的 Meta 資料擷取至 IBM InfoSphere Data Architect 的邏輯資料模型。

  • 選取 SAP 系統
    使用此頁面，可以選取及配置要用來擷取 Meta 資料的 SAP 系統。
  • 從預先定義的 SAP 模型選取頁面
    使用此頁面，可以從樹狀結構中選取預先定義的 SAP 模型。
  • 瀏覽 SAP 以找出表格
    使用此頁面，可以瀏覽及選取要從中匯入 Meta 資料的邏輯表格
  • 瀏覽 SAP 以找出 IDoc
    使用此頁面，可以搜尋要匯入至邏輯資料模型的 IDoc 類型。一個邏輯資料模型可以擷取一種 IDoc 類型。
  • 手動指定表格
    使用此頁面，可以手動指定要匯入的表格名稱。
  • 手動指定 IDoc 類型及區段
    使用此頁面，可以手動指定 IDoc 類型及區段。
  • 管理 SAP 系統連線
    使用此頁面，可以管理與 SAP 系統的連線。
  • 其他技術欄位
    使用此頁面，可以配置其他技術欄位或屬性。這些屬性會新增至邏輯資料模型中所建立的每個實體。
  • 選取邏輯資料模型中要匯入的套件
    使用此頁面，可以選取套件（非邏輯資料模型中的預設套件）來匯入 Meta 資料。
  • Meta 資料匯入摘要
    此頁面顯示 Meta 資料匯入程序的摘要（例如，建立的實體數或執行的名稱變更次數）。按一下完成，將 Meta 資料匯入至「邏輯資料模型」。
  • 從 SAP 重新擷取邏輯資料模型
    使用此選項，可以將 SAP Meta 資料重新擷取至新的邏輯資料模型。此頁面會顯示模型上先前所執行的匯入，以及所匯入物件的簡要說明。
