IBM InfoSphere Information Server Pack for SAP Applications 7.0 版
讀者意見

從預先定義的 SAP 模型選取頁面

使用此頁面，可以從樹狀結構中選取預先定義的 SAP 模型。

商業物件
您可以從樹狀結構中選取套件、商業物件及 SAP 邏輯表格。
  • 將可以分組一個以上商業物件的資料夾符號化。
  • 將已附加一個以上邏輯 SAP 表格的商業物件符號化，而所選取物件的相關資料則是儲存在這些邏輯 SAP 表格中。
  • 將 SAP 邏輯表格符號化。
自訂模型檔案
使用此選項，可以選取及載入要顯示在樹狀結構中的自訂模型檔案。按一下瀏覽，以選取及顯示樹狀結構中的自訂模型檔案。
範例模型檔案
若要使用範例精靈來匯入範例模型檔案，請選取檔案 > 新建 > 範例 > IBM InfoSphere Information Server Pack for SAP Applications > SAP 商業物件範例模型。
