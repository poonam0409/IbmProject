IBM InfoSphere Data Architect Rapid Modeler for SAP Applications

IBM® InfoSphere™ Data Architect Rapid Modeler for SAP Applications 向导将元数据从 SAP 系统抽取到 IBM InfoSphere Data Architect 的逻辑数据模型中。

  • 选择 SAP 系统
    使用该页面可选择和配置要用于抽取元数据的 SAP 系统。
  • 从预定义的 SAP 模型页面中选择
    使用该页面可从树中选择预定义的 SAP 模型。
  • 浏览 SAP 以查找表
    使用该页面可浏览和选择要从中导入元数据的逻辑表。
  • 浏览 SAP 以查找 IDoc
    使用该页面可搜索要导入到逻辑数据模型的 IDoc 类型。您可以为每个逻辑数据模型抽取一种 IDoc 类型。
  • 手动指定表
    使用该页面可手动指定要导入的表的名称。
  • 手动指定 IDoc 类型和段
    使用该页面可手动指定 IDoc 类型和段。
  • 管理 SAP 系统连接
    使用该页面可管理与 SAP 系统的连接。
  • 其他技术字段
    使用该页面可配置其他技术字段或属性。将为逻辑数据模型中创建的每个实体添加这些属性。
  • 选择逻辑数据模型中要导入的包
    使用该页面可选择逻辑数据模型中除缺省包之外的、要导入其元数据的包。
  • 元数据导入摘要
    该页面显示元数据导入过程的摘要，例如创建的实体数量或执行的名称更改次数。单击完成以将元数据导入逻辑数据模型中。
  • 从 SAP 重新抽取逻辑数据模型
    使用该选项可将 SAP 元数据重新抽取到新的逻辑数据模型中。该页面显示对模型执行的先前导入以及导入的对象的简要描述。
