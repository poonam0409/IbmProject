反馈

管理 SAP 系统连接

使用该页面可管理与 SAP 系统的连接。

现有连接
显示所有现有的 SAP 连接。选择一个连接，以修改连接参数或删除连接。
连接详细信息
显示所选连接的连接参数：
SAP 系统名称
与特定 SAP 系统的连接的名称。
使用负载均衡
选择该选项可通过在 SAP 系统间使用消息服务器来启用负载均衡，并指定相关选项/
SAP 主机
SAP 系统的应用程序主机名。
SAP 系统编号
SAP 系统的两数字编号。
SAP 客户机编号
连接 SAP 系统时要使用的客户机数字标识。
路由器字符串
SAP 系统的路由器字符串。
SAP 用户
所选连接的 SAP 用户的名称。
SAP 用户语言
指定登录 SAP 系统时要所用的语言。
SAP 用户密码
SAP 用户的密码。密码不存储在客户机上。针对每个 InfoSphere Data Architect 会话，将提示 SAP 用户输入一次密码。
