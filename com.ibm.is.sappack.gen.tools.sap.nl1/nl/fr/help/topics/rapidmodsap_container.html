IBM InfoSphere Data Architect Rapid Modeler for SAP Applications

L'assistant IBM® InfoSphere Data Architect Rapid Modeler for SAP Applications extrait des métadonnées à partir d'un système SAP vers un modèle de données logique de IBM InfoSphere Data Architect.

  • Sélectionner un système SAP
    Cette page permet de sélectionner et de configurer le système SAP à utiliser pour extraire des métadonnées.
  • Page Sélectionner à partir de modèles SAP prédéfinis
    Cette page permet de sélectionner des modèles SAP prédéfinis dans une arborescence.
  • Parcourir SAP pour rechercher les tables
    Cette page permet de parcourir et de sélectionner des tables logiques à partir desquelles importer des métadonnées.
  • Parcourir SAP pour rechercher les IDoc
    Cette page permet de rechercher des types d'IDoc à importer dans votre modèle de données logique. Vous pouvez extraire un type d'IDoc par modèle de données logique.
  • Indiquer les tables manuellement
    Cette page permet d'indiquer manuellement des noms de tables à importer.
  • Spécifier des types d'IDoc et des segments manuellement
    Cette page permet d'indiquer manuellement des types d'IDoc et des segments.
  • Gestion des connexions aux systèmes SAP
    Cette page permet de gérer les connexions aux systèmes SAP.
  • Zones techniques supplémentaires
    Cette page permet de configurer des attributs ou des zones techniques supplémentaires. Ces attributs sont ajoutés à chaque entité créée dans le modèle de données logique.
  • Sélectionner dans un modèle de données logique un module à importer
    Cette page permet de sélectionner des modules autres que celui par défaut dans le modèle de données logique pour importer des métadonnées.
  • Récapitulatif d’importation des métadonnées
    Cette page récapitule les opérations du processus d’importation des métadonnées, telles que le nombre d'entités créées ou le nombre de modifications de nom effectuées. Cliquez sur Terminer pour importer les métadonnées dans le modèle de données logique.
  • Réextraire un modèle de données logique à partir de SAP
    Cette option permet de réextraire des métadonnées SAP dans un nouveau modèle de données logique. La page affiche les importations précédemment effectuées sur le modèle ainsi qu'une courte description des objets importés.
