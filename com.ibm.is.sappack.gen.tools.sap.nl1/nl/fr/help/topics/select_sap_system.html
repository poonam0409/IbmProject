Sélectionner un système SAP

Cette page permet de sélectionner et de configurer le système SAP à utiliser pour extraire des métadonnées.

Système SAP
Sélectionnez le système SAP auquel vous voulez vous connecter. Cliquez sur Editer pour créer une connexion, s'il n'existe aucune connexion SAP, ou pour modifier une connexion existante.
Options d'extraction
Sélectionnez l'une des options d'extraction suivantes :
Sélectionner à partir de modèles SAP prédéfinis
Sélectionnez des objets métier SAP et leurs tables logiques connexes à partir d'un modèle d'objet prédéfini. Vous pouvez également charger un modèle personnalisé.
Parcourir SAP pour rechercher les tables
Parcourez le système SAP pour sélectionner les tables logiques avec lesquelles vous voulez travailler.
Parcourir SAP pour rechercher les IDoc
Parcourez le système SAP pour sélectionner les types d'IDoc et les segments lesquels vous voulez travailler.
Indiquer les tables manuellement
Sélectionnez cette option pour spécifier la liste des tables manuellement.
Indiquer les IDoc manuellement
Sélectionnez cette option pour spécifier la liste des segments IDoc manuellement.
