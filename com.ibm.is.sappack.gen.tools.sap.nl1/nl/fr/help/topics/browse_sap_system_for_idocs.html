Parcourir SAP pour rechercher les IDoc

Cette page permet de rechercher des types d'IDoc à importer dans votre modèle de données logique. Vous pouvez extraire un type d'IDoc par modèle de données logique.

Recherche de nom ou de masque
Sélectionnez le type d'IDoc en indiquant son nom ou en entrant un caractère générique tel que * ou %. Sélectionnez Type de base ou de Type d'extension, puis cliquez sur Rechercher pour afficher les types d'IDoc qui répondent à vos critères de recherche.
Types IDoc
Développez le noeud de l'IDoc pour afficher la hiérarchie de segments. Vous pouvez soit sélectionner l'intégralité du type d'IDoc ou uniquement des segments individuels dans l'arborescence de segments.
