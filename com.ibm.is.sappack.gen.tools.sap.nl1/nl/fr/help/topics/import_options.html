Options d'importation de table

Cette page permet de configurer les tables sélectionnées pour importation dans un modèle de données logique.

Tables de vérification et de texte
Importer les métadonnées pour les tables de vérification connexes
Sélectionnez cette option si vous voulez que l'assistant importe des tables de vérification connexes des tables sélectionnées. Par exemple, si une table contient une zone de code de pays, l'assistant extrait des métadonnées de la table de vérification connexe qui contient toutes les valeurs de code de pays valides. Si vous sélectionnez cette option, les options supplémentaires suivantes sont disponibles :
Collecter les tables de vérification de manière récursive
Sélectionnez cette option pour extraire des métadonnées à partir des tables de vérification des tables de vérification. Cette option doit être utilisée avec prudence car elle peut entraîner l'extraction d'un grand nombre de tables.
Importer les métadonnées pour les tables de texte connexes
Sélectionnez cette option pour importer également les tables de texte correspondantes des tables de vérification. Par exemple, la table de texte correspondante de la table de vérification qui contient le code de pays EU, contient le nom de pays Etat-Unis.
Liste noire des tables de vérification
Fichier de liste noire
Ce fichier contient la liste des tables de vérification à exclure de l'importation des tables de vérification. Assurez-vous que ce fichier contient un nom de table par ligne suivi d'un retour chariot.
