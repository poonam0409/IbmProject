테이블 가져오기 옵션

이 페이지에서 논리 데이터 모델로 가져오도록 선택한 테이블을 구성할 수 있습니다.

검사 및 텍스트 테이블
관련 검사 테이블을 위한 메타데이터 가져오기
선택한 테이블과 관련된 검사 테이블을 마법사가 가져오도록 하려면 이 옵션을 선택하십시오. 예를 들어, 테이블에 국가 코드 필드가 있는 경우 마법사는 모든 유효한 국가 코드 값을 포함한 관련 검사 테이블의 메타데이터를 추출합니다. 이 옵션을 선택하면 다음과 같은 추가 옵션을 지정할 수 있습니다.
검사 테이블을 재귀적으로 수집
검사 테이블에서 메타데이터를 추출하려면 이 옵션을 선택하십시오. 이 옵션을 사용하면 많은 테이블이 추출될 수 있으므로 주의하십시오.
관련 텍스트 테이블을 위한 메타데이터 가져오기
검사 테이블에 대한 해당 텍스트 테이블을 추가로 가져오려면 이 옵션을 선택하십시오. 예를 들어, US와 같이 유효한 국가 코드를 포함한 검사 테이블의 경우 해당 텍스트 테이블에는 미국이라는 국가 이름이 포함됩니다.
검사 테이블 블랙리스트
블랙리스트 파일
이 파일에는 검사 테이블 가져오기 중에 제외할 검사 테이블 목록이 포함됩니다. 이 파일에 캐리지 리턴에 의해 구분되는 행당 하나의 테이블 이름이 포함되어 있는지 확인하십시오.
