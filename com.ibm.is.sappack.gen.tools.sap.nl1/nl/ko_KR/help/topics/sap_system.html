SAP 시스템 연결 관리

이 페이지에서 SAP 시스템에 대한 연결을 관리할 수 있습니다.

기존 연결
기존의 모든 SAP 연결을 표시합니다. 연결 매개변수를 수정하거나 연결을 삭제할 연결을 선택하십시오.
연결 세부사항
선택한 연결에 대한 연결 매개변수를 표시합니다.
SAP 시스템 이름
특정 SAP 시스템에 대한 연결의 이름입니다.
로드 밸런싱 사용
SAP 시스템 간에 메시지 서버를 사용하여 로드 밸런싱을 사용하려면 이 옵션을 선택하고 관련 옵션을 지정하십시오.
SAP 호스트
SAP 시스템의 애플리케이션 호스트 이름입니다.
SAP 시스템 번호
SAP 시스템의 2자리 번호입니다.
SAP 클라이언트 번호
SAP 시스템에 연결하는 동안 사용할 숫자 클라이언트 ID입니다.
라우터 문자열
SAP 시스템의 라우터 문자열입니다.
SAP 사용자
선택한 연결에 대한 SAP 사용자의 이름입니다.
SAP 사용자 언어
SAP 시스템에 로그인하는 동안 사용할 언어를 지정하십시오.
SAP 사용자 비밀번호
SAP 사용자를 위한 비밀번호입니다. 비밀번호는 클라이언트 시스템에 저장되지 않습니다. SAP 사용자는 InfoSphere Data Architect 세션당 비밀번호를 한 번 입력하면 됩니다.
상위 주제: IBM InfoSphere Data Architect Rapid Modeler for SAP 애플리케이션
