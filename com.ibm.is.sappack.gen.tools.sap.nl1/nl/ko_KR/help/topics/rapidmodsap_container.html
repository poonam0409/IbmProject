IBM InfoSphere Data Architect Rapid Modeler for SAP 애플리케이션

IBM® InfoSphere™ Data Architect Rapid Modeler for SAP 애플리케이션 마법사는 SAP 시스템에서 IBM InfoSphere Data Architect의 논리 데이터 모델로 메타데이터를 추출합니다.

  • SAP 시스템 선택
    이 페이지에서는 메타데이터 추출을 위해 사용할 SAP 시스템을 선택하고 구성할 수 있습니다.
  • 미리 정의된 SAP 모델 페이지에서 선택
    이 페이지에서는 트리에서 미리 정의된 SAP 모델을 선택할 수 있습니다.
  • SAP의 테이블 찾아보기
    이 페이지에서 찾아보기 기능을 사용하고 메타데이터를 가져올 소스 논리 테이블을 선택할 수 있습니다.
  • SAP에서 IDoc 찾아보기
    이 페이지에서는 논리 데이터 모델로 가져오려는 IDoc 유형을 검색할 수 있습니다. 논리 데이터 모델당 하나의 IDoc 유형을 추출할 수 있습니다.
  • 테이블 수동 지정
    이 페이지에서 가져올 테이블의 이름을 수동으로 지정할 수 있습니다.
  • IDoc 유형 및 세그먼트 수동 지정
    이 페이지에서 IDoc 유형 및 세그먼트를 수동으로 지정할 수 있습니다.
  • SAP 시스템 연결 관리
    이 페이지에서 SAP 시스템에 대한 연결을 관리할 수 있습니다.
  • 추가 기술 필드
    이 페이지에서 기술 필드 또는 속성을 추가로 구성할 수 있습니다. 이런 속성은 논리 데이터 모델에서 작성된 모든 엔티티에 추가됩니다.
  • 논리 데이터 모델에서 가져올 패키지 선택
    이 페이지에서 메타데이터를 가져올 논리 데이터 모델에서 기본 패키지 이외의 패키지를 선택할 수 있습니다.
  • 메타데이터 가져오기 요약
    이 페이지에는 작성되는 엔티티의 수 또는 수행되는 이름 변경 횟수와 같은 메타데이터 가져오기 프로세스의 요약이 표시됩니다. 논리 데이터 모델로 메타데이터를 가져오려면 완료를 클릭하십시오.
  • SAP에서 논리 데이터 모델 다시 추출
    SAP 메타데이터를 새 논리 데이터 모델로 다시 추출하려면 이 옵션을 사용하십시오. 이 페이지에는 가져온 오브젝트에 대한 간단한 설명과 함께 모델에서 수행된 이전 가져오기가 표시됩니다.
