미리 정의된 SAP 모델 페이지에서 선택

이 페이지에서는 트리에서 미리 정의된 SAP 모델을 선택할 수 있습니다.

비즈니스 오브젝트
트리에서 패키지, 비즈니스 오브젝트 및 SAP 논리 테이블을 선택할 수 있습니다.
  • 하나 이상의 비즈니스 오브젝트를 그룹화할 수 있는 폴더를 나타냅니다.
  • 선택한 오브젝트에 관련된 데이터가 저장되도록 연결된 하나 이상의 논리 SAP 테이블이 있는 비즈니스 오브젝트를 나타냅니다.
  • SAP 논리 테이블을 나타냅니다.
사용자 정의 모델 파일
트리에 표시할 사용자 정의 모델 파일을 선택하고 로드하려면 이 옵션을 사용하십시오. 트리의 사용자 정의 모델 파일을 선택하고 표시하려면 찾아보기를 클릭하십시오.
모델 파일 예제
예제 마법사를 사용하여 모델 파일 예제를 가져오려면 파일 > 새로 작성 > 예제 > IBM InfoSphere Information Server Pack for SAP Applications > SAP Business Object 예제 모델을 선택하십시오.
