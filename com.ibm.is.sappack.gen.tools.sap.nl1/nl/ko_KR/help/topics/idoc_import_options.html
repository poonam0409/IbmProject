IDoc 가져오기 옵션

이 페이지에서 IDoc 유형을 논리 데이터 모델로 가져오는 방식을 구성할 수 있습니다. 각 세그먼트는 관계 테이블로 맵핑되고 IDoc 세그먼트 계층 구조의 상위 및 하위 관계를 설정하는 데 추가 필드가 사용됩니다.

IDoc 세그먼트 필드
논리 데이터 모델에서 다양한 유형의 IDoc 세그먼트 필드를 속성에 맵핑하는 방법을 지정하십시오.
모든 세그먼트 필드에서 널(null) 허용
모든 필드에서 널을 허용하는 플래그를 설정합니다.
모든 열에 모든 유형 허용
속성의 유형은 VARCHAR 또는 DATE와 같은 세그먼트 필드의 데이터 유형에서 파생됩니다.
VARCHAR만 사용
VARCHAR을 모든 속성에 대한 유형으로 사용하십시오. 이 옵션을 선택한 경우 원래 길이에 곱하는 길이 비율을 정의할 수 있습니다. 결과 테이블을 사용하여 길이가 SAP에서 정의된 최대 길이를 초과하는 데이터를 임시로 보유하도록 하려는 경우 이 옵션이 유용합니다.
검사 테이블 옵션
어떤 필드에 대해 유효한 값을 포함한 관련 검사 테이블을 추출하는 옵션을 지정하십시오.
관련 검사 테이블을 위한 메타데이터 가져오기
가져온 IDoc 세그먼트 필드에 대한 모든 검사 테이블을 가져옵니다.
관련 텍스트 테이블을 위한 메타데이터 가져오기
모든 검사 테이블에 대한 관련 텍스트 테이블을 가져옵니다. 이런 텍스트 테이블에는 검사 테이블의 값에 대한 설명이 포함됩니다.
검사 테이블 블랙리스트
검사 테이블 가져오기 중에 피할 검사 테이블 목록이 포함된 파일입니다. 행당 하나의 테이블 이름을 지정하십시오.
상위 주제: IDoc 유형 및 세그먼트 수동 지정
