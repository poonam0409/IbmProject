SAP-System auswählen

Auf dieser Seite können Sie das SAP-System auswählen und konfigurieren, das für das Extrahieren von Metadaten verwendet werden soll.

SAP-System
Wählen Sie das SAP-System aus, zu dem eine Verbindung hergestellt werden soll. Klicken Sie auf Bearbeiten, um eine Verbindung herzustellen, wenn keine SAP-Verbindung vorhanden ist, oder um eine vorhandene Verbindung zu modifizieren.
Extraktionsoptionen
Wählen Sie eine der folgenden Extraktionsoptionen aus:
Aus vordefinierten SAP-Modellen auswählen
Wählen Sie SAP-Geschäftsobjekte und ihre zugehörigen logischen Tabellen aus einem vordefinierten Objektmodell aus. Sie können auch ein angepasstes Modell laden.
SAP nach Tabellen durchsuchen
Durchsuchen Sie das SAP-System, um die logischen Tabellen, mit denen Sie arbeiten wollen, auszuwählen.
SAP nach IDocs durchsuchen
Durchsuchen Sie das SAP-System und wählen Sie die IDoc-Typen und Segmente aus, mit denen Sie arbeiten wollen.
Tabellen manuell angeben
Wählen Sie diese Option aus, um Tabellenverzeichnisse manuell anzugeben.
IDocs manuell angeben
Wählen Sie diese Option aus, um die Liste der IDoc-Segmente manuell anzugeben.
