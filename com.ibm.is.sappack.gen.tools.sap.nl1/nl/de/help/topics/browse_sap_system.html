SAP nach Tabellen durchsuchen

Auf dieser Seite können Sie zu den logischen Tabellen navigieren und die Seiten auswählen, aus denen Sie Metadaten importieren wollen.

Nach Name oder Muster suchen
Geben Sie den Namen einer Tabelle an und klicken Sie auf Suchen, um das SAP-System zu durchsuchen. Sie können "*" oder "%" als Platzhalterzeichen verwenden.
Tabellen
Eine Liste der mit Ihrer Abfrage übereinstimmenden Tabellen wird angezeigt.
Gefundene Tabellen
Wählen Sie aus der Liste der mit Ihrer Abfrage übereinstimmenden Tabellen die Tabellen aus, deren Metadaten Sie importieren wollen, und klicken Sie auf Tabellen hinzufügen.
Ausgewählte Tabellen
In diesem Bereich werden die ausgewählten Tabellen angezeigt. Wenn Sie eine Tabelle aus der Liste entfernen wollen, wählen Sie sie aus und klicken auf Tabellen entfernen.
