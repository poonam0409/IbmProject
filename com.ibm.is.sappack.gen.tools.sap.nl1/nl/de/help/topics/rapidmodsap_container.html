IBM InfoSphere Data Architect Rapid Modeler for SAP Applications

IBM® InfoSphere Data Architect Rapid Modeler for SAP Applications ist ein Assistent, der Metadaten aus einem SAP-System in ein logisches Datenmodell von IBM InfoSphere Data Architect extrahiert.

  • SAP-System auswählen
    Auf dieser Seite können Sie das SAP-System auswählen und konfigurieren, das für das Extrahieren von Metadaten verwendet werden soll.
  • Aus vordefinierten SAP-Modellen auswählen
    Auf dieser Seite können Sie vordefinierte SAP-Modelle aus einer Baumstruktur auswählen.
  • SAP nach Tabellen durchsuchen
    Auf dieser Seite können Sie zu den logischen Tabellen navigieren und die Seiten auswählen, aus denen Sie Metadaten importieren wollen.
  • SAP nach IDocs durchsuchen
    Auf dieser Seite können Sie nach IDoc-Typen suchen, die in Ihr logisches Datenmodell importiert werden sollen. Sie können für jedes logisches Datenmodell einen IDoc-Typ extrahieren.
  • Tabellen manuell angeben
    Auf dieser Seite können Sie diee Namen von zu importierenden Tabellen manuell angeben.
  • IDoc-Typen und -Segmente manuell angeben
    Auf dieser Seite können Sie IDoc-Typen und -Segmente manuell angeben.
  • Verbindungen zu SAP-Systemen verwalten
    Auf dieser Seite können Sie Verbindungen zu SAP-Systemen verwalten.
  • Weitere technische Felder
    Auf dieser Seite können Sie weitere technische Felder oder Attribute konfigurieren. Diese Attribute werden jeder Entität hinzugefügt, die im logischen Datenmodell erstellt wurde.
  • Paket im logischen Datenmodell für den Import auswählen
    Auf dieser Seite können Sie andere Pakete als das Standardpaket im logischen Datenmodell für den Import von Metadaten auswählen.
  • Zusammenfassung des Metadatenimportprozesses
    Auf dieser Seite wird eine Zusammenfassung des Metadatenimportprozesses angezeigt, wie z. B. die Anzahl erstellter Entitäten oder die Anzahl durchgeführter Namensänderungen. Klicken Sie auf Fertig stellen, um die Metadaten in das logische Datenmodell zu importieren.
  • Logische Datenmodelle aus SAP erneut extrahieren
    Über diese Option können Sie die SAP-Metadaten in ein neues logisches Datenmodell erneut extrahieren. Auf der Seite werden die zuvor für das Modell ausgeführten Importe angezeigt, zusammen mit einer kurzen Beschreibung von importierten Objekten.
