Opções de importação de IDoc

Use essa página para configurar o modo como os tipos de IDoc são importados para um modelo de dados lógicos. Cada segmento é mapeado para uma tabela relacional, e campos adicionais são usados para estabelecer o relacionamento pai-filho da hierarquia de segmento de IDoc.

Campos de segmento de IDoc
Especifique como diferentes tipos de campos de segmento de IDoc são mapeados para atributos no modelo de dados lógicos:
Tornar todos os campos de segmento anuláveis
Configura o sinalizador anulável em todos os campos.
Permitir todos os tipos para todas as colunas
Os tipos dos atributos são derivados dos tipos de dados dos campos de segmento, como VARCHAR ou DATE.
Usar apenas VARCHAR
Use VARCHAR como tipo para todos os atributos. Se essa opção estiver selecionada, será possível definir um fator de comprimento para o qual o comprimento original é multiplicado. Essa opção será útil se você quiser usar as tabelas resultantes para suspender temporariamente os dados cujo comprimento exceda o máximo definido pelo SAP.
Opções da tabela de verificação
Especifique as opções para extrair as tabelas de verificação relacionadas que contenham valores válidos para determinados campos.
Importar metadados para tabelas de verificação relacionadas
Importe todas as tabelas de verificação para os campos de segmento IDoc importados.
Importar metadados para tabelas de texto relacionadas
Importe a tabela de texto relacionada para cada tabela de verificação. Essas tabelas de texto contêm descrições dos valores de tabelas de verificação.
Verificar lista de bloqueio da tabela
Um arquivo que contém a lista de tabelas de verificação a serem evitadas durante a importação da tabela de verificação. Especifique um nome de tabela por linha.
