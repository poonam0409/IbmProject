IBM InfoSphere Data Architect Rapid Modeler para Aplicativos SAP

IBM® InfoSphere Data Architect O assistente Rapid Modeler para Aplicativos SAP extrai metadados de um sistema SAP para um modelo de dados lógicos do IBM InfoSphere Data Architect.

  • Selecionar Sistema SAP
    Use essa página para selecionar e configurar o sistema SAP que você deseja usar para extrair metadados.
  • Página selecionar modelos SAP predefinidos
    Use essa página para selecionar modelos SAP predefinidos em uma árvore.
  • Procurar SAP para tabelas
    Use essa página para pesquisar e selecionar as tabelas lógicas das quais você deseja importar metadados
  • Procurar SAP para IDocs
    Use essa página para pesquisar tipos de IDoc que você deseja importar para o modelo de dados lógicos. É possível extrair um tipo de IDoc por modelo de dados lógicos.
  • Especificar tabelas manualmente
    Use essa página para especificar manualmente os nomes das tabelas a serem importadas.
  • Especifique os tipos de IDoc e segmentos manualmente
    Use essa página para especificar os tipos de IDoc e segmentos manualmente.
  • Gerenciar Conexões do Sistema SAP
    Use esta página para gerenciar conexões com sistemas SAP.
  • Campos Técnicos Adicionais
    Use essa página para configurar campos técnicos ou atributos adicionais. Esses atributos são incluídos a cada entidade criada no modelo de dados lógicos.
  • Selecionar um pacote no modelo de dados lógicos para importação
    Use esta página para selecionar pacotes que não sejam padrão no modelo de dados lógicos para importar metadados.
  • Resumo de Importação de Metadados
    Essa página exibe um resumo do processo de importação de metadados como o número de entidades criadas ou o número de mudanças de nome executadas. Clique em Concluir para importar os metadados para o Modelo de Dados Lógicos.
  • Extrair novamente o modelo de dados lógicos do SAP
    Use esta opção para extrair novamente os metadados SAP em um novo modelo de dados lógicos. A página exibe as importações anteriores executadas no modelo e juntamente com uma descrição curta de objetos que foram importados.
