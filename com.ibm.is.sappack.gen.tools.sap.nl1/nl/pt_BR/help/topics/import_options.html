Opções de importação da tabela

Use essa página para configurar as tabelas selecionadas para serem importadas para um modelo de dados lógicos.

Tabelas de texto e verificação
Importar metadados para tabelas de verificação relacionadas
Selecione essa opção se você quiser que o assistente importe as tabelas de verificação relacionadas às tabelas selecionadas. Por exemplo, se uma tabela tiver um campo de código de país, o assistente extrairá os metadados da tabela de verificação relacionada que contém todos os valores de código de país válidos. Se selecionar essa opção, será possível especificar estas opções adicionais:
Coletar tabelas de verificação recursivamente
Selecione essa opção para extrair metadados das tabelas de verificação das tabelas de verificação. Use essa opção com cuidado, pois pode fazer com que muitas tabelas sejam extraídas.
Importar metadados para tabelas de texto relacionadas
Selecione essa opção para importar adicionalmente as tabelas de texto correspondentes para as tabelas de verificação. Por exemplo, para a tabela de verificação que contém o código de país válido como EUA, a tabela de texto correspondente contém o nome de país Estados Unidos.
Verificar lista de bloqueio da tabela
Arquivo de Lista de Bloqueio
Esse arquivo contém uma lista de tabelas de verificação a serem excluídas durante a importação da tabela de verificação. Verifique se esse arquivo contém um nome de tabela por linha separada pelo retorno de linha.
