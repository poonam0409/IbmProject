Gerenciar Conexões do Sistema SAP

Use esta página para gerenciar conexões com sistemas SAP.

Conexões existentes
Exibe todas as conexões SAP existentes. Selecione uma conexão para modificar os parâmetros de conexão ou excluir uma conexão.
Detalhes da Conexão
Exibe os parâmetros de conexão para a conexão selecionada:
Nome do sistema SAP
Nome da conexão com um sistema SAP específico.
Usar balanceamento de carga
Selecione essa opção para permitir o balanceamento de carga usando um servidor de mensagem entre os sistemas SAP e especifique as opções relevantes/
Host SAP
Nome do host de aplicativos do sistema SAP.
Número do sistema SAP
Número de dois dígitos do sistema SAP.
Número do cliente SAP
ID numérico do cliente a ser usado durante a conexão com o sistema SAP.
Cadeia do roteador
Sequência do roteador do sistema SAP.
Usuário SAP
Nome do usuário SAP da conexão selecionada.
Idioma do Usuário SAP
Especifique o idioma a ser usado na criação de log no sistema SAP.
Senha do usuário SAP
Senha do usuário SAP. A senha não é armazenada no computador cliente. É solicitado que o usuário SAP insira a senha uma vez por sessão no InfoSphere Data Architect.
