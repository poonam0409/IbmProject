Sugerencias y consejos para la plantilla Inicio rápido de Express

La plantilla Inicio rápido de Express proporciona un punto de partida para crear guías de inicio rápido que puede utilizar con los productos de la marca Express.

Esta plantilla tiene código que puede utilizar para iniciar la instalación de un producto, visualizar documentación y proporcionar contenido adaptado al entorno local.

Consideraciones sobre la plantilla Inicio rápido de Express:

  • launchpadContent/global.properties define todos los mandatos y programas.
  • disk1 en la página Diseño de disco incluye aplicaciones de ejemplo. Para añadir programas de instalación personalizados o aplicaciones personalizadas, defina la aplicación en launchpadContent\global.properties, siguiendo los ejemplos de las aplicaciones de ejemplo. Importe las aplicaciones en el disco adecuado del proyecto utilizando los controles de la pestaña Diseño de disco.
  • Todos los archivos HTML utilizan la misma hoja de estilo, denominada content.css, que reside en el directorio launchpadContent.

La plantilla Inicio rápido de Express contiene las páginas siguientes:

  • Bienvenida - La página de bienvenida normalmente proporciona una visión general del producto y una descripción de las características de la guía de inicio de rápido. La página de bienvenida debe ser concisa, pero debe ofrecer al usuario opciones para buscar más información acerca del producto mediante un enlace al sitio web del mismo, que se visualizará en una nueva ventana del navegador. La página de bienvenida contiene código para abrir un sitio web externo en un navegador. Utilice la función viewPage() para abrir ventanas nuevas en lugar de utilizar la función window.open() de JavaScript, pues es más probable que window.open() sea interceptado por un bloqueador de ventanas emergente.
    • La página de bienvenida también contiene código para visualizar un documento PDF y ejecutar una aplicación.
  • Acceder al software - La página Acceder al software normalmente alberga el contenido del producto y un enlace que apunta al documento de descarga.
  • Evaluar la configuración de hardware y del sistema - Esta página proporciona habitualmente información sobre requisitos del sistema y contiene un enlace que apunta al documento de requisitos detallados del sistema. Esta página contiene código para abrir un sitio web externo en un navegador.
  • Revisar la arquitectura base y los pasos de instalación - Esta página proporciona habitualmente un diagrama de la arquitectura base del producto descrito.
  • Acceder a la documentación - Esta página proporciona habitualmente información sobre cómo acceder a la documentación incluida con el producto. Esta página contiene código para abrir páginas HTML tanto locales como externas en un navegador.
  • Instalar el producto - La página de instalación del producto proporciona habitualmente un control para iniciar la instalación de un producto. La página de instalación ofrece código para iniciar la instalación de un producto.
  • Salir - El enlace de salida del menú de navegación permite salir de la guía de inicio rápido.
