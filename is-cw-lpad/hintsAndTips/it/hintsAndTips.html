Consigli e suggerimenti per Express QuickStart Template

Express QuickStart Template fornisce un punto di avvio per creare guide di avvio rapido che è possibile utilizzare con i prodotti Express.

Questo modello contiene il codice che è possibile utilizzare per avviare l'installazione di un prodotto, visualizzare la documentazione e fornire il contenuto localizzato.

Considerazioni su Express QuickStart Template:

  • launchpadContent/global.properties definisce tutti i comandi e i programmi.
  • disk1 nella pagina Layout dischi include applicazioni di esempio. Per aggiungere programmi o applicazioni di un'installazione personalizzata, definire l'applicazione in launchpadContent\global.properties, seguendo gli esempi per le applicazioni di esempio. Importare le applicazioni sul disco appropriato del progetto utilizzando i controlli rilevati nella scheda Layout dischi.
  • Tutti i file HTML utilizzano lo stesso foglio di stile denominato content.css, che si trova nella directory launchpadContent.

Express QuickStart Template contiene le seguenti pagine:

  • Benvenuti - Di solito, la pagina Benvenuti fornisce una panoramica del prodotto ed una descrizione delle funzioni della guida di avvio rapido. La pagina di benvenuto deve essere sintetica ed offrire le opzioni dell'utente per reperire ulteriori informazioni sul prodotto offrendo un collegamento al sito Web del prodotto visualizzato in una nuova finestra del browser. La pagina di benvenuto è caratterizzata dal codice per visualizzare un sito Web esterno in un browser. Utilizzare la funzione viewPage() per visualizzare le nuove finestre invece di window.open() di JavaScript poiché window.open() appare bloccato dal blocco popup.
    • La pagina di benvenuto presenta anche codici per visualizzare un PDF ed eseguire un'applicazione.
  • Accedi al software - L'accesso alla pagina del software in genere contiene i contenuti del prodotto e un link al documento di download.
  • Valuta la configurazione dell'hardware e del sistema - Questa pagina in genere fornisce informazioni sui requisiti del sistema, fornendo un link alla documentazione dettagliata dei requisiti del sistema. Questa pagina presenta un codice per visualizzare un sito web esterno in un browser.
  • Rivedi l'architettura base e i passi dell'installazione - Questa pagina in genere fornisce un diagramma dell'architettura base del prodotto disponibile.
  • Accedi alla documentazione - L'accesso al contenuto della documentazione in genere fornisce informazioni sulla modalità di accesso di un utente alla documentazione inclusa con il prodotto. Questa pagina presenta codici per visualizzare pagine locali e HTM esterne in un browser.
  • Installa il prodotto - La pagina dell'installazione del prodotto in genere fornisce un controllo per l'avvio dell'installazione di un prodotto. La pagina di installazione è caratterizzata dal codice per avviare un'installazione del prodotto.
  • Esci - Il link di uscita sul menu di navigazione chiude la guida di avvio rapido.
