Hinweise und Tipps für Express QuickStart Template

Express QuickStart Template bietet Ihnen einen Ausgangspunkt für die Erstellung von Handbüchern für den Schnelleinstieg, das Sie in Verbindung mit Produkten der Marke 'Express' verwenden können.

Diese Schablone verfügt über Code, den Sie zum Starten einer Produktinstallation, zur Anzeige der Dokumentation und zur Bereitstellung lokalisierter Inhalte verwenden können.

Wichtige Hinweise zu Express QuickStart Template:

  • launchpadContent/global.properties definiert alle Befehle und Programme.
  • Datenträger 1 im Disk Layout-Seite beinhaltet Musteranwendungen. Um angepasste Installationsprogramme oder -anwendungen hinzuzufügen, definieren Sie die Anwendung in 'launchpadContent\global.properties' nach den Beispielen für die Musteranwendung. Importieren Sie die Anwendungen zum entsprechenden Datenträger im Projekt und verwenden Sie hierzu die Steuerelemente der Datenträger-Layout-Registerkarte.
  • Alle HTML-Dateien verwenden dasselbe Style-Sheet namens 'content.css', das sich im Verzeichnis 'launchpadContent' befindet.

Express QuickStart Template enthält die folgenden Seiten:

  • Willkommen - Die Begrüßungsseite bietet in der Regel einen Überblick über das Produkt und eine Erläuterung der Funktionen des Handbuchs für den Schnelleinstieg. Die Begrüßungsseite sollte kurz gefasst sein und dem Benutzer dennoch Optionen bieten, um weitere Informationen zum Produkt zu finden, indem ein Link zur Produkt-Website bereitgestellt wird, der in einem neuen Browser-Fenster angezeigt wird. Die Begrüßungsseite verfügt über Code, mit dem Sie eine externe Website in einem Browser laden können. Verwenden Sie statt der JavaScript-Funktion window.open() die Funktion viewPage() zur Anzeige neuer Fenster, da die Wahrscheinlichkeit höher ist, dass window.open() von einem Blockierungsprogramm für Popups geblockt wird.
    • Die Begrüßungsseite enthält auch Code für die Anzeige einer PDF-Datei und die Ausführung einer Anwendung.
  • Greifen Sie auf Ihre Software zu - Ihre Softwareseite enthält normalerweise die Inhalte des Produkts und einen Link zum Download-Dokument, auf die Sie zugreifen können.
  • Untersuchen Sie Ihre Hardware- und Systemkonfiguration - Diese Seite stellt normalerweise Informationen zu den Systemanforderungen oder einen Link zu dem Dokument mit den detaillierten Systemvoraussetzungen bereit. Diese Seite verfügt über Code, mit dem Sie eine externe Website in einen Browser laden können.
  • Überprüfen Sie die Basisarchitektur und die Installationsschritte - Diese Seite verfügt normalerweise über ein Diagramm der Basisarchitektur des verfügbaren Produkts.
  • Zugriff auf Ihre Dokumentation - Greifen Sie auf Ihren Dokumentationsinhalt zu, der normalerweise Informationen dazu bereitstellt, wie ein Benutzer auf die im Produkt enthaltene Dokumentation zugreifen kann. Diese Seite verfügt über Code, mit dem Sie sowohl lokale als auch externe HTML-Seiten in einen Browser laden können.
  • Installieren Sie das Produkt - Die Seite für die Produktinstallation enthält für gewöhnlich ein Steuerelement für die Einleitung einer Produktinstallation. Die Installationsseite enthält Code für den Start einer Produktinstallation.
  • Beenden - Mit dem Link für die Beendigung im Navigationsmenü wird das Handbuch für den Schnelleinstieg verlassen.
