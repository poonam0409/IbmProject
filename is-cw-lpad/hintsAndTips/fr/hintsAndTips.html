Astuces et conseils pour le modèle Express QuickStart

Le modèle Express QuickStart vous fournit un point de départ pour la création de guides de démarrage rapide à utiliser avec les produits Express.

Ce modèle contient du code que vous pouvez utiliser pour lancer le programme d'installation d'un produit, afficher une documentation et fournir un contenu localisé.

Remarques à propos du modèle Express QuickStart :

  • launchpadContent/global.properties définit toutes les commandes et programmes.
  • disk1, dans la page Disposition du disque, comprend des modèles d'application. Pour ajouter des applications ou des programmes d'installation personnalisés, définissez l'application dans launchpadContent\global.properties ,en suivant les exemples des exemples d'application. Importez les applications sur le disque approprié dans le projet à l'aide des commandes de l'onglet de disposition du disque.
  • Tous les fichiers HTML utilisent la même feuille de style nommée content.css, qui se trouve dans le répertoire launchpadContent.

Le modèle Express QuickStart contient les pages suivantes :

  • Bienvenue - La page de bienvenue offre généralement une présentation du produit et une explication des fonctions du guide de démarrage rapide. Elle doit être concise, mais proposer cependant à l'utilisateur des options permettant de rechercher des informations supplémentaires sur le produit en offrant un lien vers le site Web du produit qui est affiché dans une nouvelle fenêtre de navigateur. La page de bienvenue comprend un code permettant d'afficher un site Web externe dans un navigateur. Utilisez la fonction viewPage() pour afficher les nouvelles fenêtres au lieu de la fonction JavaScript window.open() car cette dernière peut être bloquée par une fonction de blocage des fenêtres en incrustation.
    • La page de bienvenue contient aussi un code permettant d'afficher un PDF et d'exécuter une application.
  • Accès à votre logiciel - La page d'accès à votre logiciel comprend les contenus du produit et un lien vers le document à télécharger.
  • Evaluer votre configuration matérielle et système - Cette page fournitles informations à propos des exigences systèmes, fournissant un lien vers le document des exigences systèmes détaillées. Cette page comprend du code permettant d'afficher un site Web externe dans un navigateur.
  • Vérifier l'architecture de base et les étapes d'installation - Cette page fournit un diagramme de l'architecture de base du produit référencé.
  • Accès à votre documentation - Le contenu d'Accès à votre documentation fournit des informations sur la façon, pour un utilisateur, d'accèder à la documentation inclue dans le produit. Cette page comprend du code permettant d'afficher des pages HTML locales et externes dans un navigateur.
  • Installer le produit - La page d'installation du produit fournit généralement une commande permettant de lancer l'installation du produit. La page d'installation contient un code permettant de lancer l'installation du produit.
  • Quitter - Le lien Quitter du menu de navigation permet de sortir du guide de démarrage rapide.
