易捷快速入门模板的提示与技巧

易捷快速入门模板为您提供了一个起点，帮助创建可用于易捷品牌产品的快速入门指南。

该模板中包含可用于启动产品安装、显示文档和提供本地化内容的代码。

有关“易捷快速入门模板”的注意事项：

  • launchpadContent/global.properties 定义所有的命令和程序。
  • “光盘布局”页面中的 disk1 包含样本应用程序。 要添加定制的安装程序或应用程序，请在 launchpadContent\global.properties 中定义应用程序（按照样本应用程序中的示例操作）。 通过使用光盘布局选项卡上的控件，将应用程序导入到项目中相应的光盘上。
  • 所有 HTML 文件都使用同一个名为 content.css 的样式表，该 css 位于 launchpadContent 目录。

“易捷快速入门模板”包含以下页面：

  • 欢迎 -“欢迎”页面通常提供产品的概述和快速入门指南各功能的说明。 “欢迎”页面应该简单明了，但仍应提供一些选项以便用户可以查找有关产品的更多信息，方法是提供产品 Web 站点的链接，以在新的浏览器窗口中显示这些站点。“欢迎”页面包含用于在浏览器中显示外部 Web 站点的代码。“欢迎”页面使用 viewPage() 函数来显示新窗口， 而不是使用 JavaScript window.open()， 因为 window.open() 很有可能被弹出窗口阻止程序所阻止。
    • “欢迎”页面也包含用于查看 PDF 和运行应用程序的代码。
  • 访问软件 - “访问软件”页面通常包含产品的内容和指向下载文档的链接。
  • 评估硬件和系统配置 - 该页面通常提供系统需求相关信息，包括指向详细系统需求文档的链接。 该页面包含用于在浏览器中打开外部 Web 站点的代码。
  • 检查基础体系结构和安装步骤 - 该页面通常提供特定产品的基础体系结构图。
  • 访问文档 - “访问文档”内容通常提供有关用户如何访问产品所含文档的信息。 该页面包含用于在浏览器中打开本地和外部 HTML 页面的代码。
  • 安装产品 - 产品安装页面通常提供用于启动产品安装的控件。 安装页面包含用于启动产品安装的代码。
  • 退出 - 导航菜单上的“退出”链接用于退出快速入门指南。
