Sugestões e Dicas para o Modelo de Início Rápido do Express

O Modelo de Início Rápido do Express fornece um ponto inicial para criação de guias de início rápido que podem ser utilizados com produtos da marca Express.

Este modelo possui um código que pode ser utilizado para iniciar uma instalação do produto, exibir a documentação e fornecer o conteúdo localizado.

Pontos a serem observados sobre o Modelo de Início Rápido do Express:

  • launchpadContent/global.properties define tudo sobre os comandos e os programas.
  • O disco1 na página Layout do Disco inclui aplicativos de amostra. Para incluir programas ou aplicativos de instalação customizados, defina o aplicativo em launchpadContent\global.properties, seguindo os exemplos para os aplicativos de amostra). Importe os aplicativos no disco apropriado do projeto, utilizando os controles localizados na guia de layout do disco.
  • Todos os arquivos HTML utilizam a mesma folha de estilo chamada content.css, que reside no diretório launchpadContent.

O Modelo de Início Rápido do Express contém as seguintes páginas:

  • Bem-vindo - Geralmente, a página de boas-vindas fornece uma visão geral do produto e uma explicação dos recursos do guia de início rápido. A página de boas-vindas deve ser concisa, porém deve conter as opções do usuário para localização de informações adicionais sobre o produto, apresentando um link para o Web site do produto que é exibido em uma nova janela do navegador. A página de boas-vindas possui código para exibir um Web site externo em um navegador. Utilize a função viewPage() para exibir novas janelas, em vez de window.open() do JavaScript, porque window.open() tem grandes chances de ser bloqueado por um bloqueador de pop-up.
    • A página de boas-vindas também apresenta um código para visualizar um PDF e executar um aplicativo.
  • Acesse o Software - Geralmente, a página de acesso ao software compreende os conteúdos do produto e um link para o documento de download.
  • Avalie a Configuração de Sistema e Hardware - Geralmente, esta página fornece informações sobre os requisitos do sistema, fornecendo um link para o documento de requisitos do sistema detalhado. Esta página apresenta um código para construir um Web site externo em um navegador.
  • Revise a arquitetura de base e as etapas da instalação - Geralmente, esta página fornece um diagrama da arquitetura de base do produto apresentado.
  • Acesse a Documentação - Geralmente, o conteúdo de acesso a documentação fornece informações sobre como um usuário pode acessar a documentação incluída com o produto. Esta página apresenta um código para construir páginas de HTML local e externa em um navegador.
  • Instale o Produto - Geralmente, a página de instalação do produto fornece um controle para iniciar a instalação de um produto. A página de instalação apresenta código para iniciar a instalação de um produto.
  • Sair - O link de saída no menu de navegação fecha o guia de início rápido.
