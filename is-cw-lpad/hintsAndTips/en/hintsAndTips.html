Hints and Tips for the Express QuickStart Template

The Express QuickStart Template provides you with a starting point for creating quick start guides that you can use with Express branded products.

This template has code that you can use to start a product installation, display documentation, and provide localized content.

Things to note about the Express QuickStart Template:

  • launchpadContent/global.properties defines all of the commands and programs.
  • disk1 in the Disk Layout page includes sample applications. To add custom installation programs or applications, define the application in launchpadContent\global.properties, following the examples for the sample applications. Import the applications to the appropriate disk in the project using the controls found on the disk layout tab.
  • All HTML files use the same stylesheet named content.css, which resides in the launchpadContent directory.

The Express QuickStart Template contains the following pages:

  • Welcome - The welcome page typically provides an overview of the product and an explanation of the features of the quick start guide. The welcome page should be concise, yet offer the user options for finding more information about the product by offering a link to the product Web site that is displayed in a new browser window. The welcome page features code to bring up an external Web site in a browser. Use the viewPage() function to display new windows instead of JavaScript's window.open() because window.open() is more likely to be blocked by a popup blocker.
    • The welcome page also features code to view a PDF and run an application.
  • Access your software - The access your software page typically contains the contents of the product and a link to the download document.
  • Evalute your hardware and system configuration - This page typtically provides information about system requirements, providing a link to the detailed system requirements document. This page features code to bring up an external Web site in a browser.
  • Review the base architecture and installation steps - This page typically provides a diagram of the base architecture of the featured product.
  • Access your documentation - Access your documentation content typically provides information about how a user can access the documentation included with the product. This page features code to bring up both local and external HTML pages in a browser.
  • Install the product - The product installation page typically provides a control for initiating a product installation. The installation page features code to start a product installation.
  • Exit - The exit link on the navigation menu exits the quick start guide.
