Express QuickStart テンプレートのヒント

Express QuickStart テンプレートは開始点として準備されています。

このテンプレートには、製品のインストールの開始、資料の表示、 およびローカライズされたコンテンツの提供のために使用できるコードが含まれています。

Express QuickStart テンプレートに関する注意点:

  • launchpadContent/global.properties はすべてのコマンドおよびプログラムを定義します。
  • サンプル・アプリケーションは、「ディスク・レイアウト」ページの DISK1 に入っています。 カスタムのインストール・プログラムやアプリケーションを追加するには、launchpadContent\global.properties に アプリケーションを定義します (サンプル・アプリケーションの例に従ってください)。「ディスク・レイアウト」タブにあるコントロールを使用して、 プロジェクト内の適切なディスクにアプリケーションをインポートします。
  • すべての HTML ファイルは、launchpadContent ディレクトリーにある同じスタイルシート (content.css) を使用します。

Express QuickStart テンプレートには次のページが含まれます。

  • ようこそ - 「ようこそ」ページには通常、製品の概要と クイック・スタート・ガイドの機能の説明が提供されています。 「ようこそ」のページは簡潔にまとめられていますが、ページ内にある製品 Web サイトへのリンクを ご利用になれば、製品の詳細情報を参照することができます。 「ようこそ」ページでは、ブラウザーに外部の Web サイトを表示させるためのコードが用意されています。 JavaScript の window.open() の代わりに、viewPage() 機能を 使用して新規ウィンドウを表示します。window.open() の方がポップアップ・ブロッカーでブロックされる可能性が高いためです。
    • ウェルカム・ページには、PDF の表示およびアプリケーションの実行のためのコードも用意されています。
  • ソフトウェアへのアクセス - 通常「ソフトウェアへのアクセス」ページには、 製品の内容やダウンロードする資料へのリンクなどが含まれています。
  • ハードウェアおよびシステム構成の評価 - 通常このページには システム要件についての情報が表示され、システム要件の詳しい資料へのリンクがご利用になれます。 このページでは、ブラウザーに外部の Web サイトを表示させるためのコードが用意されています。
  • 基本アーキテクチャーおよびインストール・ステップのレビュー - 通常このページには、 表示されている製品の基本アーキテクチャーのダイアグラムが表示されます。
  • 資料へのアクセス - 通常このページには、製品に同梱の資料にアクセスする 方法が表示されます。 このページでは、ブラウザーにローカルおよび外部の HTML ページを表示させるためのコードが用意されています。
  • 製品のインストール - 製品インストールのページには、通常、 製品のインストールを開始するコントロールを提供します。 インストール・ページには、製品のインストールを開始するコードが提供されます。
  • 終了 - ナビゲーション・メニューの終了リンクによって、クイック・スタート・ガイドを終了できます。
