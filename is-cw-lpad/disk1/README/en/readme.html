IBM InfoSphere Information Server Rapid Modeler and Rapid Generator for SAP Applications

Installation and system requirements

The Rapid Modeler and Rapid Generator component is part of the IBM InfoSphere Pack for SAP Applications. It consists of two tiers:
  • Client
    The client tier is a plug-in for InfoSphere Data Architect and is installed separately using IBM Installation Manager.
  • Server
    This server tier is a web application that is installed on the WebSphere Application Server that is part of IBM Information Server. It is installed automatically by the Pack for SAP Applications installer.

Required products and components (client tier)

The client tier is only available for Windows operating systems.
The following table shows the required products and components.

Product Version Remarks
IBM InfoSphere Data Architect 8.1 or 8.5 Required
SAP JCo 3 Required
IBM InfoSphere Information Server (client tier) 9.1.2 or 11.3 Optional, used for job compilation only

Client tier installation

  1. Run launchpad.exe in the root folder of the installation package.
  2. Follow the instructions in the installation wizard and make sure that you select the package group where you installed InfoSphere Data Architect.

Server tier installation

The server tier is installed automatically as part of the Pack for SAP Applications installation.
To check if the server tier was installed successfully, open a browser window and enter the following URL:

https://<ISDomainName>:<ISDomainPort>/cwrapidgenerator/JobGenerator

where <ISDomainName> is replaced by the IS Server domain name and <ISDomainPort> is replaced by the IS HTTPS port number (the default is 9443).
