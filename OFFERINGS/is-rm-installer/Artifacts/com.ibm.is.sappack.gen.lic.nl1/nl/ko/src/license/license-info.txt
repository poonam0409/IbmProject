본 조항의 규정은 본 라이센스에 적용되는 법률에 준해서 유효하지 않거나 시행이 불가능할 경우에는 적용되지 않습니다. 아래 명시된 구성요소는 "제외 구성요소"에 해당합니다. 본 IBM 계약 또는 귀하와 IBM 간의 기타 모든 계약의 조건에도 불구하고, 제외 구성요소에는 다음 조건이 적용됩니다.

(b) 공급자는 어떠한 경우에도 제외 구성요소와 관련하여 기대했던 이익 및 비용 절감이 실현되지 못함으로 인해 발생한 손해, 데이터 분실로 인한 손해를 포함하여, 직접 손해, 간접 손해, 부수적 손해, 특별 손해, 징벌적 손해 또는 결과적 손해에 대해 일체 책임을 지지 않습니다.
(c) IBM과 공급자는 제외 구성요소와 관련된 어떠한 배상 청구에 대해서도 귀하에 대해 책임을 지거나 귀하를 방어하거나 면책하지 않습니다.

DB2 구성요소 

DB2 Connect 구성요소: 본 프로그램에는 DB2 Connect 기술의 일부가 포함되어 있습니다. 귀하는 응용프로그램을 개발하거나 DB2 for z/OS 및 DB2 for iSeries에 의해 관리되는 데이터 오브젝트를 모델링할 목적으로 라이센스를 취득한 본 프로그램과 함께 이들 구성요소의 사본 한 부를 설치하고 사용할 수 있습니다. 기타 다른 용도를 위해서는 본 프로그램에서 제공한 DB2 Connect 기술을 사용할 수 없습니다. 본 프로그램 이외의 다른 응용프로그램에 연결하는 데는 DB2 Connect 기술을 사용할 수 없습니다. 다른 응용프로그램에 대한 연결이 필요한 경우에는 적절한 DB2 Connect 라이센스를 별도로 구입해야 합니다. 

귀하가 DB2 Connect 구성요소를 사용하는 경우 DB2 Connect 구성요소에 동봉되는 라이센스 계약의 조건이 적용됩니다(본 라이센스에서 달리 명시하는 경우는 제외). 기타 다른 목적으로는 DB2 Connect 구성요소를 사용할 수 없습니다. 본 라이센스 계약의 범위를 넘어서 해당 구성요소를 사용하고자 하는 경우에는 DB2 리셀러에 문의하여 정식 DB2 Connect 라이센스를 취득해야 합니다. 

IBM WEBSPHERE APPLICATION SERVER 

WEBSPHERE APPLICATION SERVER: 본 프로그램에는 본 프로그램의 "유닛 테스트 환경"에 포함된 WebSphere Application Server 사본을 포함하여 WebSphere Application Server 사본이 여러 개 동봉됩니다. 귀하는 코드가 설계된 대로 작동된다는 것을 확인하기 위해 한 명의 개발자가 작성하였거나 개발한 코드만을 테스트할 수 있는 "유닛 테스트"를 목적으로, 이 개발자가 사용할 수 있도록 본 프로그램의 나머지(유닛 테스트 환경에 있지 않는 모든 WSSD)가 설치된 기계에 WebSphere Application Server의 각 사본을 설치할 수 있습니다. 귀하는 유닛 테스트 라이센스 증서에 의거하여 설치 및 사용된 WebSphere Application Server 구성요소의 사본을 개발자의 기계가 아닌 서버에 있는 응용프로그램을 테스트하거나 실제 업무 워크로드를 모의 실험하기 위한 목적으로 사용하거나, 코드, 응용프로그램 또는 시스템의 확장성(scalability)을 테스트하기 위한 목적으로 사용할 수 없습니다. 실제 업무용으로는 WebSphere Application Server 구성요소를 사용할 수 없습니다. 

소스 코드: WebSphere Application Server의 일부 구성요소는 소스 코드 형식으로 제공될 수 있습니다. 본 계약의 상반되는 어떠한 조항에도 불구하고, 프로그램 서비스는 WebSphere Application Server 패키지에 포함된 이러한 일부 구성요소의 수정되지 않은 2진 코드 버전에 대해서만 제공되며, 귀하가 작성한 구성요소의 수정본이나 이들 구성요소의 소스 코드에 대해서는 제공되지 않습니다. 

J2C 기술 

본 프로그램에는 J2C 커넥터 기술이 포함되어 있습니다. 다음 정보가 적용됩니다. 

IMS: 
본 프로그램에는 IMS TM Resource Adapter(IMS Connector for Java라고도 함)의 개별적인 JCA 1.0 및 JCA 1.5 버전이 포함됩니다. 귀하는 코드를 작성하거나 생성하기 위한 용도로만 이들 자원 어댑터 기능을 사용할 수 있습니다. 단, 본 프로그램으로 작성 또는 생성한 모든 코드에 대해 유닛 테스트를 수행하려면 귀하는 IMS V9에 대한 최소 한 부의 유효한 라이센스를 취득해야 합니다. 실제 업무 용도로는 본 프로그램에서 제공한 IMS TM Resource Adapter를 사용할 수 없습니다. 실제 업무 용도로 IMS TM Resource Adapter를 사용하려면 Java용 IMS Connector 웹 사이트(www.ibm.com/ims)를 참조하십시오.

JCA: 
본 프로그램의 라이센스를 취득한 사용자는 본 프로그램과 함께 'cicseci. rar' 파일에 패키지로 구성된 JCA(J2EE Connector Architecture) 자원 어댑터 및 'ctgclient.jar' 파일에 패키지로 구성된 Java 클라이언트 클래스를 자유롭게 배포할 수 있습니다. 이들 클라이언트 클래스를 배포하는 라이센스 사용자는 해당 클래스를 지원하고 유지보수할 책임이 있습니다. IBM은 본 프로그램의 라이센스 사용자가 다음 사항을 확인하는 경우에 한해 해당 배포하는 행위에 대하여 동의합니다. 

(i) 'JAR' 또는 'RAR' 파일의 원본 양식을 수정하지 않습니다. 
(ii) 'JAR' 또는 'RAR' 파일은 본 프로그램에서 지원하는 Java 환경 외에서는 사용하지 않습니다. 
(iii) 모든 통신은 'ctgclient.jar' 또는 'RAR' 파일 버전 이상의 릴리스 수준에서 CICS Transaction Gateway로 수행됩니다. 

JRPC DEVELOPMENT KIT 

다음 특정 권리 및 제한사항은 본 프로그램에 포함된 JRPC Development Kit 구성요소에 적용됩니다. 본 프로그램을 설치함으로써, 귀하는 다음 조건에 동의하게 됩니다. 

프로그램은 항공기의 온라인 제어, 항공 교통, 항공기 운행 또는 항공기 통신에 사용되거나 핵 시설의 설계, 건축, 운영 또는 유지보수에 사용되도록 설계되거나 의도되지 않았습니다. 라이센스 사용자는 이러한 용도로 본 프로그램을 사용하지 않을 것을 보증합니다.

J2C 기술: PL/I 컴파일러 구성요소 

본 프로그램의 사본에는 Windows용 IBM WebSphere Studio PL/I이 포함된 J2C 커넥터 기술(PL/I 컴파일러 구성요소)이 포함되어 있습니다. 본 프로그램에 들어 있는 PL/I 컴파일러 구성요소는 PL/I 데이터 구조의 메타데이터를 가져오기 위한 용도로 프로그램의 내부 구성요소에 의해서만 사용될 수 있습니다. 귀하는 이 코드를 실제 업무용으로 사용할 수 없습니다.

본 프로그램에는 Microsoft Corporation의 일부 재배포 가능 소프트웨어가 포함되어 있습니다. 본 라이센스 계약에서 언급된 기타 조건에 추가하여, 귀하는 이러한 소프트웨어를 리버스 엔지니어하거나 디컴파일하거나 해체할 수 없습니다(단, 이러한 제한사항에도 불구하고 해당 법률에서 명시적으로 허가하는 경우만은 제외).

귀하는 Microsoft의 사전 서면 승인 없이 Microsoft .Net Framework에 대한 벤치마크 테스트 결과를 제3자에게 공개할 수 없습니다.

어떠한 이유로든 귀하에게 발생한 손해(본 계약에서 언급한 모든 손해 및 계약상 또는 기타 직접적 또는 통상 손해 포함, 단 이에 한하지 않음)에 대해서는 본 라이센스 계약의 조항에 의거하여 귀하가 실제로 지출한 비용을 한도로 귀하에게 발생한 소프트웨어와의 상당한 인과관계가 인정되는 실손해와 미화 $5.00 중 많은 금액을 한도로 MICROSOFT 또는 그 공급자에게 책임이 있으며 이것이 귀하의 유일한 구제수단입니다. 위의 제한사항, 제외사항 및 면책사항은 구제조치가 적용되지 않는 경우라 하더라도 해당 법률에 의해 허용되는 최대한의 범위로 적용합니다. 

샘플 갤러리 

본 프로그램에는 프로그래밍 기술을 예시해 주는 샘플 소스 코드 또는 프로그램이 들어 있습니다. 이들 샘플은 샘플 갤러리에 있습니다. 귀하는 이러한 샘플이나 샘플 수정본을 어떠한 형식으로든지 내부적으로 또는 귀하의 응용프로그램이나 관련 문서의 일부로서 복사, 수정 및 배포할 수 있습니다. 이러한 샘플은 모든 조건 하에서 테스트된 것은 아니며, 아무런 지원 없이 IBM에 의해 귀하에게 제공됩니다. IBM은 본 샘플을 "현상태대로" 제공하나 강행 법규에 의해 규정된 보증은 적용됩니다.
