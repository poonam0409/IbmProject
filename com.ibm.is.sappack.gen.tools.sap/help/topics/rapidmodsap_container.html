InfoSphere Rapid Modeler for SAP Applications

InfoSphere™ Rapid Modeler for SAP Applications wizard imports metadata from an SAP system into a logical data model of IBM InfoSphere Data Architect.

  • Select SAP system
    Use this page to select and configure the SAP system you want to use to import metadata.
  • Select from predefined SAP models page
    Use this page to select predefined SAP models from a tree.
  • Browse SAP for tables
    Use this page to browse and select the logical tables from which you want to import metadata
  • Browse SAP for IDocs
    Use this page to search for IDoc types that you want to import into your logical data model. You can import one IDoc type per logical data model.
  • Specify tables manually
    Use this page to manually specify the names of tables to be imported.
  • Specify IDoc types and segments manually
    Use this page to specify IDoc types and segments manually.
  • Manage SAP system connections
    Use this page to manage connections to SAP systems.
  • Additional technical fields
    Use this page to configure additional technical fields or attributes. These attributes are added to every entity created in the logical data model.
  • Select a package in logical data model to import
    Use this page to select packages other than the default package in the logical data model to import metadata.
  • Metadata import summary
    This page displays a summary of the metadata import process such as the number of entities created or the number of name changes performed. Click Finish to import the metadata into the Logical Data Model.
  • Re-import logical data model from SAP
    Use this option to re-import the SAP metadata into a new logical data model. The page displays the previous imports performed on the model and along with a short description of objects that were imported.
