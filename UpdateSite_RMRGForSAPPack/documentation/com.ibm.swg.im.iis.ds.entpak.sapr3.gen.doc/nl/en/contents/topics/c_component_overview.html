Overview of the components

The Rapid Modeler and the Rapid Generator for SAP clients are implemented as extensions (Eclipse plug-ins) to IBM® InfoSphere™ Data Architect.

For connectivity with SAP, the SAP JCo library, a Java library that provides connectivity to an SAP system is used. The Rapid Modeler for SAP consists only this client component. The Rapid Generator for SAP consists of the client component that collects user input for generating jobs. The Rapid Generator also uses the IBM InfoSphere Information Server client for compiling the generated jobs. The actual job generation is performed by the component on the InfoSphere Information Server. This component is an enterprise application (EAR file) which is installed on the WebSphere® Application Server of the InfoSphere Information Server domain.
