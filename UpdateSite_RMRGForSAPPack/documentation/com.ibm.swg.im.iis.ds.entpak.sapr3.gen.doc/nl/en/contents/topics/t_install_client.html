Installing the client

Use the installation program to install the InfoSphere™ Rapid Modeler for SAP and the InfoSphere Rapid Generator for SAP client.

Procedure


Pre-install Instructions
  1. There shouldn't be any RMRG component for "IBM InfoSphere Information Server Pack for SAP Applications" installed on IDA using "Installation Manager". If needed, uninstall the existing application using "Installation Manager"

Installation Instructions
  1. Place the ISPackForSAP_RMRG.zip file to a temporary directory
  2. Open IDA application then click on Help -> Install New Software
  3. In “Install” window click on “Add”
  4. Click on “Archive”. Browse to the ISPackForSAP_RMRG.zip file placed at temporary location
  5. Un-check “Group items by category” if you can’t see the items in the grid
  6. Click on "Select All" then click on Next -> Next. Accept the terms of license agreement
  7. Click on "Finish", then OK
  8. Restart IDA

Uninstallation Instructions
  1. Open IDA and click on Help -> Install New Software
  2. Click on "What is already installed" hyperlink
  3. In the "Installed software" tab, select the SAP Pack component you want to uninstall, then click on the button "Uninstall"
  4. After successful uninstall, restart IDA
