Creating a connection to the SAP system

The Rapid Generator for SAP requires an existing and configured connection to an SAP system. This is a standard connection as used in the IBM® InfoSphere™ Information Server Pack for SAP Applications in InfoSphere DataStage®.

Procedure

  1. Click Start > Programs > IBM InfoSphere Information Server > IBM InfoSphere Information Server Pack for SAP Applications > DataStage Administrator for SAP.
  2. Open the project that you created, and create a connection to your SAP system.
  3. In the DataStage Connections to R/3 tab, click Add > New. The Connection Properties dialog is displayed.
  4. In the Connection and Logon Details tab, enter all required fields.
  5. To generate IDoc extraction jobs, go to the IDoc Listener settings tab, and ensure that the option Listen for IDocs received through this connection is selected.
