Configuring the InfoSphere Information Server client

To use Rapid Generator for SAP to generate InfoSphere™ DataStage® jobs, the InfoSphere Information Server client must be installed on the same system as InfoSphere Data Architect.

Procedure

  1. Click Start > Programs > IBM InfoSphere Information Server.
  2. Right-click IBM InfoSphere DataStage and Quality Stage Designer.
  3. Copy the path specified in the Target field, until but not including Clients\Classic\.....
  4. Select Preferences > IBM InfoSphere Information Server Pack for SAP Applications > Rapid Generator for SAP.
  5. Paste the path copied from the previous step into Information Server install directory field.
  6. Click Check to verify that the specified values are valid.
