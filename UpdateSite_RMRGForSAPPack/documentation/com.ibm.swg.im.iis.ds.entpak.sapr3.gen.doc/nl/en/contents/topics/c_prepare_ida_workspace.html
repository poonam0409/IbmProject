Preparing the InfoSphere Data Architect workspace

Before you use the InfoSphere™ Rapid Modeler and the InfoSphere Rapid Generator for SAP Applications, you must prepare the InfoSphere Data Architect workspace.

  • Configuring the SAP JCo connector
    Specify the custom configuration settings to configure the JCo library that connects to SAP.
  • Configuring the InfoSphere Information Server client
    To use Rapid Generator for SAP to generate InfoSphere DataStage jobs, the InfoSphere Information Server client must be installed on the same system as InfoSphere Data Architect.
