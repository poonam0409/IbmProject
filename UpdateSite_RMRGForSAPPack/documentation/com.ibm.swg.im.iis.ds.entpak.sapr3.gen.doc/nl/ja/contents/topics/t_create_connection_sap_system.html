SAP システムへの接続の作成

Rapid Generator for SAP には、SAP システムへの既存の構成済み接続が必要です。これは、InfoSphere™ DataStage® の IBM® InfoSphere Information Server Pack for SAP R/3 で使用される接続と同じ標準接続です。

手順

  1. 「スタート」 > 「プログラム」 > 「IBM InfoSphere Information Server」 > 「IBM InfoSphere Information Server Pack for SAP R3」 > 「DataStage Administrator for SAP」をクリックします。
  2. 作成したプロジェクトを開き、SAP システムへの接続を作成します。
  3. 「R/3 への DataStage 接続」タブで、「追加」 > 「新規」をクリックします。「接続プロパティー」ダイアログが表示されます。
  4. 「接続とログオンの詳細」タブで、すべての必須フィールドに入力します。
  5. IDoc Extract ジョブを生成するには、「IDoc リスナー設定」タブに移動し、「この接続を使用して受信される IDoc を listen する」オプションを必ず選択してください。
