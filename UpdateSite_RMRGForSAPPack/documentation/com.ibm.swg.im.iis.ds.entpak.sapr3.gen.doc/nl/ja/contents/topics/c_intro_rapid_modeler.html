IBM InfoSphere Data Architect Rapid Modeler for SAP Applications の概要

IBM® InfoSphere™ Data Architect Rapid Modeler for SAP Applications は、SAP システムから IBM InfoSphere Data Architect の論理データ・モデルにメタデータを抽出するウィザードです。

次の種類のメタデータを抽出できます。

論理テーブル
SAP システムは、そのビジネス・データを、ABAP (Advanced Business Application Programming) コードを使用してアクセスされるテーブルに保管します。 例えば、KNA1、KNB1、および KNVV テーブルには、顧客マスター・データの一部が含まれています。MARA および MARC テーブルには、品目マスター・データのデータが含まれています。IBM InfoSphere Data Architect Rapid Modeler for SAP は、SAP システム内のテーブルのリストを識別します。その後、これらのテーブルは、各 SAP テーブルのエンティティーおよび各列のそれぞれの属性を作成することによって、論理データ・モデルにインポートされます。
IDoc
SAP の IDoc は、他のシステムとの通信に使用される、階層型で非リレーショナルなデータ構造です。 IDoc には、階層の構造を定義するタイプがあります。 各階層レベルをセグメントと呼びます。例えば、DEBMAS IDoc タイプは、債務者 (顧客) ビジネス・オブジェクトを表します。これは、基本顧客データを表す E1KNA1M セグメントを含んでいます。 IBM InfoSphere Data Architect Rapid Modeler for SAP を使用すると、SAP システムの IDoc タイプとセグメントを参照できます。各 IDoc セグメントはエンティティーにマップされ、IDoc フィールドは属性にマップされます。階層構造は、階層を表す各エンティティーのテクニカル属性を使用してモデル化されます。
