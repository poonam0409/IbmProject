IBM InfoSphere Information Server Pack for SAP Applications、バージョン 8.0
フィードバック

前提条件

IBM® InfoSphere™ Data Architect Rapid Modeler for SAP および IBM InfoSphere Rapid Generator for SAP クライアントは、IBM InfoSphere Data Architect の拡張であり、IBM Installation Manager を使用して独自のランチパッドと一緒にインストールされます。IBM InfoSphere Rapid Generator for SAP の場合、ジョブを生成するサーバー・コンポーネントをインストールする必要があります。このサーバー・コンポーネントは、InfoSphere Information Server の WebSphere® Application Server にインストールされるエンタープライズ・アプリケーションです。 このアプリケーションは、.ear ファイルとして提供され、標準の WebSphere Application Server と一緒に、例えば Web ブラウザーを使用して管理コンソールから、インストールできます。

クライアントの前提条件

IBM InfoSphere Data Architect Rapid Modeler for SAP および IBM InfoSphere Rapid Generator for SAP クライアントは、Windows のみでサポートされます。

表 1 は、クライアント・システムで使用可能でなければならない前提条件の製品とコンポーネントを示しています。
表 1. クライアント・インストールの前提条件
製品 バージョン 注釈
InfoSphere Data Architect 7.5.3.1 必須
InfoSphere Information Server クライアント 8.5 および 8.7 IBM InfoSphere Rapid Generator for SAP のみに必須
IBM InfoSphere Information Server Pack for SAP Applications クライアント 7.0 IBM InfoSphere Rapid Generator for SAP のみに必須
SAP JCo 3 パッケージ 3.0.x このパッケージを SAP からダウンロードし、sapjco3.jar ファイルと sapjco3.dll ファイルがディレクトリーで使用可能であることを確認してください。

サーバーの前提条件

IBM InfoSphere Data Architect Rapid Modeler for SAP には、InfoSphere Information Server のインストールは必要ありません。 IBM InfoSphere Rapid Generator for SAP は、InfoSphere Information Server の Windows および AIX® プラットフォームのみでサポートされます。

表 2. サーバー・インストールの前提条件
製品 バージョン
InfoSphere Information Server 8.5 または 8.7
IBM InfoSphere Information Server Pack for SAP Applications サーバー 7.0

SAP 環境の前提条件

この構成には、設計時と実行時のユーザー許可プロファイル、および SAP システムにインストールされる機能モジュールが含まれます。 IBM InfoSphere Information Server Pack for SAP Applications に必要なステップに従ってください。 Pack のインストールには、SAP システムにインポートできる転送ファイルが含まれます。別の方法として、手動で SAP システムを構成することもできます。

注:
  • IBM InfoSphere Data Architect Rapid Modeler for SAP を使用するには、SAP ユーザーに必要な設計時の許可があることで十分です。
  • IDoc の抽出とロード (テーブルの抽出なし) のみに IBM InfoSphere Rapid Generator for SAP を使用するには、SAP ユーザーに IDoc 処理の許可があることで十分です。
  • ユーザーに SAP_ALL 特権がある場合、それ以上の構成は不要です。

サポートされるデータベース

IBM InfoSphere Rapid Generator for SAP は、IBM DB2® Database for Linux, UNIX, and Windows および Oracle 用に作成された物理データ・モデルのみで機能します。
