Rapid Modeler を使用した SAP メタデータのインポート

IBM® InfoSphere™ Data Architect Rapid Modeler for SAP ウィザードを使用して、論理テーブルからのメタデータか、SAP からの IDoc タイプを論理データ・モデルにインポートすることができます。

始める前に

  • 論理データ・モデルがデータ設計プロジェクトに存在しなければなりません。
  • IBM InfoSphere Data Architect Rapid Modeler で使用するようにワークスペースが構成されなければなりません。

手順

  1. 既存の「論理データ・モデル」 (LDM ファイル) を右クリックし、「Rapid Modeler for SAP」 > 「SAP からメタデータを抽出 (Extract metadata from SAP)」を選択します
  2. 「SAP システムの選択」から SAP システムを選択します。
  3. インポートするメタデータを選択します。
    • 論理テーブル・メタデータをインポートするには、タスク SAP から論理テーブルをインポート を完了します。
    • IDoc メタデータをインポートするには、タスク SAP からの IDoc タイプのインポート を完了します。
  4. 「インポートするテーブルの検討 (Review tables to be imported)」ページで、インポートしたいテーブルのみが表示されていることを確実にしてから、「次へ」をクリックします。
  5. 「技術分野 (Technical fields)」ページで、論理データ・モデルで作成されるエンティティーごとに追加される 1 組の属性を定義します。例えば、属性は、ソース・システムの ID、またはデータが抽出されたときのタイム・スタンプを表す数値です。
  6. 「次へ」をクリックします。
  7. 論理データ・モデルのデフォルトでないパッケージを使用するには、使用したい論理データ・モデルのパッケージを選択します。
    1. インポート宛先オプションとして「特定のパッケージを使用する (Use a specific package)」を選択します。表には、論理データ・モデル内の既存のすべてのパッケージが表示されます。
    2. パッケージ名をクリックして、インポートの宛先として選択します。
  8. 「次へ」をクリックして、インポート・プロセスを開始します。
  9. 「完了」をクリックして、論理データ・モデルにインポートされた変更を保存します。

タスクの結果

論理データ・モデルで新規エンティティーが作成されます。また、SAP 内のデータ・エレメントごとに、同じ名前を持つアトミック・ドメインが作成されます。
  • SAP から論理テーブルをインポート
    「Rapid Modeler for SAP メタデータのインポート」ウィザードを使用して、SAP 論理テーブルのメタデータをインポートすることができます。
  • SAP からの IDoc タイプのインポート
    「IBM InfoSphere Data Architect Rapid Modeler for SAP メタデータのインポート」ウィザードを使用して、SAP IDoc タイプとセグメントのメタデータをインポートできます。
