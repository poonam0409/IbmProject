ヘルプの入手

Rapid Modeler および Rapid Generator for SAP 用の以下の情報リソースからヘルプを入手できます。

インフォメーション・センター

IBM® InfoSphere™ Information Server Pack for SAP Applications のインフォメーション・センターは、「ヘルプ」 > 「ヘルプ目次」 > 「IBM InfoSphere Information Server Pack for SAP Applications」で利用可能です。

コンテキスト・センシティブ・ヘルプ

すべてのウィザード・ページでは、コンテキスト・センシティブ・ヘルプが用意されています。ヘルプ・ファイルを表示するには、キーボードで F1 を押すか、ウィザード・ページの左下隅にあるヘルプ・シンボルをクリックします。

虎の巻

SAP からのメタデータのインポート、およびインポートされたこのメタデータからのジョブの生成を行うプロセスを説明する、いくつかの虎の巻があります。 それらの虎の巻を利用するには、「ヘルプ」 > 「虎の巻 (Cheat Sheets)」 > 「IBM InfoSphere Information Server Pack for SAP Applications」に進みます。

このセクションでは、次の虎の巻を見つけることができます。
  • IBM InfoSphere Data Architect Rapid Modeler for SAP Applications を使用した、SAP からの論理テーブルのインポート
  • IBM InfoSphere Information Server Rapid Modeler for SAP Applications を使用した、SAP からの IDoc のインポート

例

IBM InfoSphere Information Server Pack for SAP Applications に関連したいくつかの例があります。 これらの例には、既存のデータ設計プロジェクトが必要です。

これらの例を作成するには、「ファイル」 > 「新規」 > 「例」 > 「IBM InfoSphere Information Server Pack for SAP Applications」に進みます。
  • ビジネス・オブジェクトのサンプル・モデル: これは、Rapid Modeler for SAP で使用できる、事前定義された SAP モデルの例です。
  • Rapid Modeler for SAP で使用されるチェック・テーブル・ブラックリストのサンプル・ファイル。 Rapid Modeler for SAP では、テーブルの名前が入っているファイルを使用できます (インポート時に無視される)。『InfoSphere Data Architect Rapid Modeler for SAP を使用した SAP メタデータのインポート』を参照してください。これは、このファイルがどのように表示されるかを示す例です。
  • ABAP 抽出ジョブのために Rapid Generator for SAP で使用される RFC 宛先用のサンプル・ファイル。

    RFC 転送方式を使用する ABAP Extract ステージを使用してジョブを生成するには、RFC 宛先とそれに関連したプログラム ID のリストをファイルで与える必要があります。これは、このファイルがどのように表示されるかを示す例です。
