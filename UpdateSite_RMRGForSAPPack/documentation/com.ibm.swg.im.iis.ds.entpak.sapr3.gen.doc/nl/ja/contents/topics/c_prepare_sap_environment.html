SAP 環境の前提条件

使用する SAP システムは、Rapid Modeler および Rapid Generator for SAP と共に作動するように構成されなければなりません。

この構成には、設計時と実行時のユーザー許可プロファイル、および SAP システムにインストールされる機能モジュールが含まれます。 IBM® InfoSphere™ Information Server Pack for SAP Applications に必要なステップに従ってください。 Pack のインストールには、SAP システムにインポートできる転送ファイルが含まれます。別の方法として、手動で SAP システムを構成することもできます。

注:
  • Rapid Modeler for SAP を使用するには、SAP ユーザーに必要な設計時の許可があることで十分です。
  • IDoc の抽出とロード (テーブルの抽出なし) のみに Rapid Generator for SAP を使用するには、SAP ユーザーに IDoc 処理の許可があることで十分です。
  • ユーザーに SAP_ALL 特権がある場合、それ以上の構成は不要です。
