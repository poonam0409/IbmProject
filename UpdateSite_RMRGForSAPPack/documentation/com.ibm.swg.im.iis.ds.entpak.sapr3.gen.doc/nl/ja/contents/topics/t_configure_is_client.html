InfoSphere Information Server クライアントの構成

Rapid Generator for SAP を使用して InfoSphere™ DataStage® ジョブを生成するには、InfoSphere Information Server クライアントが、InfoSphere Data Architect と同じシステムにインストールされなければなりません。

手順

  1. 「スタート」 > 「プログラム」 > 「IBM InfoSphere Information Server」をクリックします。
  2. 「IBM InfoSphere DataStage and Quality Stage Designer」を右クリックします。
  3. 「ターゲット」フィールドで指定されたパス (Clients\Classic\.... まで、ただしこれを含みません) をコピーします。
  4. 「設定」 > 「IBM InfoSphere Information Server Pack for SAP Applications」 > 「Rapid Generator for SAP」を選択します。
  5. 直前のステップからコピーしたパスを、「Information Server のインストール・ディレクトリー」フィールドに貼り付けます。
  6. 「確認 (Check)」をクリックして、指定値が有効であることを確認します。
