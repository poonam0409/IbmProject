Configuring the SAP JCo connector

Specify the custom configuration settings to configure the JCo library that connects to SAP.

Procedure

  1. Download JCo 3 package from SAP
  2. Copy sapjco3.jar and sapjco3.dll files into a directory.
  3. In InfoSphere™ Data Architect, select Window > Preferences > IBM InfoSphere Information Server Pack for SAP Applications.
  4. Click Configuration.
  5. Select theUse custom JCo configuration settings instead of those provided during the product installation phase check box.
  6. Browse and select the sapjco3.jar file.
  7. Browse and select the directory containing sapjco3.dll.
  8. Click OK, and then Yes to restart the workbench.
  9. Select Window > Preferences > IBM InfoSphere Information Pack for SAP Applications
  10. Click the About JCO connector tab and verify the version numbers.
