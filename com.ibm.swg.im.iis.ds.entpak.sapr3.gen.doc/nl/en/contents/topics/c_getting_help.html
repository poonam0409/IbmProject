Getting help

You can get help from the following information resources for Rapid Modeler and Rapid Generator for SAP.

Information Center

The Information Center for the IBM® InfoSphere™ Information Server Pack for SAP Applications is available in Help > Help contents > IBM InfoSphere Information Server Pack for SAP Applications.

Context-sensitive help

All wizard pages provide context-sensitive help. To display the help file, press F1 on the keyboard, or click the help symbol on the lower left corner of the wizard page.

Cheat sheets

There are a couple of cheat sheets available which guides you through the process of importing metadata form SAP and generating jobs from this imported metadata. To run those cheat sheets go to Help > Cheat Sheets > IBM InfoSphere Information Server Pack for SAP Applications.

In this section, you can find the following cheat sheets:
  • Importing logical tables from SAP by using InfoSphere Rapid Modeler for SAP Applications.
  • Importing IDocs from SAP by using IBM InfoSphere Information Server Rapid Modeler for SAP Applications.

Examples

There are a couple of examples related to the IBM InfoSphere Information Server Pack for SAP Applications. They require an existing Data Design Project.

To create the examples, go to File > New > Examples > IBM InfoSphere Information Server Pack for SAP Applications.
  • Business Object example model: This is an example for a predefined SAP model that can be used by the Rapid Modeler for SAP.
  • Example file for check table blacklists used in the Rapid Modeler for SAP. The Rapid Modeler for SAP can use files that contain the name for tables that are ignored during import. See Importing SAP metadata with InfoSphere Rapid Modeler for SAP. This is an example how this file might look like.
  • Example file for RFC destinations used in the Rapid Generator for SAP for ABAP extract jobs

    To generate jobs by using the ABAP extract stage with the RFC transfer method, one must provide a list of RFC destinations and associated program IDs in a file. This is an example how this file might look like.
