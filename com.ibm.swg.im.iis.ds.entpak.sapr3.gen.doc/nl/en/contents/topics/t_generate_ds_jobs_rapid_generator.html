Generating InfoSphere DataStage jobs by using InfoSphere Rapid Generator for SAP

You can use the InfoSphere™ Rapid Generator for SAP Applications wizard to generate ABAP extract, IDoc extract, and IDoc load jobs that extract or load data from and to an SAP system.

  • Configuring job generation for ABAP extract jobs
    You can use the InfoSphere Rapid Generator for SAP wizard to generate InfoSphere DataStage jobs that extract ABAP data from an SAP system. Every IBM® InfoSphere™ DataStage® job using the RFC data transfer method must be configured to use a unique RFC destination. You can run only one ABAP extract job at a time as ABAP stage using the same RFC destination cannot run concurrently.
  • Configuring job generation for IDoc extract job
    You can use the InfoSphere Rapid Generator for SAP wizard to generate IDoc extract jobs.
  • Configuring job generation for IDoc load jobs
    You can use the InfoSphere Rapid Generator for SAP wizard to generate InfoSphere DataStage jobs that load IDoc data from an SAP system.
