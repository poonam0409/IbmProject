Configuring the IBM InfoSphere DataStage environment for Rapid Generator for SAP

Before you use InfoSphere™ Rapid Generator for SAP Applications to generate jobs, you must configure the IBM InfoSphere DataStage® environment.

Procedure

  1. It isrecommend to create a dedicated project to be used by InfoSphere Rapid Generator for SAP. You can create a new project by using IBM InfoSphere DataStage and QualityStage™ Administrator.
  2. Open the project by using IBM InfoSphere Information ServerIBM InfoSphere Information Server Pack for SAP R3DataStage Administrator for SAPand create a connection to your SAP system.
  3. Define an ODBC data source on the InfoSphere DataStage server, as the job uses the ODBC stage for database access. The generated job reads and writes the data from a relational database (like IBM DB2®), when data is loaded or extracted from SAP systems. This configuration is performed at the operating system level and not within InfoSphere DataStage
    Note: For detailed instructions on how to create and configure ODBC data sources, contact your system administrator.
