Importing SAP metadata by using Rapid Modeler

You can use the InfoSphere™ Rapid Modeler for SAP wizard to import either the metadata from logical tables or IDoc types from SAP into a logical data model.

Before you begin

  • A logical data model must exist in a data design project.
  • The workspace must be configured for use with InfoSphere Rapid Modeler.

Procedure

  1. Right-click an existing Logical data model (LDM file), and select Rapid Modeler for SAP > Import metadata from SAP.
  2. Select an SAP system from Select SAP system .
  3. Choose the metadata to import.
    • To import logical table metadata, complete the task, Importing logical tables from SAP.
    • To import IDoc metadata, complete the task, Importing IDoc types from SAP.
  4. On the Review tables to be imported page, ensure that only the tables that you want to import are shown, and then click Next.
  5. On the Technical fields page, define a set of attributes that are added to each entity that is created in the logical data model. For example, an attribute can be an ID of your source system or a number that represents a time stamp when the data was imported.
  6. Click Next.
  7. To use a package that is not the default for the logical data model, select the package of the logical data model you want to use.
    1. Select Use a specific package as the import destination option. The table shows all existing packages in the logical data model.
    2. Click a package name to select it as the import destination.
  8. Click Next to start the import process.
  9. Click Finish to save the imported changes to your logical data model.

Results

New entities are created in the logical data model. Also for each data element in SAP, an atomic domain is created with the same name.
  • Importing logical tables from SAP
    You can import metadata for SAP logical tables by using the Rapid Modeler for SAP metadata import wizard.
  • Importing IDoc types from SAP
    You can use the IBM InfoSphere Data Architect Rapid Modeler for SAP metadata import wizard to import metadata for SAP IDoc types and segments.
