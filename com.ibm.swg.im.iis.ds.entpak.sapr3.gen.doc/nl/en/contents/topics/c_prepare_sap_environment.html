Prerequisites for the SAP environment

The SAP system that you are working on must be configured to work with Rapid Modeler and Rapid Generator for SAP.

This configuration contains user authorization profiles for design and run time, and a function module to be installed on the SAP system. Follow the steps that are required for the IBM® InfoSphere™ Information Server Pack for SAP Applications. The Pack installation contains transport files that you can import into your SAP system. Alternatively, you can configure your SAP system manually.

Note:
  • To use the Rapid Modeler for SAP, it is sufficient that the SAP user has the necessary design time authorizations.
  • To use Rapid Generator for SAP only for extracting and loading IDocs (no table extracts), it is sufficient that the SAP user has the authorizations for IDoc processing.
  • If the user has SAP_ALL privileges, no further configuration is required.
