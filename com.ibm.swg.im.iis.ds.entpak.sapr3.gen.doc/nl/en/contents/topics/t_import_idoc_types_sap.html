Importing IDoc types from SAP

You can use the InfoSphere™ Rapid Modeler for SAP metadata import wizard to import metadata for SAP IDoc types and segments.

Before you begin

Importing logical tables from SAP

Procedure

  1. On the Select SAP system page, select an import option, then click Next.
  2. Depending on your selection in the previous step different actions must be performed:
    Option Description
    Selected import option Action
    Browse SAP for IDocs
    1. Enter a search term for a table name. You can use % and * as wildcards.
    2. Specify if you want to search for a basic or an extended IDoc type.
    3. Click Search.
    4. Expand the IDoc type. The IDoc segment structure is displayed and the mandatory IDoc segments are highlighted in red.
    5. Click the check box next to the segment name to select the segments that you want to import. You can import one IDoc type per logical data model.
    Specify IDocs Manually
    1. Specify the IDoc type name in the IDoc type name text field.
    2. Specify whether the IDoc is of basic or an extended type.
    3. In the IDoc segment type names field, enter the names of the segment types (for example, E1KNA1M). You must use the names of the segment types and not the names of the segment definition. The segment type names are separated by newline.
  3. In the IDoc import options page, select the options to define how the IDocs must be imported.
  4. Click Next to start the IDoc and table collection process.
