コンポーネントの概要

Rapid Modeler および Rapid Generator for SAP クライアントは、IBM® InfoSphere™ Data Architect の拡張 (Eclipse プラグイン) として実装されます。

SAP との接続には、SAP JCo ライブラリー (SAP システムへの接続を行う Java ライブラリー) が使用されます。Rapid Modeler for SAP は、このクライアント・コンポーネントのみで構成されます。Rapid Generator for SAP は、ジョブを生成するためのユーザー入力を収集するクライアント・コンポーネントで構成されます。また、Rapid Generator は、生成されたジョブのコンパイルに IBM InfoSphere Information Server クライアントも使用します。実際のジョブ生成は、InfoSphere Information Server 上のコンポーネントで実行されます。 このコンポーネントは、InfoSphere Information Server ドメインの WebSphere® Application Server にインストールされるエンタープライズ・アプリケーション (EAR ファイル) です。 これらのコンポーネントの概要については、図 1を参照してください。
