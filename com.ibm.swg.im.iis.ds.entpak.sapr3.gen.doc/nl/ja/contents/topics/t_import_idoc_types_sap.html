SAP からの IDoc タイプのインポート

「IBM® InfoSphere™ Data Architect Rapid Modeler for SAP メタデータのインポート」ウィザードを使用して、SAP IDoc タイプとセグメントのメタデータをインポートできます。

始める前に

SAP から論理テーブルをインポート

手順

  1. 「SAP システムの選択」ページで、抽出オプションを選択してから、「次へ」をクリックします。
  2. 直前のステップの選択内容に応じて、異なるアクションを実行する必要があります。
    オプション 説明
    選択された抽出オプション アクション
    SAP 内の IDoc を参照
    1. テーブル名の検索語を入力します。% と * をワイルドカードとして使用できます。
    2. 基本 IDoc タイプを検索するか、拡張 IDoc タイプを検索するかを指定します。
    3. 「検索」をクリックします。
    4. IDoc タイプを展開します。IDoc セグメント構造が表示されます。必須 IDoc セグメントは赤で強調表示されます。
    5. セグメント名の横にあるチェック・ボックスをクリックして、インポートしたいセグメントを選択します。論理データ・モデルごとに 1 つの IDoc タイプをインポートできます。
    IDoc を手動で指定
    1. 「IDoc タイプ名」テキスト・フィールドに IDoc タイプ名を指定します。
    2. IDoc が基本タイプであるか、拡張タイプであるかを指定します。
    3. 「IDoc セグメント・タイプ名」フィールドに、セグメント・タイプの名前 (例えば、E1KNA1M) を入力します。 セグメント定義の名前ではなく、セグメント・タイプの名前を使用する必要があります。セグメント・タイプ名は改行で区切られます。
  3. 「IDoc のインポート・オプション (IDoc import options)」ページで、IDoc のインポート方法を定義するオプションを選択します。
  4. 「次へ」をクリックして、IDoc とテーブルの収集プロセスを開始します。
