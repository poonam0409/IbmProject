Pack for SAP Applications 用の IBM InfoSphere Data Architect コンポーネント

IBM® InfoSphere™ Information Server Pack for SAP Applications は、論理データ・モデルをインポートし、インポートされたメタデータを使用する InfoSphere DataStage® ジョブを生成するための、InfoSphere Data Architect ツールを提供します。

  • 概要
    ほとんどすべての SAP プロジェクトの一部として、SAP のデータ・モデルを識別し、分析し、文書化する必要があります。これらのデータ・モデルはほとんどの場合、カスタマイズされ、補完されているので、カスタム・データ・モデルは標準モデルとは異なっています。SAP システムからデータを抽出したり、SAP システムにデータをロードしたりするには、外部アプリケーションと SAP インターフェースに関する知識が必要です。企業内では一般に、異なるチームがデータ統合と SAP システムをそれぞれ担当します。この責任の分割の結果、(例えば、フラット・ファイルの交換に) 一時の間に合わせで統合が実行されることになります。
  • 前提条件
    IBM InfoSphere Data Architect Rapid Modeler for SAP および IBM InfoSphere Rapid Generator for SAP クライアントは、IBM InfoSphere Data Architect の拡張であり、IBM Installation Manager を使用して独自のランチパッドと一緒にインストールされます。IBM InfoSphere Rapid Generator for SAP の場合、ジョブを生成するサーバー・コンポーネントをインストールする必要があります。このサーバー・コンポーネントは、InfoSphere Information Server の WebSphere® Application Server にインストールされるエンタープライズ・アプリケーションです。 このアプリケーションは、.ear ファイルとして提供され、標準の WebSphere Application Server と一緒に、例えば Web ブラウザーを使用して管理コンソールから、インストールできます。
  • IBM InfoSphere Data Architect コンポーネントのインストール
    IBM InfoSphere Data Architect Rapid Modeler および IBM InfoSphere Rapid Generator for SAP 用にクライアントとサーバーのコンポーネントをインストールする必要があります。
  • Rapid Generator for SAP 用の IBM InfoSphere DataStage 環境の構成
    IBM InfoSphere Rapid Generator for SAP を使用してジョブを生成する前に、IBM InfoSphere DataStage 環境を構成しておく必要があります。
  • InfoSphere Data Architect ワークスペースの準備
    IBM InfoSphere Data Architect Rapid Modeler for SAP および IBM InfoSphere Rapid Generator for SAP を使用する前に、InfoSphere Data Architect ワークスペースを準備しておく必要があります。
  • Rapid Modeler を使用した SAP メタデータのインポート
    IBM InfoSphere Data Architect Rapid Modeler for SAP ウィザードを使用して、論理テーブルからのメタデータか、SAP からの IDoc タイプを論理データ・モデルにインポートすることができます。
  • IBM InfoSphere Rapid Generator for SAP を使用した InfoSphere DataStage ジョブの生成
    IBM InfoSphere Rapid Generator for SAP ウィザードを使用して、SAP システムからデータを抽出するか、SAP システムにデータをロードする、ABAP Extract、IDoc Extract、および IDoc Load ジョブを生成することができます。
  • SAP システムとモデルを調整する
    論理データ・モデルに SAP メタデータをインポートした後、そのメタデータは、インポート時点における SAP システムのスナップショットを表します。新規モデルに対して IBM InfoSphere Data Architect Rapid Modeler for SAP のメタデータ・インポート・ウィザードを再実行することによって、モデルを再抽出または調整したい場合があります。新規モデルが作成された後は、InfoSphere Data Architect を使用して、元の論理データ・モデルと新規論理データ・モデルを比較できます。メタデータの再抽出では、すべてのチェック・テーブルとテキスト・テーブルが再収集されます。
  • トラブルシューティング
    このトラブルシューティングのトピックでは、IBM InfoSphere Information Server Pack for SAP Applications で検出される可能性がある問題を取り扱います。
