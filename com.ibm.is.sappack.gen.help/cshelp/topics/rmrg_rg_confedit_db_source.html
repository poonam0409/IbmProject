Database source connection

In the Database source connection page you can specify the data connection.

Database settings
Stage Type
Select one of the following types:
ODBC Connector
ODBC Data Source
ODBC datasource name (mandatory field).
ODBC User
ODBC user name (This can be a mandatory field in some platforms).
ODBC Password
Password for the ODBC user (This can be a mandatory field in some platforms).
ODBC SQL WHERE Condition
SQL WHERE condition that is used to select the data from the database.
Code page
The options are: Default, Unicode and User-specified.
Code page name
Name of the code page that is used when the User-Specified option is selected.
Use different schema name
Select to use a schema name other than the schema name that exists in the Physical data model.
Schema
Schema name to be used if the Use different schema name checkbox is selected.
Sequential File
Recommended for test environment only.
File Name Prefix
Path and the filename prefix for the Sequential File stage filename.

The filename is composed of filename prefix and ABAP table or IDoc segment name, for example, C:\tables\KNA1.

There are two replacement options for the schema name and the table or IDoc segment name:

  • {schema} - schema name
  • {tab} - ABAP table or IDoc segment name

Example: prefix = C:\tables\Extract-{schema}.{tab}.txt, schema = SchemaB and ABAP table = KNA1. These values are copied to C:\tables\Extract-SchemaB.KNA1.txt as the filename.

First Line is Column Names
Select if the first data line contains the column names.
NLS Map name
The options are: Project default, ISO8859-1, UTF8 and UTF16
