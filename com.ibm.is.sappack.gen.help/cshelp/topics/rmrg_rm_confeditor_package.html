Import package

You can define a path to the package to be used as a root for the SAP metadata import.

  1. Right click the package and select and Add sub-package.
  2. Specify the name for the new package.
  3. Repeat the steps to add more sub-packages.
    To rename or remove a sub-package, right-click the package and select the appropriate action.

Note:
If you selected to import check and text tables in the Import options page, then the tables are imported into the package specified in the Check table package page.
