Import metadata from SAP

In InfoSphere Rapid Modeler for SAP Applications, you can import metadata from an SAP system into a logical data model of InfoSphere® Data Architect. You can select the SAP system, SAP object list, the Rapid Modeler configuration file, and the logical data model from which you want to import metadata.

SAP connection
Specify the name of the SAP system from which you want to import metadata.
SAP Objects
Select an ABAP table list or an IDoc segment list to define the scope of metadata import into a logical data model.
Rapid Modeler Configuration
Select the Rapid Modeler configuration. You must create the configuration files earlier.
Logical Data Model
Specify the name of the logical data model into which the SAP metadata is imported.
Use separate check table model
Select the checkbox if you want all check tables to be imported into a separate logical data model.
Check table model
Specify the name of the logical data model for check tables. The option is enabled if the Use separate check table model check box is selected.
