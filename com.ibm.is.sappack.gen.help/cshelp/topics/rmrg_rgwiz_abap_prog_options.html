ABAP program options

Use this page to configure the ABAP program settings for programs to generate the IBM® InfoSphere® DataStage® ABAP extract jobs.

ABAP upload options
Save the ABAP programs in workspace
Select the checkbox to save all ABAP programs in a compressed file named ModelName.zip in the current data design project. You can use the file to create ABAP programs manually.

To upload the ABAP program later, right-click the compressed file and click InfoSphere for SAP > Upload archived ABAP programs.
Upload ABAP programs to SAP
Select the checkbox to upload programs automatically to SAP after the job is generated.
SAP connection
Specify the SAP connection to which you need to upload the ABAP program.
CTS settings
Use CTS for ABAP upload
Select the checkbox to upload the ABAP programs with a Change and Transport System (CTS) request.
You must also select a package and a valid Change and Transport System request for the package.
Package
Select a package.
CTS Request Name
Select a Change and Transport System request for the package. The selected package is then retrieved from the specified SAP system.
