Rapid Generator configuration editor

You can view and modify the Rapid Generator configuration properties on the following tabs.

General
On the General tab you can view document properties such as the last modified date or Rapid Generator mode. You can also add some information and modify information related to the author, company and version.

In addition you can also validate your configured SAP settings.
Source
On the Source page you can specify the properties for the source stage.
For extract jobs these are SAP credentials, for load jobs these are the ODBC or Sequential File settings.
Target
On the Target page you can specify the properties for the target stage.
For load jobs these are SAP credentials, for extract jobs these are the ODBC or Sequential File settings.
ABAP Extract
On the ABAP Extract page you can specify the SAP attributes that are required for the ABAP Extract job.
Derivations
On the Derivations page you can specify the InfoSphere DataStage custom derivations that are used in each generated job.
Description
On the Description page you can specify a description that is added to each generated job.
Parameters
On the Parameters page you can specify additional job parameters that are added to the generated job.
