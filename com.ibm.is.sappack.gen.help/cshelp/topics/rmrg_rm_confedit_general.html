Rapid Modeler configuration editor

The InfoSphere Rapid Modeler configuration defines how SAP objects are imported into a logical data model. It also defines how the SAP data types are mapped to logical data types and lists the additional check tables and technical fields.

General
On the General page you can view document properties such as the last modified date and Rapid Modeler. You can modify information such as details about the author, company and version.
Import options
On the Import Options page you can specify the SAP table and IDoc options.
Database type
On the Database type page you can specify the type of the database for which the SAP metadata is imported.
Technical fields
On the Technical fields page you can specify attributes that are added to every entity.
Package
On the Package page you can specify a package for the imported tables.
Check table package
On the Check table package page you can specify a package for the imported check and text tables.
