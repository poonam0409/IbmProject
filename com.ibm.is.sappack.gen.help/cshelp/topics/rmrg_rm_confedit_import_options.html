Import options

In the Import Options page you can specify the SAP table and IDoc options.

Table options
Check and text tables
Select options how to import metadata for both check tables and text tables.
Import metadata for related check tables
Select the option to import metadata for check tables.
Collect check tables recursively
Select the option to import all metadata of check tables that are used by check tables. If this option is not selected the metadata of check tables of only retrieved SAP tables are imported.
Import metadata for related text tables
Select the option to additionally import the corresponding text tables for check tables. For example, for the check table containing the valid country code such as US, the corresponding text table contains the country names United States.
Check table blacklist
You can specify a list of check tables to be excluded when SAP objects are imported. The list must contain one table name in each line, and each table must be separated by a carriage return.
Relations
Select the option to import the relationship between the retrieved SAP tables.
Enforce
Select the option to enforce the relations between SAP tables.
IDoc options
IDoc Segment Fields
Make all segments fields nullable
Select the option if you want to allow the columns to be nullable.
Allow all types for columns
Specify if you want to allow all data types for columns.
Use VARCHAR only
Specify if you want to allow only VARCHAR for all columns.
Length factor
Specify a length factor for the VARCHAR columns
Check tables
Select options to import metadata for both check tables and text tables.
Import metadata for related check tables
Select the option to import metadata for check tables.
Import metadata for related text tables
Select the option to import metadata for text tables.
Check table blacklist
You can specify a list of check tables to be excluded when SAP objects are imported. The list must contain one table name for each line, and each table must be separated by a carriage return.
