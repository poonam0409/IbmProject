ABAP Extract jobs

In the ABAP Extract page you can specify the ABAP extract settings.

Gateway settings
(communication settings)
SAP Gateway Host
SAP Gateway hostname.
SAP Gateway Service
SAP Gateway Service name, for example, sapgw00.
RFC destination creation
Create RFC destination automatically
Select to create an RFC destination when the ABAP extraction jobs are generated.

The RFC destination is deleted after the job iss successfully run.

Note: If the RFC destination already exists and you do not choose to delete the existing RFC destination then it results in an error when the job is run.

Delete RFC destination if it exists
Select to delete an RFC destination if it exists.
RFC destination names
You can choose one of the following options to specify the names of RFC destinations:
Static (from list)
The list contains one line for each RFC destination. Each line contains the name of the RFC destination and the program ID to be used for the RFC destination separated by white space. For example, you can define two static RFC destinations as follows: 

ZDEST1 ZPROGID1
ZDEST2 ZPROGID2
Dynamic (generated from prefix)
Enter the name prefix of the RFC destination that is generated when the job is generated and not when the job is run. The complete name is generated with a unique number appended to the specified prefix.
Max. number of flows in a job
Specify the number of flows to be created per job to indicate the number of ABAP extract stages that can appear in a generated job. You cannot run two ABAP extract stages that use the same RFC destination at the same time.
Background processing
Enable background processing
Select if you want the generated ABAP extraction jobs to run the ABAP programs in the background.
Variant prefix
Specify a variant of the program to be created in SAP. The name of the variant is generated during the job generation by appending an ID to the variant name prefix.
Program startup delay in seconds
Specify the number of seconds that the ABAP stage must wait before it schedules the background job.
