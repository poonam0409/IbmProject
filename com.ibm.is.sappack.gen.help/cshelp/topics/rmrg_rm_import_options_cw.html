Import options

The Import Options page is available only if you are using the Conversion Workbench for SAP. You can specify the SAP table and check table options in this page.

Check table lists
You can create a list that lists check tables to include or exclude when SAP objects are imported. The list must contain one table name per line, and each table must be separated by a carriage return.
Select Use as Black list to create and use a file that lists check tables to be excluded during the check table import.
Select Use as White list to create and use a file that lists check tables to include during the check table import process.
Relations and keys
You can specify an option for primary key and foreign key creation.
Allow all data types for columns
Specify if you want to allow all data types for columns.
Use VARCHAR for all columns
Specify if you want to allow only VARCHAR for all columns.
Length factor
Specify a length for columns
Default value
Specify a default value for length.
Allow columns to be nullable
Select this option if you want to allow the columns to be nullable.
Columns
Specify options for columns.
Additional Technical fields
Specify whether you want to add technical fields to every entity that is created in the logical data model.
Check table options
You can select options to create both check tables and text tables or to create transcoding table schema, or select not to create check tables.
