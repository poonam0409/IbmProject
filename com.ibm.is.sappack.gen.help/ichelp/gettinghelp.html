Cheat sheets

A couple of cheat sheets are available that basically contain the same information as the movies above. To open them go to Help -> Cheat Sheets -> InfoSphere Pack for SAP. There are two subcategories:
  1. End-to-end: Compound cheat sheets that describe all steps for larger tasks like "Importing metadata". Use these cheat sheets to get started.
  2. Tasks: Short cheat sheets describing smaller tasks. These cheat sheets are sub-steps of the end-to-end cheat sheets. Use them as a reference.
Select a specific cheat sheet and click OK.

Example files

Example files for all required artifacts are available through an example wizard. Go to File -> New -> Example -> InfoSphere Pack for SAP -> Example files for ABAP Table Lists, IDoc Segment Lists, Rapid Modeler, and Rapid Generator configurations. The wizard lets you select a Data Design project where the files are copied to.

InfoCenter help

To open the Info center help go to Help -> Help Contents. In the window that opens look for the top level category InfoSphere for SAP.
