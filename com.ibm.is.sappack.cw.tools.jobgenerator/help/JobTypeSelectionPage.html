Select Job Type

The list allows for selecting from the job types available for the current model file. Subsequent wizard pages will allow you to further configure the job generation according to your selection.
  • ABAP Extract Job: An extract job is generated for those entities in the model that represent SAP logical tables. The appropriate objects are stored into a relational database through ODBC.
  • IDOC Extract Job: An extract job is generated for those entities in the model that belong to an SAP IDoc Type. Although an IDoc has a hierarchical data structure, the extracted objects are stored into a relational database via ODBC.
  • IDOC Load Job: This option generates a job which reads IDocs from a relational database, reconstructs its hierarchical structure and sends them to SAP.
  • Move Job: A move job is generated for all entities in the model. It simply copies the appropriate objects between two ODBC datasources.
  • MIH Load Job: A MIH load job.
  • MIH IDOC Load Job: A MIH IDoc load job.
